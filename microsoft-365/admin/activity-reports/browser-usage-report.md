# <a name="microsoft-365-reports-in-the-admin-center---microsoft-browser-usage"></a>管理中心中的 Microsoft 365 报告 - Microsoft 浏览器使用情况

Microsoft 365 **报告** 仪表板显示组织中产品的活动概述。 它使您能够深入了解单个产品级别报告，从而更细致地了解每个产品内的活动。 请查看[报表概述主题](activity-reports.md)。 在 Microsoft 浏览器使用情况报告中，你可以了解 Internet Explorer、旧版 Microsoft Edge 和新 Microsoft Edge 使用情况。 使用情况报告基于通过 Microsoft 浏览器访问的 Microsoft 365 联机服务。

 > [!NOTE]
 > 你必须是 Microsoft 365 中的全局管理员、全局读者或报告读者，或者是 Exchange、SharePoint 或 Skype for Business 管理员才能查看报告。

## <a name="how-to-get-to-the-microsoft-browser-usage-report"></a>如何获取 Microsoft 浏览器使用情况报告

1. 在管理中心，转到“**报表**”\> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">使用情况</a>页面。 
2. 在仪表板主页上，单击 Microsoft浏览器使用卡上的"查看更多"按钮。
