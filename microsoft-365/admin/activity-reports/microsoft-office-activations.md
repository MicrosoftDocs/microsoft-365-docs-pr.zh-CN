# <a name="microsoft-365-reports-in-the-admin-center---microsoft-office-activations"></a>管理中心中的 microsoft 365 报表-Microsoft Office 激活

Microsoft 365 " **报告** " 仪表板显示组织中各产品的活动概述。 它让你能够深入研究各产品级报表，以便更细致地了解每个产品内的活动。 请查看[报表概述主题](activity-reports.md)。
  
Office 激活报表将显示在至少一个设备上激活了其 Office 订阅的用户。 它提供了适用于 Office、Project 和 Visio Pro for Office 365 订阅激活的 Microsoft 365 应用程序的细目，以及跨桌面和设备的激活细目。 此报表对于帮助你识别可能需要更多帮助和支持才能激活其 Office 订阅的用户非常有用。
  
> [!NOTE]
> 您必须是 Microsoft 365 或 Exchange、SharePoint、团队服务、团队通信或 Skype for Business 管理员中的全局管理员、全局读取器或报告阅读器才能查看报告。  
  
## <a name="how-to-get-to-the-office-activations-report"></a>如何使用 Office 激活报表

1. 在管理中心，转到“**报表**”\> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">使用情况</a>页面。

    
2. 从 " **选择报告** " 下拉项中，选择 " **Office 365** \> **激活**"。 
  
## <a name="interpret-the-office-activations-report"></a>解读 Office 激活报表

可以通过查看" **激活**"和" **用户**"图表，了解组织的 Office 激活。 
  
![Office 激活的计数](../../media/8c0ae08d-2d71-4437-9147-12c345bb5e9d.png)
  
|Item|说明|
|:-----|:-----|
|1  <br/> |Office 激活报表提供自报表数据显示在图表右上角起，Office 许可证激活数据的当前状态。  <br/> |
|2   <br/> |每个报告中的数据通常最长为过去24到48小时。  <br/> |
|3   <br/> |" **激活**"图表显示台式计算机和其他设备上的 Office 激活次数。  <br/> |
|4   <br/> |" **用户**"图表显示在台式计算机或其他设备上有权激活和已激活 Office 订阅的用户数。  <br/> |
|5   <br/> | 在" **激活**"图表上，Y 轴是 Office 激活次数。  <br/>  在" **用户**"图表上，Y 轴是用户所执行的激活 Office 的操作。  <br/>  这两个图表的 X 轴都表示此特定报表的所选日期范围。  <br/> |
|6   <br/> |您可以通过选择图例中的项目来筛选所见图表。 例如，在 **激活** 图表中，选择 **Windows OS**、 **Mac OS**、 **Windows 10 移动**版、 **iOS** 或 **Android** 以仅查看与每个相关联的信息。 更改选择不会更改网格表中的信息。 <br/> ![设备的激活数据](../../media/59d3ec6e-2a6e-4b21-8aac-c73038c47b9f.png)  <br/> |
|7   <br/> | 下表显示了用户级 Office 激活的详细介绍。这是所有用户及向其分配的 Office 产品的列表。你可以向表中添加其他列。  <br/> ![Office 激活可用列](../../media/410a4baa-cef8-4676-bf7c-02a907a3a575.png)<br/> " **用户名**"是用户的电子邮件地址。  <br/> 如果用户为 "**显示名称**"，则为完整名称。  <br/> **产品许可证** 是分配给此用户的产品。  <br/> " **上次激活日期**"表示用户在台式计算机或其他设备上激活 Office 的日期。  <br/> 如果用户通过共享计算机激活使用 Office，则**使用的共享计算机激活**为 true。 <br/> **Windows** 指用户在其上激活 Office 的 Windows 桌面的数量。  <br/> **Mac** 是指用户在其上激活 Office 的 Mac 桌面的数量。  <br/> **Windows 10 移动** 版指的是用户在上激活 Office 的 Windows 10 移动设备的数量。  <br/> **iOS** 是指用户在其上激活 Office 的 iOS 设备的数量。  <br/> **Android** 指用户在其上激活 Office 的 Android 设备的数量。  <br/>  如果组织的策略阻止你查看显示了可识别用户信息的报表，可更改所有这些报表的隐私设置。 请参阅[Microsoft 365 管理中心的活动报告](activity-reports.md)中的 "报告" 部分的 "**隐藏用户详细信息**"。  <br/> |
