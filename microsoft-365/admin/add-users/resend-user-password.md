# <a name="resend-a-users-password---admin-help"></a>重新发送用户密码 - 管理员帮助

本文介绍如何将通知电子邮件重新发送给 Office 365 中的新用户。 在创建新用户时，他们不会收到带有新密码的电子邮件，这可能会发生这种情况。 为此，请重置用户的密码。

## <a name="before-you-begin"></a>准备工作

本文面向的是为企业、学校或非营利组织设置密码过期策略的人员。 若要完成这些步骤，你需要使用 Microsoft 365 管理员帐户登录。 [什么是管理员帐户？](../admin-overview/admin-overview.md)

您必须是 [全局管理员或密码管理员](about-admin-roles.md) 才能执行这些步骤。

## <a name="resend-user-password"></a>重新发送用户密码
  
::: moniker range="o365-worldwide"  
  
1. 在管理中心，转到“**用户**”\>“<a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">活动用户</a>”页面。

::: moniker-end

::: moniker range="o365-germany"

1. 在管理中心，转到“**用户**”\>“<a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">活动用户</a>”页面。  

::: moniker-end

::: moniker range="o365-21vianet"

1. 在管理中心，转到“**用户**”\>“<a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">活动用户</a>”页面。 

::: moniker-end

2. 在 " **活动用户** " 页上，选择该用户，然后选择 " **重置密码**"。

3. 按照 " **重置密码** " 页上的说明操作，为用户自动生成一个新密码或为其创建一个新密码，然后选择 " **重置**"。  

4. 输入用户可访问的电子邮件地址，以便他们接收新密码，并跟踪它们以确保其获得。
