# <a name="manage-an-office-365-group-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 管理中心中管理 Office 365 组

[创建 Office 365 组](create-groups.md)并添加组成员后，可以对组进行配置。 您可以编辑组名称或说明、管理所有者或成员，并指定外部发件人是否可以通过电子邮件发送组以及是否将组对话的副本发送到成员。 

## <a name="edit-a-group-name"></a>编辑组名称

::: moniker range="o365-worldwide"

1. 在 "管理中心" 中，转到 "**组** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">组</a>" 页面，然后选中要更改的组对应的复选框。

2. 选择 "**编辑名称**"。

3. 在 "**编辑详细信息**" 窗格中，输入组的新名称，然后选择 "**保存**"。

::: moniker-end

::: moniker range="o365-germany"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改的组的名称。

2. 在 "**名称**" 旁边，选择 "**编辑**"。

3. 在 "**编辑详细信息**" 窗格中，输入组的新名称，然后选择 "**保存**"。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改的组的名称。

2. 在 "**名称**" 旁边，选择 "**编辑**"。

3. 在 "**编辑详细信息**" 窗格中，输入组的新名称，然后选择 "**保存**"。

::: moniker-end

## <a name="manage-group-owners"></a>管理组所有者

::: moniker range="o365-worldwide"

1. 在 "管理中心" 中，转到 "**组** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">组</a>" 页面。

2. 选择要管理的组的名称以打开 "设置" 窗格。

3. 在 "**成员**" 选项卡上的 "**所有者**" 下，选择 "**查看全部和管理所有者**"，然后执行下列操作之一：

    - 若要添加所有者，请选择 "**添加所有者**" 并搜索或选择一个所有者，然后选择 "**保存**"。 
    - 若要删除所有者，请搜索或选择一个组所有者，选择人员姓名旁边的**X** ，然后选择 "**保存**"。 

::: moniker-end

::: moniker range="o365-germany"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改其所有者的组的名称。

2. 在 "**所有者**" 旁边，选择 "**编辑**"，然后执行下列操作之一：

    - 若要添加所有者，请选择 "**添加所有者**" 并搜索或选择一个所有者，然后选择 "**保存**"。 
    - 若要删除所有者，请搜索或选择一个组所有者，选择人员姓名旁边的**X** ，然后选择 "**保存**"。 

::: moniker-end

::: moniker range="o365-21vianet"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改其所有者的组的名称。

2. 在 "**所有者**" 旁边，选择 "**编辑**"，然后执行下列操作之一：

    - 若要添加所有者，请选择 "**添加所有者**" 并搜索或选择一个所有者，然后选择 "**保存**"。 
    - 若要删除所有者，请搜索或选择一个组所有者，选择人员姓名旁边的**X** ，然后选择 "**保存**"。 

::: moniker-end

## <a name="send-copies-of-conversations-to-group-members-inboxes"></a>将对话副本发送到组成员的收件箱
  
使用管理中心创建组时，默认情况下，用户不会获取发送到其收件箱的组电子邮件和会议邀请的副本。 他们需要转到组才能查看对话和会议。 您可以在管理中心更改此设置。

当您启用此设置时，组成员将获得组电子邮件和会议邀请发送到其 Outlook 收件箱的副本。 他们可以阅读并删除该电子邮件的副本，不影响任何人。 在"组"收件箱中，电子邮件的副本仍然存在。

组成员可以选择停止关注 Outlook 中的组，从而退出接收这些电子邮件。

::: moniker range="o365-worldwide"

1. 在 "管理中心" 中，转到 "**组** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">组</a>" 页面，然后选择要更改的组的名称。 

2. 如果想要成员在自己的收件箱中接收组邮件和日历项目的副本，请在 "**设置**" 选项卡上，选择 "**向组成员发送组对话和事件的副本**"。

3. 选择“**保存**”。

::: moniker-end

::: moniker range="o365-germany"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改的组。

2. 在 "**名称**" 旁边，选择 "**编辑**"。

3. **如果想**要成员在自己的收件箱中接收组邮件和日历项目的副本，请**将组对话和事件的 "发送" 副本转换为 "将组成员的收件箱"** 。

4. 选择“**保存**”。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在 "管理中心" 中，转到 "**组** \> **组**" 页面，然后选择要更改的组。

2. 在 "**名称**" 旁边，选择 "**编辑**"。

3. **如果想**要成员在自己的收件箱中接收组邮件和日历项目的副本，请**将组对话和事件的 "发送" 副本转换为 "将组成员的收件箱"** 。

4. 选择“**保存**”。

::: moniker-end

## <a name="let-people-outside-the-organization-email-the-group"></a>允许组织外部的人员向组发送电子邮件

如果您想要拥有公司电子邮件地址（如 info@contoso.com），则此选项非常有用。
 
::: moniker range="o365-worldwide"

1. 刷新 "管理中心" 页，使新组显示出来。

2. 在 "管理中心组" 列表中，选择要更改的组的名称，然后在 "**设置**" 选项卡上，选择 "**允许外部发件人通过电子邮件发送此组**"。
    
4. 选择“**保存**”。

::: moniker-end

::: moniker range="o365-germany"

1. 刷新 "管理中心" 页，使新组显示出来。
    
2. 在 "管理中心组" 列表中，选择要更改的组，然后在 "**名称**" 旁边，选择 "**编辑**"。 
    
3. 将" **允许组织外部的人员向组发送电子邮件**"开关设置为" **开启**"。
    
4. 选择“**保存**”。

::: moniker-end

::: moniker range="o365-21vianet"

1. 刷新 "管理中心" 页，使新组显示出来。
    
2. 在 "管理中心组" 列表中，选择要更改的组，然后在 "**名称**" 旁边，选择 "**编辑**"。 
    
3. 将" **允许组织外部的人员向组发送电子邮件**"开关设置为" **开启**"。
    
4. 选择“**保存**”。

::: moniker-end
