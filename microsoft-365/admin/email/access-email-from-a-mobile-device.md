# <a name="options-for-accessing-email-from-your-mobile-device"></a>用于从移动设备访问电子邮件的选项

组织配置设置，这些设置控制可用于访问工作帐户的电子邮件、日历和联系人的移动应用。

Outlook for Android 和 Outlook for iOS 移动应用专为你的工作或学校电子邮件设计。 这些应用提供重点收件箱[](https://support.microsoft.com/office/f445ad7f-02f4-4294-a82e-71d8964e3978)等功能，将重要电子邮件放在最前面和中间日程安排助理，[](https://support.microsoft.com/office/scheduling-made-easy-in-outlook-mobile-11c5bee5-d78a-4a2b-80c2-2b386ddb4470)这可以帮助你查找与同事合作的会议时间，并仅查找可用的会议室。
  
## <a name="end-user-experience"></a>最终用户体验

### <a name="outlook-only"></a>仅 Outlook

组织要求您使用 Outlook for Android 或 Outlook for iOS 移动应用访问公司电子邮件、日历和联系人。 下载并安装 Outlook for Android 或 Outlook for iOS 后，数据将开始同步。

![使用 Outlook 同步电子邮件的示例电子邮件](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)

有关更多 Outlook 功能，请查看"针对 [iOS 或 Android](https://support.microsoft.com/office/de075b19-b73c-4d8a-841b-459982c7e890) 手机优化 Outlook 移动应用"。 如果遇到 [任何问题，请转到 Outlook for iOS 和 Outlook for Android](https://support.microsoft.com/office/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6) 帮助中心。

### <a name="any-email-app"></a>任何电子邮件应用

贵组织建议使用 Outlook for Android 或 iOS 移动应用访问电子邮件、日历和联系人的工作或学校帐户。 如果你未在接下来几小时内执行任何操作，您的电子邮件将自动开始同步。 如果选择使用 Outlook for Android 或 iOS 移动应用，则数据将仅与 Outlook for Android 或 Outlook for iOS 移动应用同步。 如果你选择继续使用第三方应用，你的数据将立即开始同步。
