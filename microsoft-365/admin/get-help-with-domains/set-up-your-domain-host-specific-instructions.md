# <a name="set-up-your-domain"></a>设置域

若要开始使用自定义域 (contoso.com) 自定义Microsoft 365，需要验证域并配置域的 DNS 记录。 
  
您可以使用域主机中的管理工具添加和管理 DNS 记录，或让 Microsoft 控制您的域记录，我们将进行设置。

## <a name="let-microsoft-365-manage-your-dns-records"></a>让Microsoft 365管理 DNS 记录 

了解如何更改[名称服务器以设置Microsoft 365注册机构的名称服务器](change-nameservers-at-any-domain-registrar.md)。

## <a name="manage-your-own-dns-records"></a>管理你自己的 DNS 记录

有关说明，请参阅 [在任何 DNS 托管提供商上创建 DNS 记录](create-dns-records-at-any-dns-hosting-provider.md)
