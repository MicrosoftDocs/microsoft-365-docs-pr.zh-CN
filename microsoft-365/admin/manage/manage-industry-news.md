# <a name="manage-industry-news"></a>管理行业新闻

若要向用户提供有关您的行业的最新新闻标题和您的组织的信息，请使用 [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews) 服务为您的组织启用自定义的新闻源。您还可以管理 Bing 主页和 Microsoft Edge "新建选项卡" 页的设置， (从) 的边缘87的版本开始。

## <a name="what-your-users-will-see"></a>您的用户将看到的内容

进入 Bing 主页的登录用户将看到您的新闻源位于组织中的个性化信息下。

## <a name="news-settings"></a>新闻设置

作为管理员，您可以控制组织的新闻源设置，包括选定的行业和必应的主页，以及 Microsoft Edge "新建选项卡" 页面 (从第87边缘) 开始。

1. 在 Microsoft 365 管理中心，转到 " **设置** " "  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"
2. 在 " **新闻** " 面板中，单击 " **常规** " 选项卡
3. 在 **行业** 列表中，选择您的组织的行业。 这将确定组织的新订阅源中显示的常规新闻。 Microsoft 可以使用你的帐户中的信息预选择行业。 您可以通过更新行业列表来删除或添加行业。
4. 在 " **主题** " 字段中，输入您想查看有关的新闻文章的主题。 你的用户将不能更改这些主题。
5. 您可以在 **排除内容** 字段中阻止包含关键字的文章。  例如，若要避免在新闻源中显示包含关键字 "bake" 的文章，请在 "排除内容" 字段中添加关键字 "bake"。 避免在 (、it 和等 ) 的情况下包含常规术语，它们可以阻止相关内容显示在企业新闻源中。
6. 选择 **“保存”** 。 可能需要长达24小时才能显示更改。

## <a name="bing-homepage"></a>必应主页

您可以自定义 Bing 主页以包含有关您的行业的新闻。 

1. 在 Microsoft 365 管理中心，转到 " **设置**  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"。 
2. 在 " **新闻** " 面板中，单击 " **bing 主页** " 选项卡，然后选择 " **在 Bing 主页上包含"** 。
3. 行业新闻显示在 Bing.com 上的组织中的个性化信息下方。

## <a name="microsoft-edge-new-tab-page"></a>Microsoft Edge 新选项卡页 
当用户使用有效的工作或学校帐户登录 Microsoft Edge (版本87或更) 高版本时，他们可以看到为您的组织量身定做的新闻。

1. 在 Microsoft 365 管理中心，转到 " **设置**  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"。
2. 在 " **新闻** " 面板中，单击 " **Microsoft Edge 新建选项卡页** "。
3. **在 "新建" 选项卡页上选择 "允许 Office 365 内容** "。 启用后，用户可以自定义新的选项卡，以显示 Office 365 中的信息，包括建议的和最近使用的文件，以及频繁使用的 SharePoint 网站和其他信息。
4. **在 "新建" 选项卡页上选择 "显示公司信息和行业新闻** "。 您的组织和行业的新闻文章将针对选择查看其新选项卡页上的文章的用户显示。
