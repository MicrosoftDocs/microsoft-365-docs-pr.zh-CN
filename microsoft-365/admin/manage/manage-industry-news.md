# <a name="manage-industry-news"></a>管理行业新闻

若要向用户提供有关您的行业的最新新闻标题和您的组织的信息，请使用 [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews) 服务为您的组织启用自定义的新闻源。 您还可以启用每日行业新闻电子邮件更新，并为 Bing 主页和 Microsoft Edge "新建选项卡" 页自定义设置。

## <a name="what-your-users-will-see"></a>您的用户将看到的内容

你可以选择向你的用户发送带有标题的日常行业新闻电子邮件更新和完整文章的链接。 用户可以按照以下其他主题来自定义其电子邮件更新：选择何时传递更新、排除 paywalls 后面的文章以及选择要查看的文章数。

:::image type="content" source="../../media/manage-industry-news-1.png" alt-text="行业新闻页面的屏幕截图 ":::

登录到 Bing 主页的用户可以在组织的个性化信息下看到你的行业新闻源。

:::image type="content" source="../../media/manage-industry-news-2.jpg" alt-text="来自 web 的行业新闻的图像轮播屏幕截图":::

他们还可以在其 Microsoft Edge "新建" 选项卡页上查看公司、行业和内部新闻或个性化的工作信息。

:::image type="content" source="../../media/manage-industry-news-3.png" alt-text="带有行业新闻文章的 Microsoft Edge 新选项卡页":::

## <a name="news-settings"></a>新闻设置

作为管理员，您可以控制组织的新闻源设置，包括选定的行业和必应主页、Microsoft Edge 新选项卡页和电子邮件体验。

1. 在 Microsoft 365 管理中心，转到 "**设置**" "  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"
2. 在 " **新闻** " 面板中，单击 " **常规** " 选项卡
3. 在 **行业** 列表中，选择您的组织的行业。 这将确定组织的新订阅源中显示的常规新闻。
4. 在 " **主题** " 字段中，输入您想查看有关的新闻文章的主题。 你的用户将不能更改这些主题。
5. 您可以通过将关键字添加到 **排除内容** 字段来阻止主题和情景。 避免在 (、it 和等 ) 中包括常规术语，它们可以阻止相关内容显示在您的行业新闻源中。
6. 选择**保存**。

## <a name="industry-updates-in-email"></a>电子邮件中的行业更新

您可以向用户的收件箱发送每日包含相关行业新闻的电子邮件更新。 了解有关 [行业新闻电子邮件更新](https://newspro.microsoft.com/baw/faq)的详细信息。

1. 在 Microsoft 365 管理中心，转到 "**设置**  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"。
2. 在 " **新闻** " 面板中，单击 " **行业更新** " 选项卡。
3. 选择 " **发送每日电子邮件更新** " 以向你的用户发送电子邮件。
4. 若要使用户能够自定义他们在电子邮件更新中收到的新闻，请选择 " **允许用户自定义其自己的主题**"。

## <a name="bing-homepage"></a>必应主页

您可以自定义 Bing 主页以包含有关您的行业的新闻。 在 Microsoft 365 管理中心，转到 "**设置**  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"。 在 " **新闻** " 面板中，单击 " **bing 主页** " 选项卡，然后选择 " **在 Bing 主页上包含"**。 行业新闻显示在组织中的个性化信息下。

## <a name="microsoft-edge-new-tab-page"></a>Microsoft Edge 新选项卡页

当用户使用有效的工作或学校帐户登录 Microsoft Edge 时，他们可以查看个性化新闻并阅读内部 SharePoint 网站中的故事。

1. 在 Microsoft 365 管理中心，转到 "**设置**" "  >  **组织设置**  >  **服务**  >  [新闻](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)"
2. 在 " **新闻** " 面板中，单击 " **Microsoft Edge 新建选项卡页**"。
3. **在 "新建" 选项卡页上选择 "允许 Office 365 内容**"。 启用后，用户可以自定义新的选项卡，以显示 Office 365 中的信息，包括建议的和最近使用的文件，以及频繁使用的 SharePoint 网站和其他信息。
4. **在 "新建" 选项卡页上选择 "显示公司信息和行业新闻**"。 您的组织和行业的新闻文章将针对选择在新选项卡上看到 Microsoft 新闻的用户显示。
5. 在 **"新建选项卡" 页上选择 "允许 SharePoint 新闻** "，以在新选项卡页上包含在组织的内部 SharePoint 新闻网站上发布的内容。
