# <a name="manage-messages-in-message-center"></a>在消息中心管理邮件

消息中心帖子是有关新功能和已更改功能的重要公告。 本文提供有关可以使用邮件执行什么操作以帮助在组织中进行变更管理的信息。
  
## <a name="read-a-message"></a>阅读邮件

若要转到消息中心，请转到管理中心，然后选择 [](https://go.microsoft.com/fwlink/p/?linkid=2024339)**"运行状况**  >  **消息中心"。** 

选择邮件标题以在阅读窗格中打开它。 每封邮件都包含更改摘要、更改可能如何影响您的声明、有关组织如何为更改做好准备的信息，以及 (在大多数情况下) 一个可在其中了解详细信息的其他信息链接。 即使邮件过期，其他信息仍然可用。

## <a name="give-feedback-on-a-post"></a>在帖子上提供反馈

在消息中心，可以选择一条消息以查看详细信息。

如果要提供有关邮件的反馈，请选择"喜欢"或"**喜欢**"图标，在出现的文本框中提供反馈。 请勿提供任何个人信息。 可以选择选择 **"可以就** 此反馈联系我"，然后选择"**发送"。**

## <a name="share-a-message"></a>共享邮件

看到其他人需要处理的消息？ 可以通过电子邮件与任意用户共享邮件内容：
  
1. 选择邮件将其打开，然后选择"**共享"。**
  
2. 若要共享邮件，请输入最多两个电子邮件地址，用冒号分隔。 可以发送给个人并分组电子邮件地址。 或者，可以选择在电子邮件中接收邮件副本 (邮件将转到主电子邮件地址) 或添加个人邮件，为收件人提供更多上下文。
  
3. 选择 **"** 共享"发送电子邮件。
    
## <a name="get-a-link"></a>获取链接

需要跟进其他管理员以确保他们了解更改并采取措施？ 例如，可以生成链接以在电子邮件或即时消息中共享，该链接可将用户直接连接到该邮件。 你与共享链接的人必须是你的组织的 Microsoft 365 中的管理员，否则他们无法访问链接的邮件。

::: moniker range="o365-worldwide"

1. 选择邮件将其打开。

2. Select the **Copy link to Clipboard** (link) icon.

3. 允许网页访问剪贴板。 直接链接将复制到剪贴板。

4. 使用 Ctrl+V 或右键单击并选择 **"** 粘贴"插入链接。

::: moniker-end

::: moniker range="o365-germany"

1. 选择邮件将其打开。

2. 从 **操作** \> **栏中选择"更多** 获取链接"。

3. 允许网页访问剪贴板。 直接链接将复制到剪贴板。

4. 使用 Ctrl+V 或右键单击并选择 **"** 粘贴"插入链接。

::: moniker-end

::: moniker range="o365-21vianet"

1. 选择邮件将其打开。

2. 从 **操作** \> **栏中选择"更多** 获取链接"。

3. 允许网页访问剪贴板。 直接链接将复制到剪贴板。

4. 使用 Ctrl+V 或右键单击并选择 **"粘贴** "插入链接。

::: moniker-end

## <a name="read-and-unread-states"></a>读取和未读状态

消息中心中未读的任何邮件都将以粗体显示。 打开邮件会标记为已读。 可以将邮件标记为未读。

::: moniker range="o365-worldwide"

1. 在消息中心的主页上，选择一条消息。

2. 选择 **邮件顶部的" (** 信封) 标记为未读邮件" 图标。
  
::: moniker-end

::: moniker range="o365-germany"

1. 选择邮件将其打开。

2. 选择 **"** \> **更多标记为未读"。**

3. 关闭邮件。

::: moniker-end

::: moniker range="o365-21vianet"

1. 选择邮件将其打开。

2. 选择 **"** \> **更多标记为未读"。**

3. 关闭邮件。

::: moniker-end

## <a name="archive-and-restore"></a>存档和还原

如果您看到一条与自己不相关的邮件，或者您可能已经对邮件采取行动，您可以存档邮件以将其从"消息中心"视图中删除。 在消息中心看到的视图特定于你的用户帐户，因此从你的视图中存档它不会影响其他管理员。

::: moniker range="o365-worldwide"

- 在消息中心的主页上，选择一封邮件，然后选择邮件列表上方的"存档"。

- 打开邮件， **然后选择邮件底部的** "存档"。

需要返回存档的邮件？ 没关系。
  
1. 选择 **邮件中心顶部的** "存档邮件"选项卡。 将显示存档邮件的列表。 

2. 选择邮件，选择 **"还原**"，邮件将还原为活动邮件。

::: moniker-end

::: moniker range="o365-germany"

有两种方法可以存档邮件。
  
- 选择邮件 ID 右侧出现的 X。 邮件将从列表中删除。 或者，

- 打开邮件， **然后选择邮件顶部的** "存档"。

需要返回已取消的消息？ 没关系。
  
1. 从 **"视图"下拉菜单** 中选择"存档邮件"筛选器。 将显示存档邮件的列表。

2. 选择 **"** 邮件 ID 的右侧还原"，邮件将还原为活动邮件。

::: moniker-end

::: moniker range="o365-21vianet"

有两种方法可以存档邮件。
  
- 选择邮件 ID 右侧出现的 X。 邮件已从列表中删除。 或者，

- 打开邮件， **然后选择邮件顶部的** "存档"。

需要返回存档的邮件？ 没关系。
  
1. 从 **"视图"下拉菜单** 中选择"存档邮件"筛选器。 将显示存档邮件的列表。

2. 选择 **"** 邮件 ID 的右侧还原"，邮件将还原为活动邮件。

::: moniker-end

## <a name="track-your-message-center-tasks-in-planner"></a>在 Planner 中跟踪你的消息中心任务

有关 Microsoft 365 服务更改的许多可操作信息都会传送到 Microsoft 365 消息中心。 跟踪哪些更改需要完成任务、何时以及由谁执行以及跟踪每个任务完成可能比较困难。 你可能还想记下某些内容并标记它以便以后查看。 当你将邮件从 Microsoft 365 管理中心同步到 Microsoft Planner 时，你可以执行所有这些操作和更多操作。 有关详细信息，请参阅在 [Planner 中跟踪邮件中心任务](https://docs.microsoft.comoffice365/planner/track-message-center-tasks-planner)。


有关消息中心的概述，请参阅 [Microsoft 365 中的邮件中心](message-center.md)。 或者，若要了解如何设置语言首选项以启用消息中心帖子的机器翻译，请参阅消息中心帖子 [的语言翻译](language-translation-for-message-center-posts.md)。 如果你想要编程获取实时服务运行状况信息和消息中心通信的替代方法，请参考 [Microsoft 365 服务通信 API 概述](https://go.microsoft.com/fwlink/p/?linkid=848507)。
