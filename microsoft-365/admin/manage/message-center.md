# <a name="message-center"></a>消息中心


若要跟踪即将进行的更改，包括新功能和已更改的功能、计划的维护或其他重要通知，请转到 <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">消息中心</a>。 
  
打开消息中心：

::: moniker range="o365-worldwide"

- 在管理中心，转到 **"运行状况** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">消息中心"。</a>

::: moniker-end

::: moniker range="o365-germany"

- 在管理 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">中心</a>，转到 **"运行状况** > **消息中心"。**

::: moniker-end

::: moniker range="o365-21vianet"
 
- 在管理 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">中心</a>，转到 **"运行状况** > **消息中心"。**

::: moniker-end

还可以在移动设备上使用 [Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=627216) 管理应用来查看消息中心，这是使用推送通知保持最新状态很好的方法。

以下是您将在每个列中看到的信息的快速概述。 选择任意列标题以更改排序顺序。
  
|**列**|**说明**|
|:-----|:-----|

### <a name="major-updates"></a>主要更新

可以通过选择消息中心顶部的"高重要性"选项卡来查看主要更新。

需要操作时，至少提前 30 天通知主要更新，可能包括：
  
- 对日常工作效率的更改，如收件箱、会议、共享和访问

- 对可能影响自定义功能的主题、Web 部件和其他组件的更改

- 增加或减小到可见容量，如存储、规则数、项数或持续时间

- 对产品品牌所做的更改，可能：

  - 导致最终用户混淆，

  - 导致对技术支持流程和参考资料的更改，或

  - 更改 URL

- 新服务或应用程序

- 需要管理员操作的更改 (防止或修复) 

- 数据存储位置的更改
  
### <a name="preferences"></a>首选项

如果管理分布在整个组织中，你可能不希望或不需要查看有关所有 Microsoft 365 服务的帖子。 每个管理员可以：

- 设置控制邮件中心中显示的邮件的首选项。
- 筛选邮件
- 设置电子邮件首选项以接收所有邮件的每周摘要、仅主要更新的电子邮件和数据隐私邮件的电子邮件。  

::: moniker range="o365-worldwide"

1. 选择 **消息中心** 顶部的"编辑首选项"。

2. 确保选中了要监视的每个服务的复选框。 清除要筛选出消息中心视图的服务的复选框。

3. 摘要电子邮件默认打开，并发送到主电子邮件地址。 若要停止接收每周摘要，请清除"发送我的邮件的每周 **电子邮件摘要** "复选框。 <br/><br/>主要更新电子邮件通知是一个单独的控件。 如果要接收有关主要更新的电子邮件通知，请验证是否选择了"向我发送主要 **更新的电子邮件** "。 清除此复选框以停止获取有关主要更新的电子邮件。 <br><br/>选中"向我发送数据隐私邮件"旁边的复选框，以接收数据隐私邮件的单独电子邮件通知 (数据隐私邮件不包含在每周摘要) 。 <br><br/>可以选择或清除主电子邮件地址，但无法更改它。 若要指定数据隐私邮件电子邮件发送到的其他电子邮件地址，请选中"其他电子邮件地址"复选框，然后输入要发送电子邮件的其他电子邮件地址。 如果超过两个人应获取数据隐私电子邮件，请输入 Microsoft 365 组或通讯组列表的电子邮件地址。

4. 选择 **"保存** "以保留更改。
