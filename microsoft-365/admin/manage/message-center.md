# <a name="message-center"></a>消息中心


若要跟踪即将进行的更改，包括新的和已更改的功能、计划的维护或其他重要通知，请转到 "<a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">消息中心</a>"。 
  
打开 "消息中心"：

::: moniker range="o365-worldwide"

- 在管理中心中，转到**运行状况** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">消息中心</a>。

::: moniker-end

::: moniker range="o365-germany"

- 在<a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">管理中心</a>中，转到**运行状况** > **消息中心**。

::: moniker-end

::: moniker range="o365-21vianet"
 
- 在<a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">管理中心</a>中，转到**运行状况** > **消息中心**。

::: moniker-end

您还可以在移动设备上使用[Microsoft 365 管理应用](https://go.microsoft.com/fwlink/p/?linkid=627216)来查看消息中心，这是保持最新的推送通知的绝佳方式。

### <a name="messages"></a>邮件

消息中心以表格格式显示所有活动邮件的视图。 默认情况下，它会在列表顶部显示最新消息。 您可以选择一个选项卡以查看**所有活动邮件**、**重要性较高**的邮件、**未读邮件**和已**清除的邮件。** 您可以使用 "**筛选器**" 下拉菜单查看特定邮件类别中的所有邮件。

以下是您将在每个列中看到的信息的快速概述。 选择任意列标题以更改排序顺序。
  
|**列**|**说明**|
|:-----|:-----|
|复选标记  <br/> |选中列标题行中的复选标记将选择当前显示的所有邮件。 选中一个或多个邮件旁边的复选标记可使您对这些邮件执行操作。  <br/> |
|应用程序图标 <br/> |图标指示要应用邮件的应用程序。<br/> |
|邮件标题  <br/> |邮件标题是即将发生的更改的简短说明。 如果未显示完整标题，将光标悬停在该标题上，整个标题将显示在弹出框中。  <br/> |
|更多选项 <br/> |更多选项使您可以消除邮件，将其标记为已读或未读，或与其他管理员共享。若要还原已清除的邮件，请选择 "**消除的邮件**" 选项卡，选中邮件旁边的复选标记，然后选择 "**还原**"。 <br/> |
|主要更新 <br/> |此列中的感叹号表示主要更新。 <br/> |
|操作依据  <br/> |如果我们要做的更改要求您在某个截止时间执行某一操作，我们将在此处仅包含日期。 由于我们很少使用**Act**列，如果你在这里看到了一些内容，你应该格外关注它。  <br/> |
|类别  <br/> | 邮件通过以下三种类别之一进行标识： <br/><br/> **防止或解决问题**：通知您影响组织的已知问题，并且可能要求您采取措施以避免服务中断。 阻止或修复问题与服务运行状况消息不同，因为它们会提示您主动避免出现问题。 <br/> <br/> **规划更改**：通知您对 Microsoft 365 所做的更改，可能需要您采取措施避免服务中断。 例如，我们将告知你对系统要求的更改或有关要删除的功能的更改。 我们尝试为任何更改至少提供30天的通知，以要求管理员将服务保持正常运行状态。 <br/> <br/> **及时了解**：告诉你在你的组织中打开的新功能或更新功能。 这些功能通常先在[Microsoft 365 路线图](https://go.microsoft.com/fwlink/?linkid=2070821)中宣布。 <br/><br/>此外，还可以根据我们的服务级别协议来了解计划的维护。 计划的维护可能导致停机，你或你的用户不能访问 Microsoft 365、特定功能或服务（如电子邮件或 OneDrive for Business）。  <br/> |
|上次更新  <br/> |邮件发布或最后更新的日期。  <br/> |
|邮件 ID  <br/> |Microsoft 按邮件 ID 跟踪我们的邮件中心文章。 如果你想要提供反馈，或者如果你对特定邮件的支持进行了支持，可以参考此 ID。  <br/> |

若要了解有关可以使用邮件执行的操作的详细信息，请转到[管理邮件中心中的邮件](manage-messages.md)。
  
### <a name="major-updates"></a>主要更新

可以通过选择邮件中心顶部的 "**高重要性**" 选项卡来查看主要更新。

当需要采取措施时，将提前30天内传递主要更新，其中可能包括：
  
- 对每日工作效率（如收件箱、会议、委派、共享和访问）的更改

- 对主题、web 部件和其他可能影响自定义功能的组件的更改

- 增大或减小可见容量，如存储、规则数、项目或持续时间

- 对产品品牌的更改可能会：

  - 导致最终用户混淆，

  - 导致技术支持进程和参考资料的变化，或

  - 更改 URL

- 新服务或应用程序

- 需要管理员操作的更改 (独占预防或修复问题) 

- 对数据存储位置的更改
  
### <a name="preferences"></a>个性

如果管理分布在您的组织中，您可能不希望或不需要查看有关所有 Microsoft 365 服务的帖子。 每个管理员都可以：

- 设置用于控制在邮件中心中显示哪些邮件的首选项。
- 筛选邮件
- 设置电子邮件首选项以接收所有邮件的每周摘要、仅限主要更新的电子邮件和数据隐私邮件的电子邮件。  

::: moniker range="o365-worldwide"

1. 选择 "邮件中心" 顶部的 "**编辑首选项**"。

2. 确保为要监视的每个服务选中 "" 复选框。 清除要从 "消息中心" 视图中筛选出的服务所对应的复选框。

3. 默认情况下会打开摘要电子邮件，并将其发送到您的主电子邮件地址。 若要停止接收每周摘要，请清除 "**发送我的邮件的每周电子邮件摘要**" 复选框。 <br/><br/>主要更新的电子邮件通知是一个单独的控件。 如果您想要收到有关主要更新的电子邮件通知，请验证是否已选择 "**向我发送主要更新的电子邮件**"。 清除此复选框可停止获取有关主要更新的电子邮件。 <br><br/>选中 "**向我发送数据隐私邮件的电子**邮件" 旁边的复选框，以接收数据隐私邮件的单独电子邮件通知 (数据隐私邮件不包含在每周摘要) 中。 <br><br/>你可以选择或清除你的主电子邮件地址，但不能更改它。 若要指定向其发送数据隐私邮件的其他电子邮件地址，请选中 "**其他电子邮件地址**" 复选框，并输入要向其发送电子邮件的其他电子邮件地址。 输入 Microsoft 365 组或通讯组列表的电子邮件地址（如果有两个以上的人需要获取数据隐私电子邮件）。

4. 选择 "**保存**" 以保留更改。
  
::: moniker-end
