# <a name="share-calendars-with-external-users"></a>与外部用户共享日历

有时，用户必须安排与组织外部人员的会议。 为了简化查找常见会议时间的过程，Microsoft 365 使你能够向这些人员提供日历。 这些用户需要查看组织中用户的忙/闲时间，但没有 Microsoft 365 组织的用户帐户。

可以在 Microsoft 365 管理中心为组织中所有用户启用日历共享。 启用共享后，你的用户可以Outlook Web App组织内外的任何人共享其日历。 组织内部人员可以查看共享日历及其自己的日历。 组织外部的人员将收到可用于查看日历的 URL。 组织的用户决定何时共享和共享多少。

> [!NOTE]
> 如果要与使用 Exchange Server 2013（内部部署解决方案）的组织共享日历，Exchange 管理员需要设置与云的身份验证关系。 这称为联合身份验证，必须满足最低软件要求。 有关详细信息，请参阅[共享](https://technet.microsoft.com/library/dd638083%28v=exchg.150%29.aspx)。
  
## <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>使用 Microsoft 365 管理中心启用日历共享

1. 在管理中心，转到"**设置** \> **组织设置"。**

2. 在"**服务"** 选项卡上，选择 **"日历"。**
  
3. 在 **"日历** "页上，选择是否允许用户与拥有 Microsoft 365 或 Exchange 的组织外部人员共享其日历。 选择是否允许匿名用户 (凭据的用户) 通过电子邮件邀请访问日历。

4. 选择要提供给用户的日历信息类型。 你可以允许所有信息，或仅将信息限制为时间或时间、主题和位置。

## <a name="invite-people-to-access-calendars"></a>邀请其他人访问日历

启用共享后，日历所有者可以将邀请扩展到特定用户。 有关说明，请参阅[在 Outlook Web App 中共享日历](https://support.microsoft.com/office/7ecef8ae-139c-40d9-bae2-a23977ee58d5)。
