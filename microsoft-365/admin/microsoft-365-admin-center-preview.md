# <a name="about-the-microsoft-365-admin-center"></a>关于 Microsoft 365 管理中心

::: moniker range="o365-worldwide"

Microsoft 365 管理中心专为 IT 团队构建，是管理 Microsoft 365 服务的一种简化方式。 管理中心根据角色或组织的独特需求提供定制体验，提高日常任务的效率，并提供可操作见解，帮助你做出数据驱动的决策，为用户提供更好的体验。

Microsoft 365 管理中心是所有 Microsoft 365 管理员的常见入口点，可在以下位置访问 [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) 。 专家工作区（如安全或设备管理）允许更精细的控制。 有关管理中心如何协同工作的信息，请参阅本文稍后介绍的特定类型的 IT 角色和其他工作区（如安全、设备管理或 [Exchange）](#what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange) 如何操作。

::: moniker-end

### <a name="turn-on-targeted-release"></a>打开定向发布

1. 登录 [位置](https://admin.microsoft.com)admin.microsoft.com，转到导航窗格，然后选择 **"设置** \> **组织配置文件"。**

2. 转到"**发布首选项"** 卡，然后选择"编辑 **"。** 
    
3. 为选定 **用户选择所有人的** 定向发布 **或定向发布**。 如果选择所选用户的定向发布，请确保将管理员帐户 (和组织中的任何其他想要参与) 管理员添加到所选用户列表中。
    
### <a name="admin-center-feedback"></a>管理中心反馈

在管理中心内，可以通过选择每个页面底部的"需要帮助？"按钮旁边的"提供反馈"，为Microsoft 提供有关你的体验的反馈。 告诉我们您喜欢什么以及我们可以做哪些更好的工作。 此外，你可能会时时收到弹出式调查，询问总体印象或新发布的特定体验。 您还可以在本文末尾选择此信息有用吗 **？**

### <a name="switch-to-the-new-admin-center-and-back-again"></a>切换到新的管理中心，然后再次返回

当新的管理中心处于预览阶段时，可以使用位于管理中心主页顶部的切换功能，在新的管理中心与旧管理中心之间来回切换。 新的管理中心功能全面，具有旧管理中心的所有功能。

若要尝试新的管理中心，请切换为<b>"试用预览"。</b> 然后，切换将显示<b>预览。</b> 下次你登录到管理中心时，我们将记住你的选择，并登录到新的管理中心。

若要返回到使用旧管理中心，请打开从 <b>预览</b> 到关闭位置的切换。 下次你登录到管理中心时，我们将登录到旧管理中心。

::: moniker-end

## <a name="can-i-do-everything-in-the-new-admin-center-that-i-can-do-in-the-old-admin-center"></a>我能否在新的管理中心中执行可在旧管理中心中执行的所有任务？

Microsoft 365 是一个完整的智能解决方案，包括 Office 365、Windows 10 和企业移动性 + 安全性，使每个人都能够发挥创造力并安全地协同工作。 以下 Microsoft 365 订阅可在管理中心内立即试用或购买：

## <a name="i-found-a-bug-or-i-want-to-request-a-feature-enhancement-how-do-i-let-microsoft-know"></a>我发现了一个 Bug，或者想要请求功能增强功能。 如何让 Microsoft 知道？

我们喜欢倾听您的声音！ 报告 Bug 和共享反馈可帮助我们改进 Microsoft 365 管理中心。 若要提供反馈，请选择页面底部的"反馈"按钮，并使用表单将你的想法发送给我们。 如果你希望 Microsoft 365 管理中心团队中的某人跟进你的评论，请选中该复选框并确认您的电子邮件地址。 我们不能保证跟进每个反馈，但我们会尝试！ 
  
还可以在我们的 UserVoice 论坛中从管理中心外提供反馈。 可以使用此页面提出功能建议，其他论坛用户可以投票 [：UserVoice 论坛用于新的管理中心](https://go.microsoft.com/fwlink/?linkid=2024994)。

## <a name="what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange"></a>特定类型的 IT 角色和其他工作区（如安全性、设备管理或 Exchange）如何？

Microsoft 365 管理中心是管理 Microsoft 365 的所有团队和角色的常见入口点。 体验、信息和控件针对每个管理员和角色进行定制和自定义。 此外，专家工作区允许进行深入、精细的控制。 这些专家工作区包括 SharePoint、Teams &amp; Skype、Exchange、安全性、合规性、设备管理和 Azure Active Directory。 您可以在 Microsoft 365 管理中心的导航窗格中找到专家工作区 [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) 。
