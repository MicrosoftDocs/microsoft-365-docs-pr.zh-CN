# <a name="compare-ways-to-block-access"></a>比较阻止访问的方法

当员工离开你的组织时，如果有好的条款或糟糕，你需要阻止其访问 Microsoft 365。 下面是可以执行此操作的几种方法。
  
||||
|:-----|:-----|:-----|
|**阻止访问的方法** <br/> |**定义** <br/> |**最佳实践** <br/> |
|阻止登录  <br/> |阻止用户访问 Microsoft 365 的一种方法是将其登录状态更改为 **"已阻止登录"**。 这样，他们就可以阻止他们的计算机和移动设备登录到 Microsoft 365，尽管他们仍可以查看以前下载或同步的电子邮件和文档。 如果您使用的是 Blackberry 企业服务，则还可以禁用其访问。  <br/> |当员工计划离开组织或计划进行长期请假时使用。  <br/> |
|重置用户密码  <br/> |阻止用户访问 Microsoft 365 的另一种方法是重置其密码。 这样，他们仍可以查看以前下载或同步的电子邮件和文档，但不能使用其帐户。 然后，您可以将其登录并将密码更改为您选择的一个。  <br/> |在员工突然且永久退出时使用，并且您认为业务数据有问题。  <br/> |
|删除所有分配的许可证  <br/> |另一种方法是删除分配给用户的任何 Microsoft 365 许可证。 这样可以防止用户使用应用程序和服务，如 Office 套件、适用于 web 的 Office 应用程序、Yammer 和 SharePoint Online。 他们仍可以登录但不能使用这些服务。  <br/> |当您认为此用户不再需要访问 Microsoft 365 中的特定功能时，请使用此功能。  <br/> <br> **重要说明：** 删除许可证时，将在30天内删除用户的邮箱。
