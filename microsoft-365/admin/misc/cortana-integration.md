# <a name="cortana-in-office-365"></a>Office 365 中的 Cortana

Cortana 是一种基于云的数字助理以及 Microsoft 365 和 Office 365 服务，可跨设备和 Microsoft 服务工作。 Cortana 可以使用存储在 Microsoft 365 和 Office 365 中的信息来帮助组织中的人员保持最新状态，并获取有关会议、文档和联系人的见解、建议的任务和帮助。
  
## <a name="info-for-admins"></a>适用于管理员的信息

合规性是 Microsoft 向企业客户提出的承诺。 [了解有关 Microsoft 合规性框架的详细信息。](https://go.microsoft.com/fwlink/p/?LinkId=2109173)

与其他 Microsoft 365 和 Office 365 服务保持一致，合规性的 Cortana 功能受保护且受保护，包括一系列承诺，其中包括保护用户数据免受意外丢失、改变、未经授权的泄露或访问权限，或非法销毁。 所有其他 Cortana 功能（即 Cortana 可选的已连接服务）都服从[Microsoft 服务协议](https://go.microsoft.com/fwlink/p/?LinkId=2109174)和[microsoft 隐私声明。](https://go.microsoft.com/fwlink/p/?LinkId=2109175)

Cortana 服务分为两个数据类别、**合规**和**可选的已连接服务**，您可以对它们进行控制。

## <a name="turn-off-cortana-optional-connected-services"></a>关闭 Cortana 可选的已连接服务

可以为组织中的员工关闭 Cortana 可选的已连接服务。 这将在 Windows 和 Cortana 移动应用上的 Cortana 中禁用 Cortana 可选连接的服务。 这包括 Cortana 提醒、列表、任务和其他功能。 兼容类别（即 Cortana OST 体验）中的服务（如 Cortana 的简报电子邮件）和在 Outlook mobile 中播放我的电子邮件将保持活动状态。

1. 在 Microsoft 365 管理中心，选择 "**设置** > **设置**"，然后选择 " **Cortana**"。

4. 选中 "**允许 cortana 可选连接体验" 复选框，以使用组织的 Microsoft 托管数据**来启用或禁用 Cortana 连接体验。

5. 选择 "**保存更改**"。

## <a name="turn-off-cortana-ost-experiences"></a>关闭 Cortana OST 体验

您的组织的员工可以通过执行以下步骤逐个选择拒绝 Cortana OST 体验。

1. 打开 Outlook mobile。

2. 转到 "**设置**"。
  
3. 选择 **"播放我的电子邮件"**。

4. 在要禁用的帐户上将切换移到 "关闭"。

### <a name="briefing-email"></a>简报电子邮件

禁用任务栏上的 Cortana 不会禁用 Cortana 的简报电子邮件。 员工必须单独取消订阅。 组织中的个人可以通过在邮件的页脚中选择 "**取消订阅**" 来退出 Cortana 的简介电子邮件。
