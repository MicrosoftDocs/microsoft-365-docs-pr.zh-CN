# <a name="cortana-in-microsoft-365"></a>Microsoft 365 中的 Cortana

Cortana 和你的个人工作效率助手提供了 AI 功能体验，以节省时间并重点关注最重要的问题。 Cortana 将帮助您的用户在工作和生命中提高其全天的个人工作效率。 使用有效的工作或学校帐户登录后，用户可以使用 Microsoft 365 体验中的 Cortana 获取基于云的协助服务，以满足 Office 365 的企业级隐私、安全性和合规性承诺 ( "Cortana enterprise 服务" ) 。 

作为个人的工作效率助手，Cortana 旨在提供安全且安全的功能，在 Office 365 数据（如电子邮件、文件、聊天等）上进行安全和安全的处理和原因，以节省时间、提高效率，并提高用户的工作效率。

向前发展，我们将把 Cortana 集中在企业的工作效率上。

- 与其他 Office 365 服务保持一致，Cortana enterprise 服务符合相同的企业级隐私、安全性和合规性承诺在 [联机服务条款 (OST) ](https://www.microsoft.com/licensing/product-licensing/products)中反映出来。

- 新的 Microsoft 365 体验（如简报电子邮件和播放我的电子邮件）将使用 Cortana 企业服务启用，并完全遵守这些承诺。 这些功能目前适用于在美国地区拥有邮箱的用户的英语 (美国) 市场，以及针对具有美国邮箱的多国公司的员工。 有关查找使用位置的详细信息，请参阅 [查看帐户的其他属性值](https://docs.microsoft.com/microsoft-365/enterprise/view-user-accounts-with-microsoft-365-powershell?view=o365-worldwide#view-additional-property-values-for-accounts)。

- 用户可以通过 Windows 10 (版本2004及更高版本) 中的 Cortana 和客户端应用程序（例如，适用于 iOS 和 Android 的 Outlook for iOS）连接到此处介绍的 Cortana enterprise service，具体取决于许可条款。 

- 现有消费者体验，包括 Windows 10 中的 Cortana (版本1909和早期版本的) ，以及 iOS 和 Android 上的 Cortana 应用均受 [Microsoft 服务协议](https://www.microsoft.com/licensing/product-licensing/products) 和 [microsoft 隐私声明](https://privacy.microsoft.com/privacystatement) (（请参阅) 下面的 "现有服务" 使用者 "部分）。 这些术语还将管理在使用其使用者凭据登录时向用户提供的 Cortana 企业服务。

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Cortana 企业服务处理哪些数据？ 

Cortana enterprise 服务处理来自用户的查询、满足用户请求所需的 Office 数据以及由 Microsoft 系统生成的用于运行服务的其他遥测。 Cortana enterprise 服务收集的数据包括用户的口头查询的文本表示形式 (即从语音识别) 的转录。 此文本数据为客户数据，并根据  [在线服务条款](https://www.microsoft.com/licensing/product-licensing/products)进行管理。 它仅用于开发和改进与在线服务条款一致的机器学习模式。

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>在 Cortana 企业服务中，客户数据的管理模型是什么？

与其他 Office 365 服务保持一致，Cortana 企业服务也受到保护，并受 [在线服务条款](https://www.microsoft.com/licensing/product-licensing/products)的制约。 这包括保护客户数据以防意外丢失、篡改、未经授权泄露或访问或非法销毁的承诺集。 客户数据也受严格访问限制的限制。 Microsoft 仅使用客户数据提供商定的服务以及与这些服务兼容的目的。 有关详细信息，请参阅下表。

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Microsoft 365 中的 Cortana 企业服务体验

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Windows 10 中的 Cortana (版本2004及更高版本) 

Windows 10 中的 Cortana 应用可帮助用户在 Microsoft 365 中快速获取信息，使用类型化或口头查询连接人员、查看日历、设置提醒等。

Windows 中的 Cortana 可帮助用户更好地管理其日程和任务。 他们可以在 Microsoft 中添加到他们的列表中，查找本地信息、获取定义，并使用 Bing 搜索跟踪最新的新闻、天气和财务信息。

Windows 10 版本2004及更高版本中的 cortana 符合对 Cortana 企业服务的相同企业级隐私、安全性和合规性承诺，如 [在线服务条款 (OST) ](https://www.microsoft.com/licensing/product-licensing/products)中所示。

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>如何在 Windows 10 中选择拒绝 Cortana

管理员可以使用 Experience\AllowCortana MDM 策略或通过组策略（计算机配置 \ 管理模板 \Windows 组件 Components\Search\Allow Cortana）在 Windows 10 中为其组织配置 Cortana。

从 Windows 10 版本2004开始，Cortana 是预先安装了 Windows 的通用 Windows 平台 (UWP) 应用，并定期通过 Microsoft Store 进行更新。 若要接收 Cortana 的最新更新，你将需要 [通过 Microsoft Store 启用更新](https://docs.microsoft.com/windows/configuration/stop-employees-from-using-microsoft-store)。

[了解有关 Windows 10 中 Cortana 的详细信息](https://docs.microsoft.com/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>团队中的 Cortana 语音协助

团体移动应用和 Microsoft 团队中的 Cortana 语音协助在 Microsoft 团队显示设备中，Microsoft 365 企业版用户可以使用语音自然语言优化通信、协作和会议相关任务。 用户可以通过选择位于工作组移动应用程序右上角的麦克风按钮，或者说出 Microsoft 团队显示中 &#8220;Cortana&#8221; 来与 Cortana 进行对话。 若要在无人参与和转到时快速连接团队，用户可以说出诸如 &#8220;呼叫 Megan&#8221; 的查询或 &#8220;向我的下一个会议&#8221; 发送消息。 用户也可以通过说 &#8220;加入我的下一个会议&#8221; 并使用语音协助共享文件、查看其日历等，来加入会议。 这些语音协助体验使用 Cortana 企业级服务提供，这些服务完全符合 Office 365 的隐私、安全性和合规性承诺，在 [在线服务条款 (OST) ](https://www.microsoft.com/licensing/product-licensing/products)中反映出来。

**管理控制**

默认情况下，将为租户启用 Cortana 语音协助。 管理员可以通过策略 (TeamsCortanaPolicy) 控制其租户中的哪些用户可以在团队中使用 Cortana voice 协助。 可以在用户帐户级别或租户级别设置此策略。 管理员还可以使用此策略控制中的 CortanaVoiceInvocationMode 字段来确定 Cortana 是否已禁用、是否已启用按下按钮调用，或启用了唤醒 word (调用（如 Microsoft 团队显示) ）。 请注意，在 Microsoft 365 企业用户首次发布英语版时，工作组移动应用程序将不支持唤醒 word 激活，但将来会受到支持。

**用户控件**

通过单击麦克风按钮，单个用户可以在 "团队移动应用" 中试用 Cortana voice 协助。 他们可以通过简单地说出 &#8220;Cortana，在 Microsoft 团队显示设备上试用 Cortana 语音协助。 &#8221; 他们还可以控制是否通过 "工作组移动应用" 中的设置或 Microsoft 团队显示中的设置为设备启用了 Cortana： 

1. 打开 "团队移动应用程序"，或转到 Microsoft 团队显示的 "环境 (主页) " 屏幕。

2. 在团队移动应用中，转到 " **设置** "。 在 "Microsoft 团队显示" 中，选择 "用户头像"，然后选择 "设置"。 如果启用 Cortana，则说 &#8220;Cortana，请转到 "设置"。 &#8221;

3. 选择 " **Cortana** "。

4. 将切换方式移到 **"打开** " 或 " **关闭** "，具体取决于您是否希望在设备上进行 Cortana 语音协助。

[了解有关团队中 Cortana 语音协助的详细信息](https://docs.microsoft.com/microsoftteams/cortana-in-teams)

### <a name="play-my-emails"></a>播放我的电子邮件

在通过 Outlook mobile) 连接到我的电子邮件 (的情况下，用户通过电话、耳机或连接的音频设备上的扬声器收听新邮件，从而获得语音驱动的无人干预体验。 用户可以请求 Cortana 高声阅读其最近的电子邮件，并要求 Cortana 执行诸如标记、存档、删除和跳过邮件等操作。 此功能对在 commuting、多任务或旅途中追赶你的电子邮件尤其有用。 当用户在 "播放我的电子邮件" 中与 Cortana 交谈时，语音音频请求将直接转到 "Cortana enterprise 服务"。 用户电子邮件的语音读出文本在 Office 365 云中进行处理。 在此过程中，不会在用户的移动设备上处理任何 Office 365 数据，也不会保存任何电子邮件数据。  (（例如 "标记为已读"、"下一步"、"标志" 等）的一条记录将根据 Microsoft [Online Services 条款](https://www.microsoft.com/licensing/product-licensing/products)中的数据保护条款保留 ) 。

当电子邮件受到保护并在阅读邮件前短暂停顿以向用户提供足够的时间来暂停播放或跳转到下一封邮件时，Cortana 将发出呼叫。 与私人电话呼叫类似，用户在可能会 overheard 机密信息的位置启动播放时应小心谨慎。 在这些情况下，建议在使用 Outlook mobile 中的 "在电子邮件中播放电子邮件" 时，在适当的环境中，组织的员工磨损耳机。

### <a name="opt-out-of-play-my-emails"></a>选择不播放我的电子邮件

使用以下步骤，个人可以选择不播放我的电子邮件。

1. 打开 Outlook mobile。

2. 转到 " **设置** "。
  
3. 选择 **"播放我的电子邮件"** 。

4. 在要禁用的帐户上将切换移到 "关闭"。

[了解有关播放我的电子邮件的详细信息](https://support.microsoft.com/help/4558256)

### <a name="briefing-email"></a>简报电子邮件

Cortana 发送个性化的简报电子邮件，其中包含任务和承诺，您可以轻松地将其标记为 " **完成** " 或 "安排焦点" 以使其完成。 它还包括您的会议和每天相关文档的摘要。 Cortana 从用户的电子邮件中提取信息，并将其存储在 Exchange Online 邮箱中，直到它合并到简报电子邮件中。 任何时候都不能在 Exchange Online 邮箱之外访问个人数据。

### <a name="how-to-opt-out-of-briefing-email"></a>如何选择不签出简报电子邮件

管理员可以使用 Exchange Online 中的 [PowerShell](https://docs.microsoft.com/briefing/be-admin) 为其组织配置简介。 通过在邮件的页脚中选择 " **取消订阅** "，个人可以退出 Cortana 的简报电子邮件。

[了解有关简介电子邮件的详细信息](https://support.microsoft.com/help/4558259)
