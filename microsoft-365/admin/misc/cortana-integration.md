# <a name="cortana-in-microsoft-365"></a>Microsoft 365 中的 Cortana

Cortana 是你的个人生产力助手，提供支持 AI 的支持体验，以节省时间并专注于对最重要情况的关注。 Cortana 可帮助用户在工作和生活中提高他们一天的个人效率。 使用有效的工作或学校帐户登录后，用户可在 Microsoft 365 体验中通过 Cortana 获得基于云的帮助服务，以满足 Office 365 企业级隐私、安全性和合规承 ("Cortana 企业服务") 。 

作为个人生产力助手，Cortana 旨在提供一些功能，提供能够安全、安全地处理和处理 Office 365 数据（如电子邮件、文件、聊天等）的功能，以节省时间、提高效率并提高用户的工作效率。

我们继续，我们侧重于企业工作效率。

- 与其他 Office 365 服务一致，Cortana 企业服务就是在线服务条款 OST 网站中反映出来一 ([一) 。 ](https://www.microsoft.com/licensing/product-licensing/products)

- 新的 Microsoft 365 体验（如简发电子邮件和播放电子邮件）将使用 Cortana 企业服务启用，并且完全遵从这些承文。 当前提供针对在美国 (邮箱的邮箱的用户以及在具有美国邮箱的跨国企业企业的员工的英语 (美国) 市场中这些功能。 有关查找使用情况位置的详细信息，请访问["查看帐户的其他属性值"。](https://docs.microsoft.com/microsoft-365/enterprise/view-user-accounts-with-microsoft-365-powershell?view=o365-worldwide#view-additional-property-values-for-accounts)

- 在 Windows 10 (版本 2004 及更高版本) 中，用户通过 Cortana 转到此处介绍的 Cortana 企业服务，也可以通过单独的许可条款连接到客户端应用程序（如 Outlook for iOS 和 Outlook for Android）。 

- Windows 10 (版本 1909 及更早版本) 中的 Cortana 以及 iOS 和 Android 上的 Cortana 受 [Microsoft](https://www.microsoft.com/licensing/product-licensing/products) 服务协议和 Microsoft [隐私](https://privacy.microsoft.com/privacystatement) 声明 (下面的"面向消费者的现有服务"部分的) 。 这些条款还管理在使用用户使用其消费者凭据登录时提供到用户的 Cortana 企业服务。

## <a name="what-data-is-processed-by-cortana-enterprise-services"></a>Cortana 企业服务处理哪些数据？ 

Cortana 企业服务处理来自用户的查询、履行用户请求所需的 Office 数据以及 Microsoft 系统生成的其他遥测来运行服务。 Cortana 企业服务收集的数据包括用户说出的传率查询的文本表示形式 (如来自语音识别的) 。 此文本数据是客户数据，并且系统基于联机服务  [条款进行管理](https://www.microsoft.com/licensing/product-licensing/products)。 它仅用于开发和改进符合联机服务条款的机器学习模型。

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-enterprise-services"></a>Cortana 企业服务中的客户数据的管理模型是什么？

与其他 Office 365 服务一致，Cortana 企业服务是受到保护的，并且还受 [联机服务条款的约义制](https://www.microsoft.com/licensing/product-licensing/products)约。 这包括一组承易针对客户数据避免意外丢失、更改、未经授权的泄露或访问或非法破坏的证明。 客户数据还受严格的访问限制的约。也会限制客户数据。 Microsoft 仅使用客户数据来提供商订的服务，并出于与这些服务兼容的目的。 有关详细信息，请参阅下表。

## <a name="cortana-enterprise-services-in-microsoft-365-experiences"></a>Microsoft 365 中的 Cortana 企业服务体验

### <a name="cortana-in-windows-10-version-2004-and-later"></a>Windows 10 中的 Cortana (版本 2004 及更高版本) 

Windows 10 中的 Cortana 应用帮助用户在 Microsoft 365 中快速获取信息，方法是使用键入的或微信的查询与人员连接、查看日历、设置提醒等。

Windows 中的 Cortana 可以帮助用户更好地管理他们的日程安排和任务。 他们可以添加到他们在 Microsoft To Do 中的列表中、查找本地信息、获取定义，以及通过必应搜索跟踪最新新闻、天气和财务信息。

Windows 10 版本 2004 及更高版本中的 Cortana 符合与 Cortana 企业服务相同的企业级隐私、安全性和合规性承受，如联机服务条款 [ (OST) 中所反映出来 ](https://www.microsoft.com/licensing/product-licensing/products)。

### <a name="how-to-opt-out-of-cortana-in-windows-10"></a>如何在 Windows 10 中选择退出 Cortana

管理员可以使用体验\AllowCortana MDM 策略或通过组策略在 Windows 10 中为其组织配置 Cortana："计算机配置"\"管理模板"\"Windows 组件"\"搜索"\"允许 Cortana"。

从 Windows 10 版本 2004 开始，Cortana 是一个与 Windows 一起预安装) 的通用 Windows 平台 (UWP 应用程序，并通过 Microsoft Store 定期更新。 若要接收 Cortana 的最新更新，需要通过 [Microsoft Store 启用更新](https://docs.microsoft.com/windows/configuration/stop-employees-from-using-microsoft-store)。

[了解有关 Windows 10 中的 Cortana 的详细信息](https://docs.microsoft.com/windows/configuration/cortana-at-work/cortana-at-work-overview)

### <a name="cortana-voice-assistance-in-teams"></a>Teams 中的 Cortana 语音帮助

使用 Teams 移动应用中的 Cortana 语音帮助，Microsoft 365 企业版用户能够使用说出自然语言简化通信、协作和与会议相关的任务。 用户可以通过单击 Teams 移动应用右上角的麦克风按钮与 Cortana 联系。 拨打电话查询（如"呼叫 Megan"或"向我的下一个会议发送邮件"），可与此人建立连接。 用户还可以通过说出"加入我的下一个会议"并使用语音帮助共享文件、查看日历等内容，来加入会议。 这些语音助手体验使用 Cortana 企业级服务提供，这些服务完全遵循 Office 365 的隐私、安全性和合规承务，如 [联机服务条款和 OST) 中 (补充 ](https://www.microsoft.com/licensing/product-licensing/products)。

**管理员控制**

默认情况下，将为租户启用 Cortana 语音帮助。 管理员可以通过 TeamsCortanaPolicy 策略控制其租户中的人员可将 Cortana 语音 (帮助) 。 可以在用户帐户级别或租户级别设置此策略。 管理员还可以在此策略控制中使用 CortanaVoiceInvocationMode 字段来确定 Cortana 是禁用的、仅通过推送按钮调用启用，还是启用时使用多次发言词调用 (这两种字段是支持它的) 。 请注意，在美国英语版 Microsoft 365 企业版用户初步版本的时，Teams 移动应用将不支持进行 Word 激活，但以后将支持该应用。

**用户控件**

单个用户可以通过单击"小部件"按钮在 Teams 移动应用中体验 Cortana 语音帮助。 他们还可以通过 Teams 移动应用中的设置来控制是否为其设备启用了 Cortana： 

1. 打开 Teams 移动应用。

2. 转到"**设置"。**

3. 选择**Cortana。**

4. 根据你是否希望在此设备上使用 Cortana 语音协助，将切换移动到开/关闭。

[了解有关 Teams 中的 Cortana 语音帮助的详细信息](https://support.microsoft.com/en-us/office/using-cortana-voice-assistance-in-teams-274bb2f0-d962-4182-b45d-307435cea256?ui=en-us&rs=en-us&ad=us)

### <a name="play-my-emails"></a>播放我的电子邮件

播放通过 Outlook Mobile)  (连接的"我的电子邮件"是一种无语音驱动的免费体验，可让用户在其"重点收件箱"中听见新邮件，并通过电话、耳机或连接音频设备更改每天一部分的功能。 用户可以要求 Cortana 朗读其最近的电子邮件，并要求 Cortana 执行标记、存档、删除和跳过邮件等操作。 在开始操作、多任务或到目前地在处理时，此功能尤为有用。 当用户在"我的电子邮件"中与 Cortana 通时，语音音频请求会直接转至 Cortana 企业服务。 用户电子邮件语音签出的文本在 Office 365 云内部进行处理。 在此过程中，用户的移动设备上不会处理任何 Office 365 数据，也不会保存电子邮件数据。 一种传感命令 (例如"标记为已读"、"下一步"、"标记"等 ) ）可能会按照 Microsoft Online Services 条款中的数据保护条款保留语音[命令。](https://www.microsoft.com/licensing/product-licensing/products)

### <a name="briefing-email"></a>简短电子邮件
