# <a name="icann-verification-of-contact-information"></a>联系人信息的 ICANN 验证

在购买域或对域的联系信息进行更改时，需要注册器验证您的联系信息，以符合 ICANN 法规。

若要进行验证，请将电子邮件发送到注册人的地址。

 **若要验证您的联系信息，请选择验证电子邮件中的链接。** 如果域所有者未验证电子邮件地址，则该域将被挂起。 然后，转到 "管理中心" 中的 "[域" 页面](https://admin.microsoft.com/adminportal/home?ref=Domains)并检查您的域状态。
