# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>使用个人电子邮件地址注册 Office 365

## <a name="when-you-skip-adding-a-custom-domain"></a>跳过添加自定义域的情况

  
当你使用个人电子邮件地址注册 Office 365 并跳过现在添加[自定义域](../get-help-with-domains/what-is-a-domain.md)时，将限制你对某些功能的访问。 
  
如果现在选择添加自定义域，您将获得以下所有高级功能的访问权限：
  
|**功能**|**带有个人电子邮件的 Office 365**|**Office 365 with custom domain**|
|:-----|:-----|:-----|
|**OneDrive <sup>1、2</sup>**| [个人 OneDrive](https://onedrive.live.com/about/en-us/plans/)| [OneDrive for Business](https://onedrive.live.com/about/en-us/business/) |
|**Office 应用程序： Word、Excel、PowerPoint、OneNote、Outlook、Access （仅适用于 PC）**| 是 | 是 
|**业务应用程序<sup>3</sup> ： Microsoft 连接、开票、列表、预订、MileIQ 和 Outlook 客户经理**| 否 | 可访问 
|**访问 Microsoft 365 管理中心**| 受限访问（帐单、支持和域设置） | 是
|**添加用户**| 否 | 可访问
|**Office 365 服务： Exchange Online、Skype for Business、Microsoft 团队、SharePoint Online**| 否 | 可访问
|**安全性和合规性工具**| 否 | 可访问
