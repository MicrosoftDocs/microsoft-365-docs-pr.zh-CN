# <a name="teamwork--people-experiences"></a>团队合作–人员体验

通过使用 Microsoft 365 和支持 it 的技术体验，工作效率分数可深入了解您的组织的数字转换旅程。 贵组织的分数反映了人员和技术体验指标，可以与类似于您的组织的基准进行比较。 "团队合作" 类别是 "人员体验" 下的度量的一部分。 若要了解详细信息，请查看 [工作效率分数概述](productivity-score.md) 和阅读 [Microsoft 隐私声明](https://privacy.microsoft.com/privacystatement)。

## <a name="why-your-orgs-teamwork-score-matters"></a>您的组织的团队合作分数为何重要

当具有共同的目标的一组人员与其他人一起工作并将共同的资源用于集体成功时，企业工作效率的基本支柱。 研究表明当用户共享共享工作区中的信息和协作时，每周最长可节省4小时。 他们可以找到相关文档、查找之前讨论的上下文，并向共享目标传递。 查看[证据](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf)

## <a name="how-we-calculate-the-teamwork-score"></a>如何计算团队合作分数

我们在你的组织中提供了有关此类别的关键指标的体验的主要见解。 然后，下面详细说明了一个记分框架，用于计算组织的分数。

### <a name="primary-insight"></a>主要洞察力

主要真知灼见将介绍使用电子邮件和 Microsoft 团队中的邮件进行通信的所有人员，并在共享工作区中与云上的内容进行交互。 在 Microsoft 365 中，Microsoft 365 组是用户在共享工作区中进行协作的基础，它能够向组邮箱发送电子邮件、在 SharePoint 团队网站上共享文件以及通过 Microsoft 团队发送频道消息。

:::image type="content" source="../../media/teamwork-score.png" alt-text="显示团队合作分数中的主要见解的图表。":::

1. **标头：** 提供执行以下任一活动的人员 inyour 组织的关键指标：
      - 通过 Exchange 将电子邮件发送到组邮箱。
      - 通过团队发送频道消息
      - 读取和创建内容 (我们统称为 SharePoint 团队网站中的内容交互) 。

        作为组织中所有用户执行以下任一活动 (在共享工作区的内部或外部) 的百分比：
        - 通过 Exchange 发送电子邮件。
        - 在 Microsoft 团队)  (聊天或频道消息发送邮件。
        - 阅读和创建 OneDrive 或 SharePoint 上的内容。

            并至少能够访问以下服务之一： Exchange、Microsoft 团队或 SharePoint

1. **正文：** 提供了有关如何在共享工作区中进行通信和交互的详细信息，在共享工作区中完成时，可能会对组织中的工作效率产生积极的成果。
2. **可视化 (当前状态) ：**
      - 水平条，其中蓝色部分表示在标头中表示的百分比。
      - 突出显示用于计算标头中所示百分比的分子/分母)  (分数
        - 分子：组织中通过 Exchange 向组邮箱发送电子邮件的人员数量，或者通过团队发送频道的人员或在 SharePoint 团队网站中读取和创建内容的人员的数量。
        - 分母：组织中通过 Exchange 发送电子邮件的人员数，或在 Microsoft 团队中发送电子邮件 (聊天或频道消息) ，或者在 OneDrive 或 SharePoint 上阅读和创建内容，并且至少可以访问以下服务之一： Exchange、Microsoft 团队或 SharePoint。
   - 关键指标的对等基准值也显示为百分比。

### <a name="scoring-framework"></a>记分框架

组织的团队合作分数在聚合 (组织) 级别进行度量，无论用户是在最近28天内一致地在共享工作区中的文件活动中进行通信还是参与操作。

在单个用户级别上不提供分数。

        - 分母：用户在最近28天内发送电子邮件。 这是在通信效率分数的主要洞察力中标记为发送电子邮件的同一组人员。
      - **发送邮件**：彩色部分和分数表示在 Microsoft 团队中的频道中发送邮件的人所占的百分比。 分数由以下部分组成：
        - 分子：在最近28天内发送频道消息的人。
        - 分母：在最近28天内发送聊天或频道消息的人。 这是在工作效率分数的通信类别的主要洞察力中标记为以 Microsoft 团队发送邮件的同一组人员。
    - **创建内容**：彩色部分和分数表示读者在 Microsoft 365 SharePoint 团队网站上阅读或创建内容的百分比。
        - 分子：在 Microsoft 365 组连接的团队网站上阅读或创建内容的用户数。
        - 分母：具有 SharePoint 访问权限的用户数，在最近28天内，在 OneDrive 或 SharePoint 网站中读取或创建任何类型的内容。
