# <a name="teamwork--people-experiences"></a>团队合作 – 人员体验

Productivity Score 通过 Microsoft 365 的使用和支持它的技术体验，提供组织数字转型旅程的见解。 贵组织的分数反映了人员和技术体验度量，可以与类似你组织的基准进行比较。 团队合作类别是衡量人员体验的一部分。 若要了解更多信息，请查看 [Productivity Score 概述](productivity-score.md) 并阅读 [Microsoft 隐私声明](https://privacy.microsoft.com/privacystatement)。

## <a name="prerequisites"></a>先决条件

若要开始使用团队合作见解，需要获得针对组织人员的许可：

- Microsoft Teams
- SharePoint
- Exchange Online

有关详细信息，请参阅向 [用户分配许可证](../manage/assign-licenses-to-users.md)。

在用户过去 28 天内至少活跃一次上述产品后，你将开始看到见解。

## <a name="why-your-orgs-teamwork-score-matters"></a>为什么您的组织的团队合作分数很重要

组织工作效率的一个基本支柱是，一组具有共同目标的人相互合作，并共同拥有共同的成功资源。 研究表明，当用户在共享工作区中共享信息和协作时，他们每周最多节省 4 小时。 他们可以查找相关文档、查找以前讨论的上下文，并实现共同目标。 查看[证据](https://www.microsoft.com/microsoft-365/blog/wp-content/uploads/sites/2/2019/04/Total-Economic-Impact-Microsoft-Teams.pdf)

## <a name="how-we-calculate-the-teamwork-score"></a>如何计算团队合作分数

我们提供包含组织中此类别的关键指标的体验的主要见解。 然后，评分框架（如下所述）用于这些指标来计算组织的分数。

### <a name="primary-insight"></a>主要见解

主要见解将查看使用 Microsoft Teams 上的电子邮件和消息进行通信，并与共享工作区中的云内容进行交互的所有人员。 在 Microsoft 365 中，Microsoft 365 组是人员在共享工作区中协同工作的基础，能够向组邮箱发送电子邮件、在 SharePoint 团队网站上共享文件，以及通过 Microsoft Teams 发送频道消息。

:::image type="content" source="../../media/teamwork-score.png" alt-text="显示团队合作分数的主要见解的图表。":::

1. **标头：** 提供贵组织内执行以下任何活动的人的关键指标：
      - 通过 Exchange 将电子邮件发送到组邮箱。
      - 通过 Teams 发送频道消息
      - 阅读和创建 (我们统称为 SharePoint 团队网站中) 交互的内容。

        在共享工作区内或外部执行以下任一活动的 (百分比) ：
        - 通过 Exchange 发送电子邮件。
        - 在 Microsoft Teams (聊天或) 发送消息。
        - 在 OneDrive 或 SharePoint 上读取和创建内容。

            并且至少有权访问以下服务之一：Exchange、Microsoft Teams 或 SharePoint

1. **正文：** 提供有关在共享工作区中与内容通信和交互如何对组织工作效率产生积极结果的更多信息。
2. **可视化 (当前状态) ：**
      - 蓝色部分代表标题中表示的百分比的水平栏
      - 突出显示用于 (页眉) 百分比的分子/分母分数
        - Numerator：组织中通过 Exchange 向组邮箱发送电子邮件的用户数，或者通过 Teams 发送频道消息，或者阅读和创建 SharePoint 团队网站中的内容的用户数。
        - 分母：组织中的用户数：通过 Exchange 发送电子邮件、在 Microsoft Teams 上发送电子邮件 (聊天或频道消息) 或在 OneDrive 或 SharePoint 上读取和创建内容，并且至少有权访问以下服务之一：Exchange、Microsoft Teams 或 SharePoint。
   - 关键指标的对等基准值也显示为百分比。
3. **查看有关团队合作的资源：** 选择此链接可查看帮助内容。

### <a name="scoring-framework"></a>记分框架

组织的团队合作分数以 (组织) 级别衡量用户过去 28 天内是否一致地在共享工作区中进行通信或参与文件活动。

未在单个用户级别提供分数。

### <a name="breakdown-of-how-people-engage-in-shared-workspaces"></a>人员如何参与共享工作区的细分

1. **标头：** 显示所测量的不同类型的团队合作的详细细分。
2. **正文：** 提供有关在共享工作区中工作的价值的信息，以帮助团队更有效。
3. **可视化：** 可视化显示与内容通信或交互的人在共享工作区中执行此操作的程度，如下所示：
      - **发送电子邮件：** 彩色部分和小数表示向组邮箱发送电子邮件的用户百分比。 分数由以下组成：
        - Numerator： People sending emails to group mailboxes in the last 28 days.
        - 分母：过去 28 天内发送电子邮件的人。 这是在通信工作效率分数的主要见解中标记为发送电子邮件的同一组人员。
      - **发送邮件：** 彩色部分和小数表示在 Microsoft Teams 的频道中发送消息的百分比。 分数由以下组成：
        - 分子：过去 28 天内发送频道消息的人。
        - 分母：过去 28 天内发送聊天或频道消息的人。 这是在"工作效率分数"中对通信类别的主要见解中标记为在 Microsoft Teams 中发送消息的同一组人员。
    - **创建内容**：彩色部分和分数表示在 Microsoft 365 SharePoint 团队网站上阅读或创建内容的人的百分比。
        - Numerator：在 Microsoft 365 组连接的团队网站上阅读或创建内容的人数。
        - 分母：有权访问 SharePoint 的人数，他们过去 28 天内在 OneDrive 或 SharePoint 网站中读取或创建了任何类型的内容。

### <a name="breakdown-of-workspace-engagement-by-size-and-age"></a>按大小和年龄细分工作区参与度

:::image type="content" source="../../media/workspace-engagement-bysizeage.png" alt-text="显示按大小和年龄分类的工作区参与度图表。":::

1. **标头：** 显示工作区中服务参与的分类，按工作区中成员数的大小和工作区年龄（以月表示）进行细分。
2. **正文：** 提供有关鼓励组织中人员仅保留促进更有效的团队合作所需的工作区的价值的信息。
3. **可视化：** 参与细分以跨两个维度的热图形式显示。

      - **工作区的大小：** 工作区根据成员数分为三类：2-10 人、11-100 人以及 100 多人。 " &quot; 所有 &quot; "类别包括所有大小类别。
      - **工作区年龄：** 工作区按自首次创建工作区以来的月数进行分类。 " &quot; 所有 &quot; "类别包括所有年龄类别。

        图表的每个单元格都有一个数字和颜色，该数字和颜色基于属于该类别的已参与工作区的百分比。 工作区类别基于该单元格的交集中显示的年龄和大小。 例如，如果 11-100 人与 4-12 个月的交集处单元格的值为 52%，则意味着 52% 的工作区（成员在 11-100 之间，介于 4-12 个月之间）具有某种形式的参与度。 百分比的计算公式为：

        - **分子：** 过去 28 天内以通信形式参与 (和频道消息) 或内容交互的工作区
        - **分母** ：组织过去 28 天内可用的所有工作区
