# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 管理中心中创建通讯组
  
当您希望将电子邮件发送给一组用户而无需键入每个收件人的姓名、通讯组列表 (通讯组列表) 由特定的讨论主题 (（如 "营销" ) 或共享要求他们经常进行通信的常见工作）的用户进行组织时使用。 通讯组列表还提供一种自动向多个电子邮件地址转发电子邮件的方式。

通讯组有时称为通讯组列表。
  
> [!VIDEO https://www.microsoft.com/videoplayer/embed/e8747233-4f1c-4bf4-81cf-2a4ea9ccd80d?autoplay=false]
  
## <a name="create-a-distribution-group-list"></a>创建通讯组 (列表) 

::: moniker range="o365-worldwide"

转到位于 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> 的管理中心。

::: moniker-end

::: moniker range="o365-germany"

转到位于 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a> 的管理中心。

::: moniker-end

::: moniker range="o365-21vianet"

转到位于 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a> 的管理中心。

::: moniker-end

1. 选择应用启动器图标 ，然后选择" **管理员** "。
    
    找不到需要的应用？ 在应用启动器中，选择 " **所有应用** " 以查看可供您使用的应用程序的按字母顺序排列的列表。 可在此处搜索特定应用。 
    
2. **Groups** \> 在左侧导航窗格中选择 "组 **组** "，然后选择 " **添加组** "。 
      
3. 在 " **选择组类型** " 字段中，选择 " **分发** "，然后选择 " **下一步** "。
  
4. 输入新组的名称、说明和电子邮件别名，并选择是否希望组织外部的人员向此组发送电子邮件。 
    
5. 按 " **添加** " 以创建一个组，然后查看您的组并选择 " **关闭** "。 
    
6. 若要将用户添加到通讯组列表，请参阅 [向通讯组添加用户或联系人](../email/add-user-or-contact-to-distribution-list.md)。
    
请参阅 [使用联系人组 (以前的通讯组列表) 在 outlook](https://support.microsoft.com/office/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39) 主题中的使用 "在 2016 outlook 中使用的组" 和 "outlook 网页版"。 
  
查看 [通讯组问题](https://docs.microsoft.com/office365/troubleshoot/groups/distribution-list-issues) 以获取有关通讯组列表问题的帮助。
