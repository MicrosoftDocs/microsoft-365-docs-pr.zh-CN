# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>获取最新版本的 Microsoft 365 使用情况分析

模板应用程序可能会使用新数据或新的可视化效果每年刷新几次。 现有实例将继续有效，但如果您想要获取最新版本，则必须创建一个新实例，并且必须将任何自定义项应用于新实例。 请参阅[Enable Microsoft 365 usage analytics](enable-usage-analytics.md)。
