# <a name="get-access-to-microsoft-bookings"></a>获取 Microsoft Bookings 访问权限

本主题介绍如何首次获取对预订的访问权限。 此外，还介绍了如何在组织中打开或关闭预订。 此外，还介绍了如何将许可证分配给用户，以便他们可以访问和使用预订。

## <a name="access-bookings-for-the-first-time"></a>首次访问预订

如果您的 IT 管理员已授予您对预订的访问权限，则可以通过 Office online 访问该应用。

首次使用 " [预定](https://outlook.office.com/bookings/onboarding) " 时，将创建一个日历并设置您的业务。

1. 选择 " **立即获取** "，然后选择 " **添加预订日历**"。 如果这不是您首次在此页上，则可以选择另一个预订日历或对另一个预订日历执行搜索。

:::image type="content" source="media/bookings - get it now.png" alt-text="屏幕截图：立即获取 "记帐" 页面。选择 "立即获取" 以开始使用预订":::

2. 输入你的业务名称和业务类型，然后选择 " **继续**"。

你现在可以为你的组织设置预订。 您可以从应用启动器或从 office.com 转到此 [链接](https://outlook.office.com/bookings/onboarding)的预订页面。 使用以下主题可继续设置你的预订功能。

- [输入你的业务信息](enter-business-information.md)
- [将员工添加到预订](add-staff.md)
- [定义服务选项](define-service-offerings.md)
- [安排节假日、休息时间和休假时间](schedule-closures-time-off-vacation.md)
- [设置计划策略](set-scheduling-policies.md)
- [设置员工工作时间](employee-hours.md)
- [自定义和发布预定页面](customize-booking-page.md)
- [创建手动预订](create-a-manual-booking.md)
