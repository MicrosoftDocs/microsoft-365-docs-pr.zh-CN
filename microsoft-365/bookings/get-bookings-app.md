# <a name="get-the-microsoft-bookings-app-for-ios-and-android"></a>获取适用于 iOS 和 Android 的 Microsoft 预订应用程序

感谢下载 Microsoft 预订应用！ Microsoft 预订可用作 iOS 和 Android 的移动应用程序。 适用于 iOS 的预订应用程序适用于 Apple 支持的所有地区和国家/地区。 您可以从 [ITunes 应用商店](https://apps.apple.com/app/microsoft-bookings/id1065657468)下载应用程序。 适用于 Android 的预订应用程序可从美国和加拿大的 [Google Play 商店](https://play.google.com/store/apps/details?id=com.microsoft.exchange.bookings) 下载。

在开始之前，您需要在 web 上设置预定。

1. 找不到需要的应用？ 在应用启动器中，选择 "所有应用" 以查看可供您使用的 Microsoft 365 应用程序的按字母顺序排列的列表。 在这里，可以搜索特定的应用程序

   ![应用启动器的图像](../media/bookings-all-apps-launcher.png)

2. 从 [Office 主页](https://office.com) 和应用启动器中选择 " **预定**"。

3. 选择 " **立即获取**"。

4. 提供您运行的业务的名称和类型，如头发 salon 或牙科实践，然后选择 " **预订**"。

5. 你现在可以为你的组织设置预订。 按照 [Microsoft 记帐](bookings-overview.md) 主题中的步骤完成预定设置。 返回到你的移动设备并注销移动应用。 重新登录以转到新的预定日历。

## <a name="view-only-mode"></a>仅查看模式

在预订中未被授予读写访问权限的任何用户仍可在仅查看模式下使用移动应用。 拥有预订许可证的任何人都可以在预订日历中查看他们自己的同事的日程安排、约会详细信息和业务信息。 具有仅查看访问权限的用户无法进行更改或编辑，也无法访问客户列表。
