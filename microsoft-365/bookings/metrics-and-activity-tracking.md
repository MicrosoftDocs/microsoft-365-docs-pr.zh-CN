# <a name="microsoft-bookings-metrics-and-activity-tracking"></a>Microsoft 预订指标和活动跟踪

主 "预订" 页面上提供了各种指标，使您可以轻松跟踪收入和客户活动。

| 跃点数 | Description |
|:---|:---|
| 预订活动 | 30天活动期显示最近30天，包括今天。 将**进行预订**、**估计收入**和**预订的客户**全部计算。 |
| 已进行预订 | 当前发生或在最近30天内发生的预订的数量。 您可以使用此信息来确定您是否符合估计的销售预期，并预测未来30天。 |
| 估计收入 | 这将显示您通过预订获得的预计收入。 这是根据您为每个服务定义的价格计算得出的。 这是一种估计，因为在服务时间可能会收取不同的费用，费用设置为每小时费用，你已获得提示，或者客户尚未支付。 例如，如果您的服务每小时充电 $10，并且预订时间为2小时，则预订将仅估计该服务的 $10。<br/><br/>**重要说明：** 这只是一个估计，并不能保证实际收入。 |
| 已预订的客户 | 目前和最近30天预订约会的客户数。<br/><br/>**注意：** 此指标显示已预订约会的客户数，而不是约会总数。 例如，如果一个客户在最近30天中预订了三个约会，并且两个客户每个客户都进行了一次预订，您将看到3个已预订的客户。 |
