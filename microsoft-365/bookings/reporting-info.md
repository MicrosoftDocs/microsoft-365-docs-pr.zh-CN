# <a name="reporting-info-for-bookings"></a>用于预订的报告信息

现在，您可以在 TSV 文件中看到预订日历的120天视图。 TSV 文件是以制表符分隔的值文件。 可以使用任何文本编辑器或电子表格程序（如 Excel）查看或编辑此类文件。

## <a name="see-120-days-booking-activity"></a>请参阅120天预定活动

1. 在 "预定日历" 仪表板上，选择 "将更多数据导出为 TSV"。

:::image type="content" source="../media/bookings-activity.png" alt-text="屏幕截图：30天预定活动":::

1. 使用新名称保存该文件，并指定 .xls 或 .xlsx 格式。

1. 打开文件以查看预订日历的120天视图。

对于前60天的预订日历上的每个事件，文件都包含一行，在即将到来的60天内。 每行包括事件的日期和时间信息、客户名称、客户电子邮件、客户电话号码、客户地址、分配的员工、所选的服务、位置和约会持续时间。
