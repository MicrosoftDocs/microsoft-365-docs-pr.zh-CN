# <a name="install-office-apps"></a>安装 Office 应用 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/acce002c-0756-4b64-ac5d-2198ee96a9b1?autoplay=false]

## <a name="try-it"></a>试一试！

Microsoft Office"开始"  **菜单上找到** 这些应用。 如果看不到&#39;，可以自行安装。

1. 转到 office.com。 你可能需要使用你的工作帐户登录。
2. 选择 **"安装 Office**   >   **Microsoft 365 应用**   >   **运行"，** 然后选择"**是"。**
3. 将安装 Office 应用。 此过程可能需要几分钟时间。 完成后，**选择关闭。**
4. 若要安装 Microsoft Teams，请转到"office.com"页面，然后选择 **"Teams"。**
5. 获取 Windows 应用，  **然后选择运行**。 安装完成后，Teams 将显示提示。

您安装的 Office 应用现在显示在"开始  **"** 菜单中。
