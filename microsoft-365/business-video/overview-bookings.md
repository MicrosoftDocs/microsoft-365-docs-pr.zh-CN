# <a name="overview-of-microsoft-bookings"></a>Microsoft Bookings 概述

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE26B1q?autoplay=false]

Microsoft Bookings 是一款联机和移动应用，适用于小型企业，它们以约会方式为客户提供服务。 企业的示例包括发夹店、商店、spa、法律公司、金融服务提供商、顾问和汽车店。

## <a name="get-started-with-bookings"></a>Bookings 入门

1. 若要开始登录 Microsoft 365，然后从应用启动器中选择 **Bookings。**
1. 在 **Bookings** **页面上** 选择"现在获取它"。
1. 添加业务信息，你已准备好开始使用 Bookings。

    若要在移动设备上管理你的预订，请从应用商店下载 Bookings。

## <a name="use-bookings"></a>使用 Bookings

- 在 **Bookings** 仪表板上，查看你的公司地址、小时和其他业务信息，并根据需要进行更新。
- 在 **"日历** "页上，你和员工可以添加或更新客户约会。 可以按日、周或月查看日历。
- 在 **"客户** "页上，可以从其他帐户（如 Gmail 或设备）添加或导入新客户。
- 通过 **"** 员工"页面，可以添加和管理员工。 添加他们的电子邮件地址非常重要，以便他们可以立即收到新约会的通知。
- " **服务** "页允许您添加或管理客户在预订约会时选择的服务。
- 最后，在 **Bookings** 页面上，你将找到许多设置，如日程安排策略、电子邮件通知、时区设置等。

保存并发布 Bookings 日历后，可嵌入公司网站或 Facebook 页面。
