# <a name="overview-of-sharing-business-files"></a>共享业务文件概述

你需要与你的企业中的客户、客户、合作伙伴、供应商和其他人员共享文件。 如何共享文件以便每个人都可以找到、打开和编辑它们？

## <a name="share-a-file-with-someone-outside-of-your-company"></a>与公司外部的其他人共享文件

- 如果您只需要客户端或客户 (，可以通过电子邮件与公司外部的其他人共享文件) 或客户共享文件。
- 如果需要响应，或希望他们填写一些信息并将其发送回，可以使用 SharePoint 或 Microsoft Teams 与编辑链接共享。

详细了解如何使用安全链接 [在](securely-share-files-externally.md) 组织外部共享文件，以及如何与任何人 [链接共享文件](share-files-externally.md)。

## <a name="share-and-collaborate-with-a-client-or-business-partner"></a>与客户或业务合作伙伴共享和协作

作为项目的一部分共享？ 还是作为与业务合作伙伴或供应商持续关系的一部分？ 只需多投入一点时间，在 Microsoft Teams 中设置团队，就可以有一个持续的位置来共享和协作。 与企业外部人员合作非常简单，甚至可以为其他供应商或合作伙伴设置单独的频道，使一切保持简单。

1. 创建团队。
1. 上传文件。
1. 邀请合作伙伴或供应商作为"来宾"用户加入团队，以授予他们访问权限。

详细了解如何使用安全链接在组织 [外部共享](https://support.microsoft.com/office/7266f44e-3e06-4736-b9d3-0580c24bba34)文件。

## <a name="share-inside-your-business"></a>在企业内部共享

如果你正在处理在企业内部共享的文件，在 Microsoft Teams 中设置团队也是一个好主意。 在那里共享你的文件，你的企业中的每个人都可以轻松查找和编辑它们，并且你知道正在同一页面上工作。

1. 创建团队并邀请团队加入。
1. 上传文件。

详细了解公司[文件存储和](https://support.microsoft.com/office/e4d98e10-3532-4eed-85d6-92728454e32b)[员工文件存储](https://support.microsoft.com/office/12dbe3e4-dbef-48f8-a90e-87f1bc607073)。
