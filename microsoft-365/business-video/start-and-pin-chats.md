# <a name="chat-with-employees-and-customers"></a>与客户和员工聊天

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWedV6?autoplay=false]

可以在内部与同事聊天，也可以与来宾用户（如客户或合作伙伴）聊天。 如果 [为企业启用](https://docs.microsoft.com/microsoftteams/set-up-guests) 来宾访问，则只需键入电子邮件地址以将来宾添加到聊天。

## <a name="try-it"></a>试一试！

从 1：1 开始或群组聊天，以便与较小的组协作或私人聊天，并固定聊天以便快速访问。

1. 在  **聊天** 中，你可以与人员进行一对一或群组聊天。
2. 选择 **"新建聊天"。** :::image type="icon" source="../media/new-chat-icon.png" border="false":::  
3. 键入要联系的一个或多个人员的名称。
4. 选择向下箭头以向聊天组添加名称。

通过聊天进行视频或音频通话

- 选择  **视频** :::image type="icon" source="../media/video-call-icon.png" border="false"::: 呼叫 **或音频呼叫** :::image type="icon" source="../media/audio-call-icon.png" border="false"::: ，直接从聊天中呼叫某人。
