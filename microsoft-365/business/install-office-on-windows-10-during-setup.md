# <a name="install-office-on-windows-10-during-setup"></a>在设置过程中，在 Windows 10 上安装 Office

![指向的标题https://aka.ms/aboutM365preview。](../media/m365admincenterchanging.png)

默认情况下，此设置处于关闭状态。 你可以将其打开以确保用户在其所有 Windows 10 设备上拥有最新版本的 Office，但先验证是否满足所有先决条件。 在 microsoft [365 商业版的 Office 客户端部署](prepare-for-office-client-deployment.md)中介绍的其他要求之外，用户必须通过 Microsoft 365 商业版的许可，才能将软件推送到设备上。
  
建议您在安装过程中接受默认设置。
