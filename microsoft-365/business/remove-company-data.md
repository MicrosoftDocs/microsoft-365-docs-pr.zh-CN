# <a name="remove-company-data-from-devices"></a>从设备中删除公司数据

## <a name="remove-company-data"></a>删除公司数据

可使用 Microsoft 365 商业版 删除用户存储在其[设备](app-protection-settings-for-android-and-ios.md)或受 Microsoft 365 保护的 [Windows 电脑](protection-settings-for-windows-10-devices.md)上的公司数据。 **从设备中删除公司数据后，该数据不可恢复** 。 
  
1. 使用全局管理员凭据登录 [Microsoft 365 商业版](https://portal.office.com)。 
    
2. 在管理中心内的" **设备操作**"卡中，选择" **删除公司数据**"。
    
    ![On the Devices card, choose Remove company data](media/b6fcf74b-0d7d-4e1a-894f-40f9d4a215b8.png)
  
3. 在" **删除公司数据**"窗格中，选择或搜索要删除其数据的用户，然后选择名称。 
    
4. 在下一个窗格中，从" **托管设备**"列表中选择要从中删除数据的一个或多个设备，然后选择" **下一步"**。 
    
    ![On the remove comapany data pane, select the device from which you want to remove the data.](media/f3725ff9-ebdb-4c13-9523-b2df362640cf.png)
  
5. 在" **确定删除设备上的公司数据**"窗格中，选择" **删除**"\>" **关闭**"。
