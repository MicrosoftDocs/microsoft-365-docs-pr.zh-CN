# <a name="remove-company-data-from-devices"></a>从设备中删除公司数据

![指向的标题https://aka.ms/aboutM365preview。](media/m365admincenterchanging.png)

## <a name="remove-company-data"></a>删除公司数据

可使用 Microsoft 365 商业版 删除用户存储在其[设备](app-protection-settings-for-android-and-ios.md)或受 Microsoft 365 保护的 [Windows 电脑](protection-settings-for-windows-10-devices.md)上的公司数据。 **从设备中删除公司数据后，该数据不可恢复** 。 
  
1. 转到管理中心<a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>。
    
2. 在左侧导航中, 选择 "**设备**\> **管理**"。  
  
3. 在 "**管理**" 页上, 选择或搜索要删除的数据的用户, 然后选择名称。 
    
4. 在下一个窗格中, 从 "**设备**" 列表中选择一个或多个设备, 并在打开的设备窗格中, 选择将设备重置为出厂设置, 或删除 "公司数据" (具体取决于设备类型)。 
    
    ![On the remove comapany data pane, select the device from which you want to remove the data.](media/resetorremove.png)
  
5. 在 "确认" 窗格中, 选择 "**确认** \> **关闭**"。
