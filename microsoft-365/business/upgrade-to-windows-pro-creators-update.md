## <a name="upgrade-to-windows-10-pro"></a>升级到 Windows 10 专业版
  
若要升级到 Windows 10 专业版，有几个选项。 可执行下列操作：
    
- 从 [Microsoft 软件](https://go.microsoft.com/fwlink/?LinkID=836951 )下载网站安装升级 如果您登录的设备与要更新的设备位于同一台设备上，请 &ndash; 选择此选项。 

    - 从软件下载站点 **，单击"** 立即更新"以开始将设备升级到 Windows 10 专业版创意者更新。 
    
- 使用媒体创建工具创建安装[](https://go.microsoft.com/fwlink/?LinkID=836960)媒体 选择此选项可创建 Windows 10 专业版创意者更新安装媒体 (U 盘或 ISO 文件) 以在不同于你使用的电脑上安装 &ndash; Windows 10。

    - 阅读有关如何使用该工具和创建安装媒体的说明。 

> [!NOTE]
> 如果你拥有运行 Windows 7 专业版、Windows 8 专业版 或 Windows 8.1 专业版的 Windows 设备，则你的 Microsoft 365 商业版订阅将授权你进行 Windows 专业版 10 升级。
    
请参阅 [为 Microsoft 365](set-up-windows-devices.md) 商业版用户设置 Windows 设备以完成 Windows 10 设备的设置。 
  
请参阅 [为 Microsoft 365](set-up-mobile-devices.md) 商业版用户设置移动设备以完成 Android 和 iOS 设备的设置。
