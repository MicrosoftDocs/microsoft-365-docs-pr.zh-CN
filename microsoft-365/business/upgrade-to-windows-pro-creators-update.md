# <a name="upgrade-windows-devices-to-windows-pro-creators-update"></a>将 Windows 设备升级到 Windows 专业版创意者更新

## <a name="upgrade-to-windows-10-pro-creators-update"></a>升级到 Windows 10 Pro 创建者更新

1. 登录到[Microsoft 管理中心](https://portal.office.com/adminportal/home)并找到**Windows 10 升级**卡片。 
    
    ![在管理中心内的 Windows 10 升级卡片。](media/066f47bf-7b88-4fea-8fd0-82798ea66716.png)
  
2. 升级到 Windows 10 专业人员创建者更新，您具有多个选项。您可以：
    
- 从 Microsoft 软件下载网站安装升级-选择此选项，您登录的设备是否是您要更新为同一设备上。
    
  a.选择**安装升级**。
    
  b.软件下载站点中，单击**立即更新**以开始升级到 Windows 10 专业人员创建者更新的设备。 
    
- 共享的下载链接-如果您想要将电子邮件发送给组织中的人员，以便他们可以通过本身升级到 Windows 10 专业人员，请选择此选项。
 
   答： 选择**共享下载**链接，以打开包含 Windows 软件的链接的电子邮件下载站点。 
    
 - 创建使用媒体创建工具安装媒体-选择此选项可创建 Windows 10 专业人员创建者更新安装介质 （USB 闪存驱动器或 ISO 文件） 不同于是您使用的 PC 上安装 Windows 10。
    
    a.选择**创建安装介质**。
    
    b.阅读如何使用该工具并创建安装媒体上的说明。 
    
请参阅[Set up Windows 设备的 Microsoft 365 业务用户](set-up-windows-devices.md)完成 Windows 10 设备的设置。 
  
完成设置 Android 和 iOS 设备，请参阅[Set up Microsoft 365 业务用户的移动设备](set-up-mobile-devices.md)。
