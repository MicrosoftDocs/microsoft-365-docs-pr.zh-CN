# <a name="what-subscription-is-right-for-your-small-business"></a>适合您的小型企业的订阅是什么？

Microsoft 为小型企业提供了几个计划, 建议的订阅是 Office 365 商业高级版或 Microsoft 365 商业版, 最高可达300个用户。 Microsoft 365 企业提供了比 Office 365 商业高级版更多的安全功能, 以增加成本。
