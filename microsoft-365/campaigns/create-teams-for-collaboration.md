# <a name="create-teams-for-collaboration-in-microsoft-teams"></a>在 Microsoft 团队中创建协作团队

Microsoft 团队是一种协作应用程序, 可帮助你的员工在任何设备上进行组织和对话。 你可以使用 Microsoft 团队与你的员工或你的组织外部的来宾的成员进行即时对话。 您还可以拨打电话、主持会议和共享文件。

## <a name="best-practices"></a>最佳做法

例如, 市场活动可以创建以下团队以安全地进行通信和协作:

1. **市场活动领导团队**-将其设置为私有团队, 以便只有关键市场活动成员可以访问它并讨论潜在的敏感问题。
2. **一个普通的市场活动团队**-这适用于日常通信和工作的每个人使用。 个人、组或委员会可以在此团队中设置频道以完成其工作。 例如, 活动规划人员可以设置频道以聊天和协调市场活动事件的物流。
3. **合作伙伴团队**-您可以与供应商、合作伙伴或志愿者合作, 而无需让其成为任何敏感内容。

您还可以为特定项目创建团队, 并根据需要包含的用户应用适当的保护量。 

![Microsoft 团队窗口的关系图, 其中包含三个单独的团队, 以允许进行安全通信和协作](media/m365-democracy-teams-collab.png)

创建团队时, 将创建其他内容:

- 一个新的[Office 365 组](https://docs.microsoft.com/en-us/MicrosoftTeams/office-365-groups)
- 用于存储团队文件的[SharePoint Online](https://docs.microsoft.com/en-us/MicrosoftTeams/sharepoint-onedrive-interact)网站和文档库
- [Exchange Online](https://docs.microsoft.com/en-us/MicrosoftTeams/exchange-teams-interact)共享邮箱和日历
- OneNote 笔记本
- 与其他 Office 365 应用程序 (如 Planner 和 Power BI) 相关

在 Microsoft 团队中, 您可以找到:
1. **团队**-查找属于或创建您自己的频道。 在频道内, 您可以保存热点会议、进行对话以及共享文件。

2. **会议**-查看在一天或一周内已完成的所有内容。 或者, 安排会议。 此日历与 Outlook 日历同步。
 
3. **呼叫**-在某些情况下, 如果您的组织已设置, 则可以从 Microsoft 团队呼叫任何人, 即使他们没有使用 microsoft 团队也是如此。

4. **活动**-追赶所有未读邮件、@mentions、回复等。 

使用顶部的命令框搜索特定项目或人员、执行快速操作并启动应用。
