# <a name="sign-in-to-microsoft-365-for-campaigns"></a>登录到 Microsoft 365 以获取市场活动

## <a name="for-campaign-admins"></a>对于市场活动管理员
如果你注册了 Microsoft 365 以获取市场活动，则你是 Microsoft 365 管理员。下面介绍了如何登录： 
1. 查找我们发送到的电子邮件地址的用户名和密码，这些用户名和密码是在步骤2中 [进行注册的步骤](m365-campaigns-sign-up.md#steps-to-sign-up)2 中给出的。
2. 在浏览器中，转到 Microsoft 365 管理中心 <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a> 。 
3. 键入您的用户名和密码。 选择“**登录**”。
4. 在页面的右上部，查找 "在控件 **上预览** "。 选择 **"在打开时预览** "，可以使用 [为市场活动启用凹凸保护](m365-campaigns-security-overview.md)时所述的所有控件。

## <a name="for-campaign-staff"></a>针对市场活动员工
设置市场活动员工，如 " [添加用户](../business/add-users-m365b.md?toc=/microsoft-365/campaigns/toc.json) " 中所述。也可以在 " **添加用户** " 页上重置和重新发送密码。
所有市场活动员工均可在中登录 <a href="https://office.com" target="_blank">https://Office.com</a> 。
