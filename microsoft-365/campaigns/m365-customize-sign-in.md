# <a name="customize-your-sign-in-page-with-a-privacy-and-consent-notice"></a>使用隐私和同意通知自定义登录页面

通过向你的登录页面添加隐私和同意通知，你的企业或宣传活动可以更轻松地让执法机构向在线犯罪人士提出法律费用。

可以使用品牌自定义登录页。 还可以添加文本来帮助用户登录，或指出获取 Microsoft 365 资源访问权限的法律要求或限制。

## <a name="design-customization-the-text-on-your-sign-in-page"></a>设计自定义登录页上的文本

若要更新登录页上的可自定义元素，你必须是全局管理员。有关具体说明，请参阅 [添加公司品牌](https://docs.microsoft.com/azure/active-directory/fundamentals/customize-branding) 文章。

可以更新的元素包括：

- **登录页文本** 添加隐私和同意声明的简便位置。
- 登录页背景图像
- 横幅徽标
- 用户名提示

有关隐私和同意通知的示例，请参阅"搜索和安装计算机以及获取犯罪调查中的电子证据"中的 [附录](https://www.justice.gov/sites/default/files/criminal-ccips/legacy/2015/01/14/ssmanual2009.pdf)A。
