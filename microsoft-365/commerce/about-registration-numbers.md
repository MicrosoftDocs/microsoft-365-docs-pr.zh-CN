# <a name="about-registration-numbers-and-under-review-notifications"></a>关于注册号码和审核通知

本文仅适用于直接从 Microsoft 购买或激活产品或服务的商业客户。 本文不适用于批量许可或云解决方案提供商 (云解决方案提供商) 直接与合作伙伴合作的客户。

## <a name="what-is-a-registration-number"></a>什么是注册号码？  

我们使用注册号码查看你的帐户的详细信息。 这使我们可以确定 Microsoft 能否向您提供产品和服务。 请参阅 [下面的"按国家](#registration-numbers-by-country) /地区列出注册号码"部分，了解有关要输入此字段中的值的详细信息。

对于必须注册号码的国家/地区，文本框上方的标签指示需要哪种类型的号码。

对于注册号码是可选的国家/地区，你可以选择提供公司合法注册号码。 请勿在此字段中输入个人 ID。

如果你没有有效的注册号码， [请参阅按国家](#registration-numbers-by-country) /地区表示的注册号码了解详细信息。

## <a name="what-should-i-do-if-i-get-an-under-review-notification"></a>如果我收到审查不足的通知，该怎么办？  

完成购买后，可能会收到帐户正在审查的通知。 在审查过程中，您可以通过浏览到"计费帐户"页面并选择用于完成购买的帐户来  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank"></a>检查状态。

审阅过程通常需要大约一天才能完成，但可能需要更长时间。

<!-- The following screenshot shows the review notification displayed during checkout.
add screenshot -->

还会将电子邮件通知发送给帐户上的所有全局管理员和帐单管理员。 在某些情况下，通知将发送给在帐户上具有"计费帐户所有者"或"计费帐户参与者"角色的用户。 该通知表明当前正在进行审阅。 审核过程完成后，将发送确认电子邮件通知。
