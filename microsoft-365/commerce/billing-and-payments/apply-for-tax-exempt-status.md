# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>适用于 Office 365 商业版的免税状态

如果你的国家或地区的免税状态符合条件，请[联系支持人员](../../admin/contact-support-for-business-products.md)，以创建服务请求，以为你的订阅建立免税状态。 提交请求时，必须提供以下文档： 
  
|**Location**|**文档**|
|:-----|:-----|
|美国和加拿大  <br/> |销售使用税务证书  <br/> |
|波多黎各  <br/> |Certificado de Compras Exentas  <br/> |
|爱尔兰  <br/> |13B/56A 税务豁免证书  <br/> |
|持有免税的国际组织  <br/> |来自相关当地税务主管机构的认证/信函确认  <br/> |
