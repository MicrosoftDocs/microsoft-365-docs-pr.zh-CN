# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>Microsoft 365 商业版产品密钥有问题？

如果尝试兑换 **Microsoft 365** 商业版的产品密钥时出现错误消息，请尝试这些常见解决方案。 
  
 **立即需要帮助？** [致电 Microsoft 支持人员](../admin/contact-support-for-business-products.md)。 
  
 **有关 Microsoft 365 家庭版或 Microsoft 365 个人** 版产品密钥的帮助，请参阅使用 [Office 产品密钥](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)。
