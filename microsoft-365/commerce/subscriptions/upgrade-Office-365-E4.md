# <a name="upgrade-from-an-office-365-e4-subscription"></a>从 Office 365 E4 订阅升级

本文将引导你完成从 Office 365 E4 升级到新订阅的过程。 有关从 Office 365 E4 升级时可用的选项的信息，请参阅 [Office 365 e4 客户的重要信息](important-information-e4.md)。

> [!IMPORTANT]
> 本文适用于从 Microsoft 直接通过信用卡或发票购买的 Office 365 E4 订阅。 如果你的订阅是通过另一种方式（如通过合作伙伴或批量许可服务中心）购买的，请联系你的 Microsoft 帐户代表或合作伙伴以帮助你升级计划。

## <a name="what-are-my-options-for-how-to-upgrade"></a>我有关如何升级的选项是什么？

升级计划最简单的方法是使用 Microsoft 365 管理中心中的 " **升级** " 选项卡。 但是，并非所有情况下都支持使用 " **升级** " 选项卡。 如果您的方案不受支持，则可能能够手动升级计划。

## <a name="what-is-the-upgrade-tab"></a>什么是 "升级" 选项卡？

" **升级** " 选项卡可为您执行以下任务：

- 引导你完成新计划的购买过程。
- 将旧计划的所有用户许可证重新分配给新计划。
- 取消您的旧计划。

## <a name="what-does-it-mean-to-upgrade-plans-manually"></a>手动升级计划意味着什么？

手动升级计划意味着要完成以下单独的过程，而不是使用 " **升级** " 选项卡。

- 购买具有正确许可证数量的新订阅。
- 验证新订阅是否准备好可供使用。
- 向用户重新分配许可证。
- 取消原始的 Office 365 E4 订阅。

## <a name="find-out-if-you-can-use-the-upgrade-tab-to-upgrade-to-a-new-plan"></a>了解是否可以使用 "升级" 选项卡升级到新计划

1. 在管理中心中，转到 "**付费**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">产品</a>" 页面。
2. 选择 Office 365 E4 订阅。
3. 选择 " **升级** " 选项卡。如果你看到列出了其他计划，则意味着你可以自动升级计划。
4. 如果无法自动升级，则会看到一条消息，说明无法升级的原因。

无法自动升级计划的原因有多种。 大多数问题是临时的问题，如可以解决的服务运行状况问题。 有关详细信息，请参阅 [为什么无法升级计划？](upgrade-to-different-plan.md#why-cant-i-upgrade-plans) 如果你需要有关升级的帮助，请 [联系支持人员](../../admin/contact-support-for-business-products.md)。

## <a name="will-a-credit-check-be-required"></a>是否需要信用检查？

如果你通过发票为新计划付款，或购买超过一定费用，则可能需要核实信用。 如果需要信用检查，则升级最长可能需要两个工作日。 在信用检查完成之前，管理员无法访问 Microsoft 365 管理中心。 但是，在升级完成前，用户仍具有 E4 计划的完全访问权限。

## <a name="upgrade-your-plan-by-using-the-upgrade-tab"></a>使用 "升级" 选项卡升级您的计划

### <a name="before-you-begin"></a>准备工作

在开始之前有必要了解以下些许事项。

- **规划管理故障时间。** 在升级计划时，管理员无法使用 Microsoft 365 管理中心。 根据您拥有的用户数，升级可能需要几分钟到几小时的时间。 建议您计划在不需要使用 Microsoft 365 管理中心进行更新时执行升级。

    在计划升级过程中，用户不会遇到任何服务中断，在升级过程中，他们将继续拥有对 E4 订阅的完全访问权限。 升级完成后，用户将有权访问新计划。
- **用户、许可证、帐单和自定义域。** 若要了解如何在升级过程中处理用户和许可证，升级计划如何影响您的帐单以及如何处理自定义域，请参阅 [升级计划对我的服务和帐单有何作用？](upgrade-to-different-plan.md#what-does-upgrading-a-plan-do-to-my-service-and-billing)
- **更改用户许可证的数量。** 您不能将许可证作为升级计划的一部分删除。 但是，您可以减少升级计划之前或之后 [的许可证数量](../licenses/buy-licenses.md) 。

### <a name="start-the-upgrade-by-using-the-upgrade-tab"></a>使用 "升级" 选项卡启动升级

1. 在管理中心中，转到 "**付费**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">产品</a>" 页面。
2. 选择 Office 365 E4 订阅。
3. 在 "订阅详细信息" 页上，选择 " **升级** " 选项卡。
4. 找到要购买的订阅，然后选择 " **升级**"。
5. 在 " **购物车** " 页面上，验证一切是否正确。 选择是按月还是按年支付，并验证 " **数量**" 下的许可证数量。
    > [!NOTE]
    > 与 Office 365 E4 订阅关联的任何附加订阅（如 Office 365 额外文件存储）也会列出。 但是，如果你有要升级到的订阅中包含的任何附加订阅，我们会将其删除。
6. 查看订单后，选择 " **转到签出**"。
7. 在 "**签出**" 页面上，按**此顺序**查看 "**销售到**"、"**付款给**" 和 "项目"。 选择任一项目旁边的 " **更改** " 以编辑信息。
    > [!NOTE]
    > 将发票用作付款方式的选项仅适用于在特定成本额以上的订单。 如果需要进行信用检查，选择发票付款选项可将升级过程延迟最长为两个工作日。
8. 完成后，选择 " **下订单**"。

> [!NOTE]
> 如果没有错误或问题，则升级计划通常需要不到10分钟的时间。 您可以通过查看您的旧订阅或新订阅来检查升级状态。

## <a name="upgrade-your-plan-manually"></a>手动升级您的计划

若要将用户手动升级到其他订阅，请按所示顺序完成以下步骤。

- [步骤1：购买新订阅](#step-1-buy-a-new-subscription)
- [步骤2：验证订阅的许可证数是否正确](#step-2-verify-that-your-subscription-has-the-right-number-of-licenses)
- [步骤3：将许可证重新分配给用户](#step-3-reassign-licenses-to-users)
- [步骤4：取消 Office 365 E4 订阅](#step-4-cancel-the-office-365-e4-subscription)

### <a name="step-1-buy-a-new-subscription"></a>步骤1：购买新订阅

如果还没有新的订阅，可以 [购买另一个 Microsoft 365 for business 订阅](../buy-another-subscription.md)。

如果已有订阅，请继续执行下一步。

### <a name="step-2-verify-that-your-subscription-has-the-right-number-of-licenses"></a>步骤2：验证订阅的许可证数是否正确

在继续下一步之前，请务必确保已设置新订阅中的所有服务。 如果在您首次检查时订阅未准备就绪，请稍后重试。

> [!NOTE]
> 如果您选择通过发票支付新的订阅，可能需要进行信用检查。 在订阅可用之前，最长可能需要两个工作日内。

1. 在管理中心中，转到 "**付费**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">产品</a>" 页面。
2. 从 " **订阅状态** " 下拉下拉项中，选择 " **活动**"。
3. 确保您的新订阅已显示，并且许可证数量与您在 Office 365 E4 中所拥有的数量相同。
4. 如果需要购买更多许可证，请按照 [购买或删除订阅许可证](../licenses/buy-licenses.md)中的步骤操作。

### <a name="step-3-reassign-licenses-to-users"></a>步骤3：将许可证重新分配给用户

您可以使用 Microsoft 365 管理中心为最高20个用户一次重新分配许可证。 若要了解如何操作，请参阅 [将用户移动到其他订阅](move-users-different-subscription.md)。

> [!TIP]
> 如果有很多用户，则可以 [使用 Office 365 PowerShell 批量分配用户许可证](https://docs.microsoft.com/office365/enterprise/powershell/assign-licenses-to-user-accounts-with-office-365-powershell)。

### <a name="step-4-cancel-the-office-365-e4-subscription"></a>步骤4：取消 Office 365 E4 订阅

在将所有用户重新分配给新订阅之后，请 [取消 Office 365 E4 订阅](cancel-your-subscription.md)。
