# <a name="add-or-remove-members-from-a-case"></a>向事例添加成员或从事例中删除成员

您可以添加或删除成员来管理可以访问案例的成员。 但是，在成员可以访问高级电子数据展示案例 (并执行) 情况下的任务之前，必须将用户添加到安全与合规中心的"权限"页上的"电子数据展示管理员"角色组中。  有关详细信息，请参阅 [分配安全与合规中心中的电子数据展示权限](./assign-ediscovery-permissions.md)。

1. 在 **"高级电子数据展示"** 页面上，转到要添加成员的情况。

2. 单击" **设置"** 选项卡 **，然后在** "Access **&权限"磁贴中单击"** 选择"。

3. 单击“更新”。

4. 在 **"管理成员"** 下 **，单击"添加** "将成员添加到案例。 您还可以选择通过单击"管理角色组"下的"  **添加** "将角色组 **添加到案例**。

5. 在可添加为案例成员的人或角色组列表中，选中要添加的人或角色组的名称旁边的复选框。

   > [!NOTE]
   > 向案例添加角色组时，只能添加作为成员的角色组。

6. 选择要添加为案例成员的人或角色组后，单击"添加 **"。**

7. 在" **管理此案例"** 飞出页中，单击 **"保存** "以保存新案例成员列表。
