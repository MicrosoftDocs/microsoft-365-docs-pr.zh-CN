# <a name="learn-about-communication-compliance-in-microsoft-365"></a>了解 Microsoft 365 中的通信合规性

通信合规性是 Microsoft 365 中的内部风险解决方案，可帮助你检测、捕获和操作组织中不适当的邮件，从而最大程度地降低通信风险。 预定义和自定义策略允许你扫描内部和外部通信中的策略匹配项，以便指定的审阅者可以检查它们。 审阅者可以调查组织中扫描的电子邮件、Microsoft Teams、Yammer 或第三方通信，并采取适当的措施确保他们符合组织的邮件标准。

Microsoft 365 中的通信合规性策略可帮助你克服许多与合规性以及内部和外部通信相关的现代挑战，包括：

- 扫描不断增加的通信通道类型
- 不断增加的邮件数据量
- 监管实施和罚款风险

此外，IT 管理员和合规性管理团队之间可能职责分离。 通信合规性支持在策略配置与邮件调查和审阅之间分离。 例如，组织的 IT 组可能负责设置通信合规性角色权限、组和策略，而审核员和审阅者可能负责邮件会审、审阅和缓解操作。

有关通信合规性的快速概述，请参阅 Microsoft Mechanics 频道上的 [Microsoft 365](https://youtu.be/z33ji7a7Zho) 中的"检测工作场所的骚扰和沟通合规性 ["视频](https://www.youtube.com/user/OfficeGarageSeries)。

## <a name="scenarios-for-communication-compliance"></a>通信合规性方案

通信合规性策略可帮助在几个重要的合规性方面查看组织中的消息：

- **公司策略**

    用户必须在其所有与业务相关的通信中遵守可接受的使用、安全标准和其他公司策略。 通信合规性策略可以检测策略匹配项，并帮助您采取纠正措施来帮助缓解这些类型的事件。 例如，您可以扫描组织中用户的通信，以发现潜在的人力资源问题，如骚扰或使用不当或冒犯性语言。

- **风险管理**

    组织负责分发整个基础结构和企业网络系统的所有通信。 使用通信合规性策略帮助识别和管理潜在的法律曝光和风险，有助于在风险可能损害公司运营之前将风险降至最低。 例如，您可以扫描组织中邮件中有关机密项目（如即将进行的收购、合并、收益披露、重组或领导团队变更）的未经授权的通信和利益冲突。

- **法规合规性**

    大多数组织必须遵守某种类型的法规合规性标准，作为其正常操作程序的一部分。 这些法规通常要求组织对适合其行业的邮件实施某种类型的监管或监督流程。 金融行业监管局 (FINRA) 规则 3110 是要求组织制定监管程序以扫描用户通信及其所参与业务类型的一个很好的示例。 另一个示例可能是，需要检查组织中代理经销商的通信，以防范潜在资金非法交易、内部交易、串通或中间人活动。 通信合规性策略通过提供一个过程来扫描并报告公司通信，可帮助组织满足这些要求。 有关支持金融组织的信息，请参阅美国银行市场和资本市场的关键合规性 [和安全性注意事项](../solutions/financial-services-secure-collaboration.md)。

## <a name="key-feature-areas"></a>主要功能区域

Microsoft 365 中的通信合规性提供了几个重要功能，可帮助解决邮件平台上的合规性问题：

- 智能可自定义模板
- 灵活的修正工作流
- 可操作见解

![通信合规性主页](../media/communication-compliance-home.png)

### <a name="intelligent-customizable-templates"></a>智能可自定义模板

通信合规性中的智能可自定义模板允许你应用机器学习，以智能检测组织中通信冲突。

- **可自定义的预配置模板**：新的策略模板可帮助解决最常见的通信风险。 通过预定义的反骚扰和冒犯性语言、敏感信息、利益冲突以及法规合规性模板，初始策略创建和后续更新现在更快。
- **新的机器学习支持**：内置威胁、欺骗、亵渎和图像分类器有助于减少扫描邮件 [](classifier-get-started-with.md)中的误报，从而节省审阅者在调查和修正过程中的时间。
- **改进的条件** 生成器：现在，配置策略条件已简化为策略向导中的单一集成体验，从而减少了对策略应用条件方式的混淆。

### <a name="flexible-remediation-workflows"></a>灵活的修正工作流

内置修正工作流允许你快速识别组织中策略匹配的邮件并采取措施。 以下新功能提高了调查和修正活动的效率：

- **灵活的修正工作流**：新的修正工作流可帮助你快速对策略匹配采取行动，包括将邮件上报给其他审阅者以及向策略匹配的用户发送电子邮件通知的新选项。
- **对话线程**：消息现在按原始消息和所有关联的回复消息进行直观分组，从而在调查和修正操作期间为您提供更好的上下文。
- **关键字突出显示**：邮件文本视图中突出显示了字词匹配策略条件，以帮助审阅者快速查找和修正策略警报。
- **精确和接近重复的检测**：除了扫描与通信合规性策略完全匹配的字词外，几乎重复检测组在文本上相似的术语和消息一起，以帮助加快你的审阅过程。
- **OCR (OCR)  (预览版) ：** 扫描、检测和调查嵌入或附加到电子邮件或 Microsoft Teams 聊天消息的图像中的打印和手写文本。
- **新筛选器**：使用多个字段的邮件筛选器更快地调查和修正策略警报，包括发件人、收件人、日期、域等。
- **改进的邮件视图**：现在使用新的邮件源、文本和注释视图可以更快地执行调查和修正操作。 现在可以查看邮件附件，以在采取修正操作时提供完整上下文。
- **用户历史记录视图**：所有用户邮件修正活动（如策略匹配项的过去通知和升级）的历史视图现在为审阅者提供了修正工作流过程中更多的上下文。 用户的第一次或重复策略匹配项实例现已存档且易于查看。
- **模式检测到通知**：许多攻击和威胁操作会随着时间的过去发生，并且涉及用户重新出现相同行为的实例。 警报详细信息中显示的新模式检测到的通知有助于引起关注这些警报和这种类型的行为。
- **显示"翻译"视图**：使用修正工作流中的翻译支持快速调查八种语言的邮件详细信息。 其他语言的邮件将自动转换为审阅者的显示语言。

### <a name="actionable-insights"></a>可操作见解

警报、策略匹配、操作和趋势的新交互式仪表板可帮助您快速查看组织中挂起和已解决警报的状态。

- **主动智能警报**：需要立即注意的策略匹配的警报包括按严重性排序的待处理项目的新仪表板，以及发送给指定审阅者的新自动电子邮件通知。
- **交互式仪表板**：新仪表板显示策略匹配项、挂起和已解决的操作以及用户和策略的趋势。
- **审核支持**：可以轻松地从 Microsoft 365 合规中心导出策略和审阅活动的完整日志，以帮助支持审核审核请求。

## <a name="integration-with-microsoft-365-services"></a>与 Microsoft 365 服务集成

通信合规性策略跨多个通信渠道扫描和捕获邮件，以帮助您快速查看和修正合规性问题：

- **Microsoft Teams：** 作为独立频道源或其他 Microsoft 365 服务的通信合规性支持公共和专用 [Microsoft Teams](/MicrosoftTeams/Teams-overview) 频道的聊天通信和个人聊天。 当你选择要在通信合规性策略中监督的用户和组时，你需要手动添加单个用户、通讯组或特定的 Microsoft Teams 频道。
- **Exchange Online：Microsoft** 365 组织中托管在 [Exchange Online](/Exchange/exchange-online) 上的所有邮箱都有资格进行扫描。 与通信合规性策略条件匹配的电子邮件和附件可立即用于监控和合规性报告。 Exchange Online 现在是可选源频道，在通信合规性策略中不再需要。
- **Yammer：Yammer** 中的私人消息和公共 [](/yammer/yammer-landing-page)社区对话在通信合规性策略中受支持。 Yammer 是可选通道，必须位于 [本机模式下](/yammer/configure-your-yammer-network/overview-native-mode) 以支持邮件和附件扫描。
- **Skype for Business Online**：通信合规性策略支持扫描 Skype [for Business Online](/SkypeForBusiness/skype-for-business-online)中的聊天通信和相关附件。
- **第三方源**：可以扫描来自第三 [](archiving-third-party-data.md)方源的邮件，以检查导入到 Microsoft 365 组织中邮箱的数据。 通信合规性支持与多个热门平台（包括 Instant Bloomberg 和其他平台）的连接。

若要了解有关通信合规性策略中的消息传递通道支持的信息，请参阅 [受支持的通信类型](communication-compliance-feature-reference.md#supported-communication-types)。

## <a name="workflow"></a>工作流

通信合规性可帮助你解决与遵守内部策略和法规合规性要求相关的常见问题。 借助重点策略模板和灵活的工作流，您可以使用可操作见解快速解决检测到的合规性问题。

识别和解决 Microsoft 365 中通信合规性的合规性问题使用以下工作流：

![通信合规性工作流](../media/communication-compliance-workflow.png)

### <a name="configure"></a>配置

在此工作流步骤中，确定合规性要求并配置适用的通信合规性策略。 策略模板是一种很好的方法，不仅可以快速配置新的合规性策略，还可以根据要求的变化快速修改和更新策略。 例如，在为组织中所有用户配置策略之前，你可能希望快速测试策略中针对少数用户的通信使用冒犯性语言和反冒犯性。

>[!Important]
>默认状态下，全局管理员没有对通信合规性功能的访问权限。 若要启用通信合规性功能的权限，请参阅["在组织中提供通信合规性"。](communication-compliance-configure.md#step-1-required-enable-permissions-for-communication-compliance)

可以从 Microsoft 365 合规中心中的以下策略模板中选择：

- **冒犯性** 或冒犯性语言：使用此模板快速创建一个策略，该策略使用内置分类器自动检测可能被视为滥用或冒犯性的内容。
- **敏感信息**：使用此模板快速创建策略，以扫描包含已定义的敏感信息类型或关键字的通信，以帮助确保重要数据不会与不应具有访问权限的人共享。
- **法规合规性**：使用此模板快速创建一个策略，以扫描通信中是否引用了与法规标准相关的标准财务条款。
- **冲突：** 使用此模板可以快速创建一个策略来监视两个组或两个用户之间的通信，以帮助避免兴趣冲突。
- **自定义策略**：使用此模板配置特定通信通道、单个检测条件以及组织中要监视和查看的内容量。

### <a name="investigate"></a>调查

在此步骤中，您将深入了解检测到与通信合规性策略相匹配的问题。 此步骤包括 Microsoft 365 合规中心提供的以下操作：

- **警报**：当邮件与策略条件匹配时，将自动生成警报。 对于每个警报，你可以看到状态、严重性、检测到的时间，以及是否分配了高级电子数据展示案例及其状态。 新警报显示在通信合规性主页和警报页面上，并按严重性顺序列出。
- **问题管理**：对于每个警报，你可以采取调查操作来帮助修正邮件中检测到的问题。
- **文档审阅**：在调查问题期间，可以使用邮件的几个视图来帮助正确评估检测到的问题。 视图包括对话摘要、纯文本、批注和通信对话的详细信息视图。
- **查看用户活动历史记录**：查看策略匹配项的用户邮件活动和修正操作（如过去的通知和升级）的历史记录。
- **筛选器**：使用筛选器（如发件人、收件人、日期和主题）快速缩小要查看的邮件通知范围。

### <a name="remediate"></a>修正

下一步是使用下列选项修正已调查的通信合规性问题：

- **解决**：查看问题后，可以通过解决警报进行修正。 解决警报会将其从挂起的警报队列中删除，并且该操作将保留为匹配策略的"已解决"队列中的条目。 将警报标记为误报、向用户发送有关警报的通知或打开警报的新案例后，将自动解决警报。
- 标记 **邮件**：作为问题解决的一部分，可以将检测到的邮件标记为合规、不符合或与组织的策略和标准相关，或值得怀疑。 标记可帮助你针对升级或作为其他内部审阅过程的一部分进行微筛选策略警报。
- **通知用户**：通常，用户意外或无意违反通信合规性策略。 可以使用通知功能向用户提供警告通知并解决问题。
- **上报给另** 一个审阅者：有时，问题的初始审阅者需要其他审阅者的输入来帮助解决事件。 作为解决方案过程的一部分，可以轻松地将邮件问题上报给组织其他领域的审阅者。
- **标记为误报**：错误地检测为合规性策略匹配项的邮件偶尔会进入审阅过程。 可以将这些类型的警报标记为误报，并自动解决问题。
- **在 Teams (预览**) 中删除消息：在 Microsoft Teams 频道或个人和群聊消息中显示不当消息可能会被删除。 删除的不当邮件将替换为邮件因违反策略而被删除的通知。
- **上报以进行调查**：在最严重的情况下，可能需要与组织的其他审阅者共享通信合规性信息。 通信合规性与其他 Microsoft 365 合规性功能紧密集成，可帮助你解决端到端风险。 通过升级案例进行调查，你可以将案例的数据和管理转移到 Microsoft 365 中的高级电子数据展示。 高级电子数据展示提供了端到端工作流，用于保留、收集、审阅、分析和导出响应组织内部和外部调查的内容。 它允许法律团队管理整个法定保留通知工作流。 若要详细了解高级电子数据展示事例，请参阅 Microsoft [365](overview-ediscovery-20.md)中的高级电子数据展示概述。

### <a name="monitor"></a>监视

跟踪和管理由通信合规性策略标识的合规性问题涉及整个工作流过程。 随着警报的生成以及调查和修正操作的实施，现有策略可能需要审阅和更新，并且可能需要创建新策略。

- **监视并报告**：使用统一审核日志中记录的通信合规性仪表板小组件、导出日志和事件，持续评估和改进合规性状态。
