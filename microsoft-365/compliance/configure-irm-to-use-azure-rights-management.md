# <a name="configure-irm-to-use-azure-rights-management"></a>将 IRM 配置为使用 Azure 权限管理

随着 Office 365 邮件加密和 OME (新功能的发布) ，不再需要单独设置 IRM。 Microsoft 不建议使用旧版 OME 和 IRM 和 Azure 权限管理设置新部署。 有关新的 OME 功能详细信息，请参阅 Office [365 邮件加密常见问题解答](./ome-faq.md)。 如果你已准备好在组织中开始使用新的 OME 功能，请参阅设置基于 Azure 信息保护构建的新 [Office 365 邮件加密功能](./set-up-new-message-encryption-capabilities.md)。
