## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="2aeac-110">如何标识重复邮件</span><span class="sxs-lookup"><span data-stu-id="2aeac-110">How duplicate messages are identified</span></span>

- <span data-ttu-id="2aeac-112">**InternetMessageId** - 此属性指定电子邮件的 Internet 邮件标识符，该标识符是引用特定邮件的特定版本的全局唯一标识符。</span><span class="sxs-lookup"><span data-stu-id="2aeac-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="2aeac-113">此 ID 由发送邮件的发件人的电子邮件客户端程序或主机电子邮件系统生成。</span><span class="sxs-lookup"><span data-stu-id="2aeac-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="2aeac-114">如果用户向多个收件人发送邮件，则每个邮件实例的 Internet 邮件 ID 都相同。</span><span class="sxs-lookup"><span data-stu-id="2aeac-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="2aeac-115">对原始邮件的后续修订将收到不同的邮件标识符。</span><span class="sxs-lookup"><span data-stu-id="2aeac-115">Subsequent revisions to the original message will receive a different message identifier.</span></span>

<span data-ttu-id="2aeac-124">在电子数据展示导出过程中，将针对符合搜索条件的每封邮件比较这三个属性。</span><span class="sxs-lookup"><span data-stu-id="2aeac-124">During the eDiscovery export process, these three properties are compared for every message that matches the search criteria.</span></span> <span data-ttu-id="2aeac-125">如果两条或多条 (邮件的这些属性相同) 则确定这些邮件是重复的，如果启用了重复数据删除，则只会导出邮件的一个副本。</span><span class="sxs-lookup"><span data-stu-id="2aeac-125">If these properties are identical for two (or more) messages, those messages are determined to be duplicates and the result is that only one copy of the message will be exported if de-duplication is enabled.</span></span> <span data-ttu-id="2aeac-126">导出的邮件称为"源项目"。</span><span class="sxs-lookup"><span data-stu-id="2aeac-126">The message that is exported is known as the "source item".</span></span> <span data-ttu-id="2aeac-127">有关重复邮件的信息包含在 **Results.csvManifest.xml导出** 的搜索结果中包含的一些报告。</span><span class="sxs-lookup"><span data-stu-id="2aeac-127">Information about duplicate messages is included in the **Results.csv** and **Manifest.xml** reports that are included with the exported search results.</span></span> <span data-ttu-id="2aeac-128">在Results.csv **文件中** ，通过"复制到项目"列中有一个值 **来标识重复** 邮件。</span><span class="sxs-lookup"><span data-stu-id="2aeac-128">In the **Results.csv** file, a duplicate message is identified by having a value in the **Duplicate to Item** column.</span></span> <span data-ttu-id="2aeac-129">此列中的值与导出的邮件的 **"项目** 标识"列中的值匹配。</span><span class="sxs-lookup"><span data-stu-id="2aeac-129">The value in this column matches the value in the **Item Identity** column for the message that was exported.</span></span>
