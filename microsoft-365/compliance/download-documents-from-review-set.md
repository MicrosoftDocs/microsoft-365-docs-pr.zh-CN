# <a name="download-selected-documents-from-a-review-set"></a>从审阅集中下载选定的文档

下载提供了一种从以本机格式的审阅集下载内容的简单方法。 高级电子数据展示中的下载工具利用浏览器的数据传输功能。 下载准备就绪后，将显示浏览器提示。 使用此方法下载的文件将压缩到容器文件中，并将包含项目级文件。 这意味着，如果您选择下载附件，您将收到包含附件的电子邮件。 同样，如果导出嵌入在 Word 文档中的 Excel 电子表格，则下载时将包含包含嵌入式 Excel 电子表格的 Word 文档。 下载项目时，将保留上次修改的数据属性，并可将其视为文件属性。

若要从高级电子数据展示事例中的审阅集下载内容，请先选择要下载的文件，然后选择 "**操作项目**  >  **下载**"。
