# <a name="export-case-data-in-advanced-ediscovery-classic"></a>在高级电子数据展示和经典 (导出) 

> [!NOTE]
> 若要使用高级电子数据展示，组织必须订阅随附高级合规性加载项的 Office 365 E3，或订阅 E5。如果没有此计划，但又要试用高级电子数据展示，可以[注册 Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
高级电子数据展示中的导出过程允许导出高级电子数据展示内容和结果，以用于外部审阅。 
  
## <a name="guidelines-for-exporting-data"></a>导出数据指南

- 例如，您可以查看从高级电子数据展示生成的导出数据，以将相关元素分发给文档审阅团队或导出到外部文档审阅工具。
    
- 您可以在高级电子数据展示处理期间和之后随时生成和处理导出输出。 在标准方案中，在相关性培训和批计算成功完成后，会获得显著结果。 例如，您可以生成 1，000 个最相关的文件的示例，并评估高级电子数据展示性能，然后再继续。
    
- 将大量数据导出到数据库会对所需的数据库大小和高级电子数据展示数据库所需的配置产生显著影响。
