# <a name="export-case-data-in-advanced-ediscovery"></a>在高级电子数据展示中导出事例数据

有三种方法可从审阅集中导出数据：

**下载：** 使用浏览器) 一小组的本机文件下载 (。 这是导出少量数据集的最快方法。 此方法保留本机文件名称。 有关详细信息，请参阅 [从审阅集下载文档](download-documents-from-review-set.md)。

**导出：** 自定义要导出的数据。 这包括导出文件元数据、本机文件、文本文件和已保存到 PDF 文件的编辑文档。 在导出的数据准备好后，可以将数据从 " **导出** " 选项卡下载到本地计算机。 有关详细信息，请参阅 [从审阅集导出文档](export-documents-from-review-set.md)

**添加到另一个审阅集：** 将数据从一个审阅集复制到不同的审阅集。 有关详细信息，请参阅 [将数据从一个评审集添加到另一个评审集](add-data-to-review-set-from-another-review-set.md)。

> 在 Microsoft 365 中，将对数据进行哈希运算，并在输出文件中提供这些哈希以供验证之用。 这由审核日志和报告功能（如集合统计信息、负载设置报告和导出报告 (）补充，其中包括导出加载文件) 。
