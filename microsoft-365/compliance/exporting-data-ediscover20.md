# <a name="export-case-data-in-advanced-ediscovery"></a>在高级电子数据展示中导出事例数据

有三种方法可从审阅集中导出数据：

**下载：** 下载（通过使用浏览器）一组较小的本机文件。 这是导出少量数据集的最快方法。 此方法保留本机文件名称。

**导出：** 自定义要导出的数据。 这包括导出文件元数据、本机文件、文本文件和已保存到 PDF 文件的编辑文档。 将导出的数据上载到 Azure 存储位置后，可以将其下载到本地计算机。 有关详细信息，请参阅： 

   - [从审阅集中导出文档](export-documents-from-review-set.md)

   - [下载导出作业](download-export-jobs.md)

**添加到另一个审阅集：** 将数据从一个审阅集复制到不同的审阅集。 有关详细信息，请参阅[将数据从一个评审集添加到另一个评审集](add-data-to-review-set-from-another-review-set.md)。
