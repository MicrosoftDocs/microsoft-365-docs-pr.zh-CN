# <a name="information-barriers-in-microsoft-365"></a>Microsoft 365 中的信息障碍

Microsoft 365 支持跨组和组织进行通信和协作，并支持在必要时限制特定用户组之间的通信和协作的方法。 这可能包括要限制两个组之间的通信和协作以避免组织中发生利益冲突的情况或场景。 如果您需要限制组织内部的某些人之间的通信和协作，以保护内部信息，也可能会出现这种情况。

在 Microsoft 团队、SharePoint Online 和 OneDrive for Business 中支持信息障碍。 合规性管理员或信息屏障管理员可以定义策略以允许或阻止 Microsoft 团队中的用户组之间的通信。 信息屏障策略可用于以下情况：

- 第一天的用户 trader 组不应与市场营销团队进行通信或与之共享文件
- 从事机密公司信息的财务人员不应与组织内的特定组进行通信或共享文件
- 具有贸易秘密材料的内部团队不应与组织内特定组中的人员进行呼叫或联机聊天
- 研究团队应仅与产品开发团队进行在线通话或聊天
