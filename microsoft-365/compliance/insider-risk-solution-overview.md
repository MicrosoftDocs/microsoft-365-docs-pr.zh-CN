# <a name="insider-risk-solutions-in-microsoft-365"></a>Microsoft 365 中的内幕风险解决方案

内幕风险是新式工作场所中安全性和合规性专家的首要考虑因素之一。 行业研究表明，内幕风险通常与特定的用户事件或活动相关联。 针对这些风险保护组织可能是确定和难以缓解的难题。 内幕风险包括各种方面的漏洞，并可能导致贵组织出现主要问题，包括丢失知识产权和 workplace 骚扰等。 下图概括了常见的内幕风险：

![内幕风险威胁](../media/ir-solution-threats.png)

Microsoft 365 风险防护功能已设计并内置到我们的内幕风险产品和解决方案中。 这些解决方案协同工作，并使用高级服务和第三方指示器来帮助您快速确定、会审和操作风险活动。 大多数解决方案为您的数据分析师和调查人员提供了综合的检测、警报和修正工作流，以用于快速操作和最大限度地减少这些风险。

## <a name="microsoft-365-insider-risk-solutions"></a>Microsoft 365 内幕风险解决方案

为了帮助您的组织防御内幕风险，请使用以下 Microsoft 365 功能和功能。

| **解决方案/功能** | **说明** | **许可** |
| :------------------------ | :-------------- | :------------ |
| 通信合规性 | 通信合规性通过帮助您在组织中检测、捕获和操作不适当的邮件来帮助最大限度地减少通信风险。 | Microsoft 365 E5 <br> Microsoft 365 E3 订阅 + Microsoft 365 E5 内幕风险管理加载项 <br> Microsoft 365 E3 订阅 + Microsoft 365 E5 合规性加载项 |
| 内部风险管理 | 内幕风险管理通过使您能够检测、调查和操作组织中的恶意和意外活动，帮助最大限度地减少内部风险。 | Microsoft 365 E5 <br> Microsoft 365 E3 订阅 + Microsoft 365 E5 内幕风险管理加载项 <br> Microsoft 365 E3 订阅 + Microsoft 365 E5 合规性加载项 |
| 信息屏障 | 信息障碍允许您限制两个内部组之间的通信和协作，以避免组织中发生利益冲突。  | Microsoft 365 E5/A5 <br> Office 365 E5/A5 <br> Office 365 高级合规版 <br> Microsoft 365 合规性 E5/A5 <br> Microsoft 365 内幕风险管理 |
| 特权访问管理 | 特权访问管理允许对 Office 365 中的特权 Exchange Online 管理任务进行精确的访问控制。 它可帮助保护您的组织免受使用现有特权管理员帐户的破坏，以访问敏感数据或访问关键配置设置。 | Microsoft 365 E5/A5 <br> Office 365 E5/A5 <br> Microsoft 365 E5/A5 合规性 <br> Microsoft 365 E5/A5 信息保护和治理 |

## <a name="deploy-microsoft-365-insider-risk-solutions"></a>部署 Microsoft 365 内幕风险解决方案

为了帮助您的组织防御内幕风险，请设置和部署以下 Microsoft 365 解决方案：

![内幕风险解决方案纵深防御](../media/ir-solution-defense-in-depth.png)

1. 配置和创建 [通信合规性策略](communication-compliance-solution-overview.md)
2. 配置和创建 [内幕风险管理策略](insider-risk-management-solution-overview.md)
3. 可选：配置和创建 [信息屏障策略](information-barriers-solution-overview.md)
4. 可选：启用和配置 [特权访问管理](privileged-access-management-solution-overview.md)
