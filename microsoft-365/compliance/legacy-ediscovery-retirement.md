# <a name="legacy-ediscovery-tools-being-retired"></a>旧的电子数据展示工具即将停用

我们在第一半的2020中淘汰了两个旧的电子数据展示工具：高级电子数据展示 v1.0 和就地电子数据展示和保留。

高级电子数据展示 v1 （也称为 Office 365 高级电子数据展示）是在安全 & 合规中心的电子数据展示事例中，通过单击 "切换到高级电子数据展示" 可访问的高级电子数据展示的版本。  提前的电子数据展示 v1.0 不会影响高级电子数据展示（也称为高级电子数据展示 v2 或 Microsoft 365 中的高级电子数据展示）的最新版本，可在安全 & 合规中心的 "电子数据展示 > 高级电子数据展示" 页上访问。

Exchange 管理中心中的就地电子数据展示和保留工具也会被停用。 此工具用于在 Exchange Online 中搜索、保留和导出邮箱内容。 在 Microsoft 365 合规性中心提供了类似的功能。

即将推出有关停用这些旧电子数据展示工具的详细信息。
