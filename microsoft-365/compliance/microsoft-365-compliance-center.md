# <a name="microsoft-365-compliance-center"></a>Microsoft 365 合规中心

如果你对组织的合规性状态感兴趣，你将喜欢 [Microsoft 365 合规中心](https://compliance.microsoft.com)。 Microsoft 365 合规中心提供对需要管理的数据和工具的轻松访问，以满足组织的合规性需求。

阅读本文以熟悉 Microsoft 365 合规中心、如何获取[](#how-do-i-get-the-compliance-center)它、[常见问题](#frequently-asked-questions)以及你的[下一步](#next-steps)。

[![Microsoft 365 合规中心主页](../media/m365-compliance-center-home.png)](https://compliance.microsoft.com)

## <a name="welcome-to-microsoft-365-compliance"></a>欢迎使用 Microsoft 365 合规性

当你第一次转到 Microsoft 365 合规中心时，你将看到以下欢迎消息：

![Microsoft 365 合规中心简介](../media/m365-compliance-center-welcome-steps.png)

欢迎横幅为你提供了一些有关入门的指针、下一步，并邀请你向我们提供反馈。

## <a name="card-section"></a>“卡”部分

首次访问 Microsoft 365 合规中心时，主页上的卡片部分将一目了然地显示组织在数据合规性方面的情况、可供组织使用的解决方案以及任何活动警报的摘要。

在这里，你可以：

- 查看 **Microsoft 合规性管理器** 卡，它可引导你找到 [合规性管理器](compliance-manager.md) 解决方案。 合规性管理器有助于简化合规性管理方式。 它计算基于风险的分数，以衡量完成建议操作的进度，以帮助降低数据保护和监管标准方面的风险。 它还提供工作流功能和内置控件映射，以帮助您高效地执行改进操作。

    ![合规性管理器卡 Microsoft 365 合规中心](../media/m365-compliance-center-compliance-manager-card.png)

- 查看新的 **解决方案目录** 卡，该卡片链接到可用于帮助您管理 [](microsoft-365-solution-catalog.md)端到端合规性方案的集成解决方案集合。 解决方案的功能和工具可能包括策略、警报、报告等的组合。

    ![解决方案目录卡 Microsoft 365 合规中心](../media/m365-compliance-center-solution-catalog-card.png)

- 查看 **"活动** 警报"卡片，其中包括最活跃警报的摘要，并 [](alert-policies.md)包含一个链接，可在其中查看更详细的信息，如严重性、状态、类别等。

    ![活动警报卡 Microsoft 365 合规中心](../media/m365-compliance-center-active-alerts-card.png)

您还可以使用"添加卡"功能添加其他卡片，例如显示组织的云应用合规性卡，另一张显示有关具有共享文件的用户的数据，以及指向[Cloud App Security](https://docs.microsoft.com/cloud-app-security/)或其他可浏览数据的工具的链接。

![其他合规中心卡详细信息](../media/m365-compliance-center-additional-cards.png)

## <a name="how-do-i-get-the-compliance-center"></a>如何获取合规中心？

- 如果你还没有新的 Microsoft 365 合规中心，你很快就会拥有它。 Microsoft 365 合规中心现已向 Microsoft 365 SKU 客户提供。

- 若要访问 Microsoft 365 合规中心，请作为全局管理员、合规性管理员或合规性数据管理员转到 [https://compliance.microsoft.com](https://compliance.microsoft.com) 并登录。

## <a name="frequently-asked-questions"></a>常见问题解答

**为什么需要我到安全与&中心完成某些任务，例如定义某些策略？**

我们仍在开发 Microsoft 365 合规中心，并未来几个月添加更多功能和解决方案。 同时，安全与合规中心内必须完成一些&任务 [https://protection.office.com](https://protection.office.com) () 。 在这种情况下，将自动定向到可以完成当前任务的位置，例如创建或编辑监督策略。

**为什么我还没有看到新的 Microsoft 365 合规中心？**

首先，确保你拥有相应的许可证和权限。 然后，登录 [https://compliance.microsoft.com](https://compliance.microsoft.com) 。 如果你尚未看到新的合规中心，你很快就会拥有它。

**我的某些合规性功能在 Microsoft 365 合规中心中不可用。我该怎么办？**

我们仍在向 Microsoft 365 合规中心添加功能。 如果找不到内容（如搜索审核日志，请使用安全与合规&中心 [https://protection.office.com](https://protection.office.com) () 。 你的配置会自动保存在现有的安全与&合规中心和新的 Microsoft 365 合规中心中。

To go there， in the Microsoft 365 compliance center， in the navigation pane on the left side of the screen， choose **More resources，** and then， under **Office 365 Security & Compliance Center，** choose **Open.**

## <a name="next-steps"></a>后续步骤

- **访问 Microsoft 合规性管理器** ，查看合规性分数并开始管理组织的合规性。 若要了解更多信息，请参阅 [合规性管理器](compliance-manager.md)。

- **配置内部风险管理策略** 以帮助最大程度地降低内部风险，并使您能够检测、调查和对组织中存在风险的活动采取行动。 请参阅 [内部风险管理](insider-risk-management.md)。

- **查看组织的数据丢失防护策略，** 并在必要时进行必要的更改。 若要了解相关内容，请参阅 [数据丢失防护策略概述](data-loss-prevention-policies.md)。

- **熟悉并设置 Microsoft Cloud App Security。** 请参阅 [快速入门：Microsoft Cloud App Security 入门](https://docs.microsoft.com/cloud-app-security/getting-started-with-cloud-app-security)。

- **了解并创建通信合规性策略** ，以快速识别和修正公司行为准则策略违反情况。 请参阅 [Microsoft 365 中的通信合规性](communication-compliance.md)。

- **经常访问 Microsoft 365** 合规中心，并确保查看出现的任何警报或潜在风险。 转到 [https://compliance.microsoft.com](https://compliance.microsoft.com) 并登录。
