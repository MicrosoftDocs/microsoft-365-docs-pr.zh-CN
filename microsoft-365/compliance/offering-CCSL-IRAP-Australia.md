# <a name="compliance-offering-australian-government-certified-cloud-services-list-ccsl"></a>合规性服务：澳大利亚政府认证云服务列表（CCSL）

## <a name="ccsl-overview"></a>CCSL 概述

认证云服务列表（CCSL）确定已通过澳大利亚政府成功完成 IRAP 评估的云服务，并已由澳大利亚网络安全中心（ACSC）授予认证。 该认证可识别已成功完成、审查和接受通过已注册的信息安全进行的全面评估的评估员，因此所有澳大利亚政府机构都可以使用它。 CCSL 还可以由新西兰政府组织引用，因为在新西兰和澳大利亚的 ISM 中进行了调整。

信息安全注册的评估员程序（IRAP）由 ACSC 进行管理和管理。 IRAP 提供了针对澳大利亚政府策略和准则独立评估系统安全的综合过程。 IRAP 目标是通过重点介绍存储、处理和传达澳大利亚的信息和通信技术基础结构，最大限度地提高澳大利亚联邦、州和本地政府数据的安全性。

## <a name="microsoft-and-ccsl"></a>Microsoft 和 CCSL

Microsoft 的 Azure、Dynamics 365 CRM 和 Office 365 已完成信息安全注册评估员计划（IRAP）评估，并在澳大利亚网络安全中心（ACSC）的 CCSL 上经过认证。 Azure 和 Office 365 服务已认证为两种未分类：散播限制标记（DLM）和受保护的数据。 Dynamics 365 CRM 仅认证了 DLM 数据。

对于每个评估，Microsoft 按照 ACSC 资格鉴定的 IRAP 评估员负责检查 Microsoft IT 运营团队、物理数据中心、入侵检测、加密、跨域和网络安全性所使用的安全控制和流程。范围内服务的访问控制和信息安全风险管理。 IRAP 评估发现，Microsoft 系统体系结构基于合理的安全原则，并且适用的信息安全手册（ISM）控件在评估服务中已就绪且完全有效。

- 在2014中，Azure 作为第一个 IRAP 评估的云服务在澳大利亚，以墨尔本和悉尼的数据中心托管。 这两个数据中心使澳大利亚客户能够控制其客户数据的存储位置，同时还通过在两个位置进行备份而在灾难发生时提供增强的数据耐用性。
- 在早期2015中，Office 365 成为了完成此评估的第一个云生产效率服务。
- 在4月2015，该 .ASD 宣布了 Azure 和 Office 365 的 CCSL 证书，在11月2015的 Dynamics 365
- 在6月2017，.ASD 宣布了 Microsoft Azure 和 Office 365 的 recertification，以提供一组显著扩展的服务。
- 2018年4月，ACSC 宣布了 Azure 和 Office 365 在受保护的分类中的证书。 Microsoft 是第一个也是唯一一个实现此级别的认证的公共云提供商。

Microsoft 的服务认证有助于向政府及其合作伙伴的公共部门客户提供 Microsoft 在处理、存储和传输 DLM 方面的适当且有效的安全控制措施。受保护的保密信息。 这包括澳大利亚的大多数政府、医疗保健和教育数据。

了解[CCSL 在 Microsoft 云上的优势](https://go.microsoft.com/fwlink/p/?linkid=2086844)。

## <a name="frequently-asked-questions"></a>常见问题

**IRAP 标准适用于哪些人？**

此标准适用于所有使用云服务的澳大利亚联邦、州和本地政府机构。 新的新西兰政府机构要求遵守与澳大利亚 ISM 类似的标准，因此他们也可以使用 IRAP 评估。

**我可以在组织的认证过程中使用 Microsoft 合规性吗？**

是。 如果您的组织要求或正在寻求与 ISM 的资格鉴定，则可以在合规性评估中使用 Azure、Office 365 和 Dynamics 365 CRM 的认证。 不过，您需要参与评估员，以评估在 Microsoft 平台上部署的实施，以及您自己的组织中的控件和流程。

**从哪里开始我的组织自己的合规性工作？**

从[IRAP 工具包](https://www.cyber.gov.au/node/291)开始，并在其后面提供以下信息：

- [什么是 IRAP 评估](https://www.asd.gov.au/infosec/irap/irap_assessments.htm)
- [谁提供 IRAP 培训？](https://www.cyber.gov.au/irap/training)
- [合规性和不合规要求信息](https://www.cyber.gov.au/irap/compliance)
