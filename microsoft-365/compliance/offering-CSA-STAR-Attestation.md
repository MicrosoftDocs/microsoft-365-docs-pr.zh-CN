## <a name="csa-star-attestation-overview"></a><span data-ttu-id="9c341-105">CSA STAR 证明概述</span><span class="sxs-lookup"><span data-stu-id="9c341-105">CSA STAR attestation overview</span></span>

<span data-ttu-id="9c341-106">云安全联盟 (CSA) 维护安全、信任及保证注册项目 (STAR)，这是一个可公开访问的免费注册项目，云服务提供商 (CSP) 可在其中发布他们与 CSA 相关的评估。</span><span class="sxs-lookup"><span data-stu-id="9c341-106">The Cloud Security Alliance (CSA) maintains the Security, Trust & Assurance Registry (STAR), a free, publicly accessible registry where cloud service providers (CSPs) can publish their CSA-related assessments.</span></span> <span data-ttu-id="9c341-107">STAR 包括三个级别的保证，分别与 CSA 云控制矩阵 (CCM) 中的控制目标对应。</span><span class="sxs-lookup"><span data-stu-id="9c341-107">STAR consists of three levels of assurance aligned with control objectives in the CSA Cloud Controls Matrix (CCM).</span></span> <span data-ttu-id="9c341-108">（CCM 涵盖 16 个域的基本安全原则，可帮助云客户对云服务的整体安全风险进行评估。）：</span><span class="sxs-lookup"><span data-stu-id="9c341-108">(The CCM covers fundamental security principles across 16 domains to help cloud customers assess the overall security risk of a cloud service.):</span></span>

- <span data-ttu-id="9c341-109">第 1 个级别：STAR 自我评估</span><span class="sxs-lookup"><span data-stu-id="9c341-109">Level 1: STAR Self-Assessment</span></span>

- <span data-ttu-id="9c341-111">第 3 个级别：STAR 持续监视（CSA 仍在制定计划要求）</span><span class="sxs-lookup"><span data-stu-id="9c341-111">Level 3: STAR Continuous Monitoring (program requirements are still under development by CSA)</span></span>

<span data-ttu-id="9c341-112">STAR 认证包含一项对云提供商安全状况进行的严格独立审计，该审计基于 SOC 2 类型 2 审计，并符合 CCM 标准。</span><span class="sxs-lookup"><span data-stu-id="9c341-112">STAR Attestation involves a rigorous independent audit of a cloud provider’s security posture based on a SOC 2 Type 2 audit with CCM criteria.</span></span> <span data-ttu-id="9c341-113">评估云提供商的产品/服务是否可获得 STAR 证明的独立审核员必须是注册会计师 (CPA)，并且必须拥有云安全知识 (CCSK) 领域的 CSA 证书。</span><span class="sxs-lookup"><span data-stu-id="9c341-113">The independent auditor that evaluates a cloud provider’s offerings for STAR Attestation must be a certified public accountant (CPA) and is required to have the CSA Certificate in Cloud Security Knowledge (CCSK).</span></span>  
  
<span data-ttu-id="9c341-114">SOC 2 类型 2 审核基于美国注册会计师学会 (AICPA) 信任服务原则和标准（包括安全性、可用性、机密性和处理完整性）和 CCM 中的标准。</span><span class="sxs-lookup"><span data-stu-id="9c341-114">A SOC 2 Type 2 audit is based on American Institute of Certified Public Accountants (AICPA) Trust Services Principles and Criteria, including security, availability, confidentiality, and processing integrity, and the criteria in the CCM.</span></span> <span data-ttu-id="9c341-115">STAR 证明提供了审核员关于 Microsoft 云服务中 SOC 2 控制措施的设计适用性和运行有效性的调查结果。</span><span class="sxs-lookup"><span data-stu-id="9c341-115">STAR Attestation provides an auditor’s findings on the design suitability and operating effectiveness of SOC 2 controls in Microsoft cloud services.</span></span> <span data-ttu-id="9c341-116">目标是满足上面提及的 AICPA 标准和 CCM 中规定的要求。</span><span class="sxs-lookup"><span data-stu-id="9c341-116">The objective is to meet both the AICPA criteria mentioned above and requirements set forth in the CCM.</span></span>
