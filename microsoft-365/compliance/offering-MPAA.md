## <a name="microsoft-and-mpaa"></a><span data-ttu-id="c4f14-108">Microsoft 和 MPAA</span><span class="sxs-lookup"><span data-stu-id="c4f14-108">Microsoft and MPAA</span></span>

<span data-ttu-id="c4f14-109">2016 年 2 月，Microsoft Azure 成为首个成功完成由 MPAA 独立审核员执行的正式评估的超大规模多租户云服务，该评估符合所有三个 MPAA 内容安全最佳做法框架：共同准则、应用准则和云安全性准则。</span><span class="sxs-lookup"><span data-stu-id="c4f14-109">In February 2016, Microsoft Azure became the first hyperscale, multitenant cloud service to successfully complete a formal assessment by independent MPAA auditors and comply with all three of the MPAA content security best practices frameworks: Common, Application, and Cloud Security Guidelines.</span></span>

<span data-ttu-id="c4f14-110">MPAA 评估包括共同准则中的 48 个安全主题，以及应用准则和云安全性准则中的另外 6 个主题。</span><span class="sxs-lookup"><span data-stu-id="c4f14-110">The MPAA assessment covers 48 security topics in the Common Guidelines, and an additional six in the Application and Cloud Security Guidelines.</span></span> <span data-ttu-id="c4f14-111">这些内容都建立在 ISO/IEC 27001 和 NIST 800-53 等行业公认的安全标准之上，符合云安全联盟 (CSA) 云控制矩阵等的最佳做法。</span><span class="sxs-lookup"><span data-stu-id="c4f14-111">These are built on industry-accepted security standards such as ISO/IEC 27001 and NIST 800-53, and are aligned to best practices, such as the Cloud Security Alliance (CSA) Cloud Controls Matrix.</span></span>

<span data-ttu-id="c4f14-112">Azure 合规性的正式评估意味着，与主流电影公司有业务往来的公司可使用 Azure 来帮助降低通常与安全创作、管理、存储和内容分发相关的 IT 成本，并同时帮助遵守 MPAA 要求。</span><span class="sxs-lookup"><span data-stu-id="c4f14-112">The formal assessment of Azure compliance means that companies who do business with major studios can use Azure to help reduce the IT costs that are normally associated with the secure creation, management, storage, and distribution of content — all while complying with MPAA requirements.</span></span> <span data-ttu-id="c4f14-113">Azure 媒体服务、存储、虚拟网络以及 30 多个其他服务可在云中提供内容工作流引擎，这比传统本地部署的生产流程更加安全且更具伸缩性，而且能够更加有效地保护下游媒体资产。</span><span class="sxs-lookup"><span data-stu-id="c4f14-113">Azure Media Services, Storage, Virtual Networks, and more than 30 other services provide a content workflow engine in the cloud that is more secure and scalable than traditional on-premises production processes and more effective at protecting media assets downstream.</span></span>
