## <a name="meity-overview"></a><span data-ttu-id="bbaba-105">MeitY 概述</span><span class="sxs-lookup"><span data-stu-id="bbaba-105">MeitY overview</span></span>

<span data-ttu-id="bbaba-106">电子产品和信息技术（MeitY）是印度政府的机构，为所有政府和州公共部门组织提供策略准则。</span><span class="sxs-lookup"><span data-stu-id="bbaba-106">The Ministry of Electronics and Information Technology (MeitY), an agency of the government of India, provides policy guidelines to all government and state public sector organizations.</span></span> <span data-ttu-id="bbaba-107">在管控行业（如金融服务和电信）中，专用部门的组织也经常采用其指南。</span><span class="sxs-lookup"><span data-stu-id="bbaba-107">Its guidelines are also frequently adopted by private sector organizations in regulated industries, like financial services and telecommunications.</span></span>

<span data-ttu-id="bbaba-110">获得资格鉴定后，云服务提供商将列在政府云服务目录中，公共事业部门可以对其进行比较和购买获得资格鉴定的云服务。</span><span class="sxs-lookup"><span data-stu-id="bbaba-110">Once accredited, cloud service providers are listed in a government cloud services directory where public sector organizations can compare and procure accredited cloud services.</span></span> <span data-ttu-id="bbaba-111">目录是[MeghRaj 云计划](https://meity.gov.in/content/gi-cloud-meghraj)（或印度\[政府的印度\]云）的一项服务，它促进了政府中的云计算的使用，并控制了公共事业部门 IT 服务的实施。</span><span class="sxs-lookup"><span data-stu-id="bbaba-111">The directory is a service of the [MeghRaj Cloud Initiative](https://meity.gov.in/content/gi-cloud-meghraj) (or GI \[Government of India\] Cloud), which promotes the use of cloud computing in government, and governs the implementation of public sector IT services.</span></span>
