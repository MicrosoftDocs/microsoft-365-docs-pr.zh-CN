# <a name="plan-for-security-amp-compliance"></a>规划安全 &amp; 合规性

管理安全性和合规性是一种合作关系。 你负责保护你的数据、标识和设备，而 Microsoft 会积极保护 Microsoft 365 服务。 可以将 Microsoft 365 和企业移动性 + 安全性 (EMS) 一起，以帮助你为组织实现适当级别的保护。

引导自己了解 Office 365 信息保护海报中的信息保护功能。 
  
[使用 Microsoft 365 部署针对数据隐私法规的信息保护](../solutions/information-protection-deploy.md?view=o365-worldwide)
  
## <a name="step-2-check-your-secure-score"></a>步骤 2：检查安全分数

设置 Microsoft 365 订阅后，记下起始分数。 安全分数提供了可用于增加分数的配置建议。 目标是了解您可以利用的一些机会来保护环境，这些机会不会对用户的工作效率产生负面影响。
  
- [Office 365 安全分数介绍](../security/mtp/microsoft-secure-score.md)
