# <a name="run-reports-in-advanced-ediscovery-classic"></a>在高级电子数据展示和经典 (中运行) 

> [!NOTE]
> 若要使用高级电子数据展示，组织必须订阅随附高级合规性加载项的 Office 365 E3，或订阅 E5。如果没有此计划，但又要试用高级电子数据展示，可以[注册 Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
本主题介绍如何在高级电子数据展示中运行报告。
  
## <a name="running-reports"></a>运行报告

可以下载包含所选进程的报告的 .csv 文件。
  
1. 在" **报告** "选项卡中，从"报告名称"列表中选择 **一** 个选项。 从三 **个报告名称** 选项中进行选择 **：相关性决定**、主题 **列表** 或 **标记文件**。
    
    ![电子数据展示分析报告](../media/f16aee7a-508f-4acc-99bc-a2c8dec01312.png)
  
2. 可以设置可用参数以及排序和筛选选项，具体取决于所选报告。 
    
3. 单击 **"下载 CSV"。** 生成并下载请求的报告。
