# <a name="run-the-process-module-and-load-data-in-advanced-ediscovery-classic"></a>运行流程模块，在高级电子数据展示和经典 (加载) 

> [!NOTE]
> 若要使用高级电子数据展示，组织必须订阅随附高级合规性加载项的 Office 365 E3，或订阅 E5。如果没有此计划，但又要试用高级电子数据展示，可以[注册 Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
本节介绍高级电子数据展示流程模块的功能。 
  
除了文件数据之外，元数据（如文件类型、扩展名、位置或路径、创建日期和时间、作者、保管人和主题）还可以加载到高级电子数据展示中，并针对每个案例进行保存。 某些元数据由高级电子数据展示计算，例如加载本机文件时。 
  
高级电子数据展示提供系统元数据值，例如近重复分组或相关性分数。 管理员可以添加其他元数据，如文件注释。 
  
## <a name="running-process"></a>正在运行的进程

> [!NOTE]
> 批处理号在进程期间分配给文件，以允许跟踪文件。 批处理号还允许标识用于重新处理选项的进程批处理。 其他筛选器可用于按批次编号和会话进行筛选。 
  
执行以下步骤以运行 Process。
  
1. [打开安全 &amp; 合规中心](go-to-the-securitycompliance-center.md) 。 
    
2. 转到 **"搜索 &amp; 调查** \> **电子数据展示"，** 然后单击"**转到高级电子数据展示"。**
    
3. 在高级电子数据展示中，在显示的事例页面中选择相应的事例，然后单击 **"转到事例"。**
    
4. 在 **"** \> **准备** \> **进程设置**"中，从可用容器列表中选择一个容器。
    
    ![单击"进程"将搜索结果添加到案例](../media/50bdc55c-d378-4881-b302-31ef785fa359.png)
  
5. 如果要 **将** 容器添加为种子文件或预标记文件，请单击"高级设置..."。 
    
    使用种子文件加快针对低丰富度问题的培训 (通常为 2% 或更少) 。 对于种子文件，建议选择各种明显相关的文件，并处理每个问题大约 20-50 个种子 (种子文件过多可能会扭曲相关性结果) 。 种子文件应由将训练该问题的同一个人审阅。
    
    使用预标记文件自动执行相关性培训。 应标记至少 1，500 个文件，并保持与非相关文件的相关比例与添加到相关性的集合中的比例相同。 应手动标记这些文件，并且您应该确信标记的质量。
    
    ![用于处理批处理文件的"高级设置"页的屏幕截图](../media/3c25cb78-4484-41e5-bd34-3753c7ab6cf2.jpg)
  
  - 在 **"种子"** 部分： 
    
    选择 **"标记为种子文件** "以将容器标记为种子文件。 还需要从"For issue"下拉列表中选择按问题 **分配** 它们。 从 **"标记** " **下拉列表** 中选择"相关 **"或"** 不相关"。 
    
    > [!NOTE]
    > 将文件设置为 **种子** 后，你无法将它们标记为 **预标记**。 
  
  - 在 **"预标记文件"部分** ： 
    
    选择 **"标记为预标记文件"** 以将容器标记为预标记文件。 还需要从"For issue"下拉列表中按问题 **分配** 它们。 从 **"标记** " **下拉列表** 中选择"相关 **"或"** 不相关"。 
    
    > [!NOTE]
    > 将文件设置为预 **标记后**，不能将它们标记为 **种子**。 
  
  - 在 **"电子邮件标记"** 部分。 设置要标记为种子或预标记的已处理电子邮件的哪一部分。 
    
6. 若要开始，请单击"**进程"。** 完成后，将显示进程结果。
    
7.  (可选) 如果需要将数据源分配给特定保管人，可以在保管人管理中添加和编辑保管人名称，在保管人分配中 \> 分配 **保管** \> **人**。 
    
如果添加到案例，可以再次处理。
