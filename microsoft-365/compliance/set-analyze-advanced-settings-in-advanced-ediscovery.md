# <a name="set-analyze-advanced-settings-in-advanced-ediscovery"></a>设置"分析高级电子数据展示"中的高级设置

> [!NOTE]
> 若要使用高级电子数据展示，组织必须订阅随附高级合规性加载项的 Office 365 E3，或订阅 E5。如果没有此计划，但又要试用高级电子数据展示，可以[注册 Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
高级电子数据展示提供分析模块设置的默认高级参数。 以下过程介绍可指定的设置。
  
1. 在 **"准备 \> 分析 \> 安装程序** "选项卡中 **， (** 页面底部的"高级设置") 。 将显示以下面板。 
    
    ![设置分析高级设置](../media/c9ea3017-e19a-456b-a742-c3d07121a3f6.png)
  
2. 在 **"近重复项"和"电子邮件线程"** 参数中，如有必要，选择以下值：
    
  - **最少字** 数：单词的最小数目，低于此数字时不提交文件进行近重复分析。 
    
  - **最大字** 数：最大字数，超过此数目时不提交文件进行近重复分析。
    
  - **电子邮件相似** 性：两封电子邮件的相似性最低级别。 值始终等于或大于文档相似性。 默认值为 90%。
    
3. 在 **"主题"参数** 中，选中"在主题分析中包括数字"复选框以在"分析"期间处理主题时包括数字。 
    
4. 单击“**保存**”。
