# <a name="themes-in-advanced-ediscovery"></a>高级电子数据展示中的主题

用户如何编写文档？ 它们通常从要在文档中传达的一个或多个想法开始，并使用与想法一致的单词进行撰写。 概念越普遍，与该想法相关的词越常见。 这将通知用户也如何使用文档。 从阅读文档中了解的重要一点是文档试图传达的想法、在何处显示了哪些想法以及这些想法之间的关系。

这可以扩展到用户想要使用一组文档的方式。 他们想要了解哪些想法在集合中，以及哪些文档在谈论这些想法。 此外，如果他们找到感兴趣的特定文档，他们希望能够查看讨论类似想法的文档。

高级电子数据展示中的主题功能尝试通过分析审阅集中讨论的 *主题* 并将主题分配给审阅集中的文档来模拟有关文档的人为原因的情况。 在高级电子数据展示中，主题更进一步，并确定每个文档中的 *主要主题* 。 主主题是在文档中显示的最常用主题。

## <a name="how-does-themes-work"></a>主题的工作原理是什么？

主题功能使用审阅集中的文本分析文档，以分析在审阅集中的所有文档中显示的常见主题。 高级电子数据展示将这些主题分配到它们出现的文档中。 它还将每个主题标记为主题的代表文档中使用的单词。 由于文档可以包含各种类型的主题，因此高级电子数据展示通常会为文档分配多个主题。 文档中显示最醒目的主题被指定为其主要主题。
