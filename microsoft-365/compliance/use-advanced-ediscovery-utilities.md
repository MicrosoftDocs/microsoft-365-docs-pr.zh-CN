# <a name="use-advanced-ediscovery-classic-utilities"></a>使用高级电子数据展示（经典）实用工具

> [!NOTE]
> 若要使用高级电子数据展示，组织必须订阅随附高级合规性加载项的 Office 365 E3，或订阅 E5。如果没有此计划，但又要试用高级电子数据展示，可以[注册 Office 365 企业版 E5 试用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
在高级电子数据展示中显示和可用的实用程序取决于上下文和用户角色。
  
## <a name="case-log"></a>案例日志

案例日志提供了应用程序处理活动的详细列表，可用于跟踪、疑难解答和解决错误和警告。 日志可以在主机或服务器上本地生成和存储，也可以直接发送到电子邮件地址。
  
也可以日志文件客户端的计算机下载该证书。 客户端下载选项可能会根据配置和用户角色启用或禁用。
  
1. 在菜单栏中，单击 **Cogwheel** 图标。 
    
2. 在 **"设置和实用程序 \> 实用程序"** 选项卡中，**选择"案例日志 \> 设置"。**
    
3. 选择 **日志级别** ，如下所示： 
    
  - **标准**：包括基本日志数据。 此选项通常是监视所必需的，除非另有建议，否则应使用此选项。
    
  - **最小**：用于非常大的情况，并且仅返回最新数据。
    
4. 单击 **"运行案例日志"。** 将生成日志并显示路径。 当前和最后一个任务的任务进度信息显示在任务状态窗格中。
    
## <a name="clear-data"></a>清除数据

如果需要删除或重新初始化案例数据，则必须初始化数据库实例。 Clear 数据实用程序从案例数据库、文本文件、案例文件夹和累积结果中删除所有指定的条目。 该函数只能由管理员执行。
  
> [!IMPORTANT]
> 此操作不可反转，并且将清除专家执行的所有相关性标记和分析。 如有必要，保存数据备份。 请谨慎使用此选项。 删除标记和排名文件可能会影响相关性结果。 
  
1. 在菜单栏中，单击 **Cogwheel** 图标。 
    
2. 在 **"设置和实用程序 \> 实用程序"** 选项卡中，选择 **"清除数据 \> 设置"。**
    
3. 为要初始化的信息选择一个选项：
    
  - **相关性**：删除相关性中完成的所有工作，包括负载定义和要加载的文件关联。 它将删除所有示例和标记。
    
  - **近重复项和电子邮件线程**：删除近重复项和电子邮件线程的所有分析信息。
    
  - **主题**：删除与主题相关的数据。
    
  - **导出历史记录**：删除导出批处理的历史记录信息。
    
4. 单击 **"清除数据"。** 将清除案例数据。 当前和最后一个任务的任务进度信息显示在 **任务状态窗格中** 。 
    
## <a name="modify-relevance"></a>修改相关性

本节介绍如何跳过或回滚相关性示例。
  
1. 在菜单栏中，单击 **Cogwheel** 图标。 
    
2. 在 **"设置和实用程序 \> 实用程序"** 选项卡中，选择 **"修改相关性"。**
    
3. 从选项中进行选择： 
    
  - **跳过当前示例 - 对于当前** 用户： This will tag， as **Skip**， all untagged files in the open case sample of the user running the utility. 不会对标记为 Skip 的文件执行相关性 **处理**。
    
  - **跳过当前示例 - 所有打开的示例**：这会将所有用户的所有打开示例中的所有未标记文件标记为 **Skip。** 如果用户当前正在标记示例，则不建议使用此选项。
    
  - **回滚最后一个示例**：将回滚上一个完成的相关性培训示例，而不管该示例是在"计算"过程之前还是之后。 不允许回滚捕获示例。
    
4. 单击 **"执行** "运行。 
    
## <a name="transparency-analysis"></a>透明度分析

透明度分析实用工具可详细查看文件及其分配的相关性分数。 该报告可以用作一种有效性检查，或比较由人工审阅者定义的文件与高级电子数据展示分配的相关性。 
  
除了相关性分数之外，高级电子数据展示还计算并分配考虑关键字上下文的关键字权重。 根据上下文和位置，可以给文件中相同的单词分配不同的权重。 每个关键字都使用从黄色到深橙色以及不同的灰色底纹的颜色强度比例进行标记。 颜色编码用于直观地指示字词对相关性分数的相对正或负贡献。 
  
在多问题案例场景中，可以针对每个问题生成透明度分析报告。
  
1. 在菜单栏中，单击 **Cogwheel** 图标。 
    
2. 在 **"设置和实用程序 \> 实用程序"** 选项卡中，选择 **"透明度分析设置 \> "。**
    
3. 在** 文件 ID **中，输入要处理的文件的文件 ID。
    
4. 在 **"问题** "列表中，选择相关问题。 
    
5. 单击 **"透明度分析"。** 完成后，将显示文件的透明度分析报告，其中显示标记的关键字颜色与总体相关性分数的关联情况。
