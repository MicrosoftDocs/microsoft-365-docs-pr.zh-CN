# <a name="working-with-microsoft-compliance-score-preview"></a>使用 Microsoft 合规性分数（预览）

## <a name="managing-your-workflow-with-improvement-actions"></a>使用改进操作管理工作流

使用合规性分数中的提高操作将集中管理合规性工作流。 改进操作建议建议的操作与数据保护规章和标准一致，并提供详细的实施指南。 您可以将其分配给用户，以执行必要的实现和测试工作。 您还可以存储文档和注释，并在改进操作本身中直接记录状态更新。

## <a name="view-your-improvement-actions"></a>查看改进操作

合规性分数仪表板显示了你的**关键改进操作**，其中包含最重要的问题的最多可用点。

若要查看所有改进操作，请选择仪表板上的 "**改进操作**" 选项卡，或在仪表板上选择 "查看关键改进操作的列表" 下的 "**所有改进操作**"。 这将转到 "**改进操作**" 页面，您可以在其中查看组织的所有改进操作。

如果您的操作列表很长，则筛选视图可能很有帮助。 若要执行此操作，请选择 "操作" 列表右上角的 "**筛选器**"。 当 "**筛选器**" 浮出控件窗格出现时，根据法规和标准、解决方案和组选择所需的条件。 您还可以通过选择右上角的 "**组**" 来自定义视图，然后从下拉菜单中选择按组、解决方案、类别、操作类型或状态进行查看。

此页面的默认视图不显示测试状态为 "已**通过**" 的提高操作。 若要查看已通过测试的操作，请选中 "**筛选器**浮出控件" 窗格中的 "**传递**" 框。 仅具有已**通过**的测试状态与得分接近的操作。

"改进操作" 页显示每个改进操作的以下数据点：

- **得分影响**：完成操作时，整体得分将增加的点数
- **法规**：与操作相关的法规或标准
- **组**：为其分配操作的组
- **解决方案**：可在其中执行操作的解决方案
- **评估**：评估（它会对控制以满足特定合规性目标进行组织），操作驻留在其中
- **类别**：相关的数据保护类别（即，保护信息、管理设备等）
- **测试状态**：
    - **无**-未记录状态更新
    - **未评估**-测试尚未启动
    - **不在范围**内-已从合规性分数计算中排除，并且不会增加你的成绩
    - **部分测试**-测试尚未完成
    - **失败的高风险**-实施测试失败，并且适用的标准不符合标准的风险较高
    - 已成功测试已**通过**的实施
- 积分：显示可获得的最大**积分。**

### <a name="improvement-actions-details"></a>改进操作详细信息

每个改进操作都有详细信息页面。 此页面包含详细的实现说明，说明了如何采取建议的操作来解决 "**概览**" 标题下列出的相关标准和法规要求。

[！注释] [！注释] [！注释] [！注释] [！注释] 可以启动推荐操作或将工作分配给其他用户、更新状态以及附加注释和文档。

若要查看改进操作的详细信息页面，请执行以下操作：

1. 转到 "改进操作" 页面。
2. 单击或点击预期的 "改进" 操作的行中的任意位置，这将打开其详细信息页面。

通过选择屏幕右上角的向上或向下箭头，可以轻松查看列表中的下一个或上一个 "改进" 操作。 如果您在 "改进操作" 页面上筛选了列表，则向上或向下移动将转到该筛选列表中的下一项。

## <a name="assign-improvement-actions"></a>分配提高操作

若要开始实施改进操作，您可以自己执行工作，也可以将其分配给其他用户。 分配的人员可以是：

- 业务策略所有者
- IT 实现者
- 另一个负责执行任务的员工 

一旦确定了正确的人员，请确保他们在合规性分数（合规性管理员、合规性数据管理员、安全管理员或全局管理员）中保留了足够的[角色](compliance-score-setup.md#set-user-permissions-and-assign-roles)以执行此工作，然后执行以下步骤： 

1. 从 "改进操作详细信息" 页中，选择屏幕左上角旁边的 "**编辑状态**"。 

2. 在 "编辑状态" 浮出控件窗格中，在 "**分配对象**" 框中单击或点击，这将填充 "**建议的人员**" 用户列表。 您可以从列表中选择用户，也可以键入要向其分配该操作的人员的电子邮件地址。

3. 选择 "**保存并关闭**" 以完成工作分配。 分配的用户将收到已向其分配改进操作的电子邮件，然后他们可以从其仪表板中打开 "改进" 操作。

然后，分配的用户可以执行实施说明中所述的建议操作。

## <a name="perform-work-and-store-documentation"></a>执行工作和存储文档

执行实施工作时，您可以将文件和注释直接上载到 "**备注和文档**" 部分中的 "改进" 操作。 这提供了一个安全、集中的存储库，可帮助您演示控制的满意度，以满足合规性标准和法规要求。 任何具有只读访问权限的用户都可以读取此部分中的内容。 上载、下载或删除字段，或输入或编辑备注的功能仅限于具有编辑权限的角色。

"**备注和文档**" 部分中的字段包括：

**上载的文档**

- 选择 "**管理文档**" 以上传任何相关文件。
- 当 "管理文档" 浮出控件窗格打开时，选择 "**添加文档**"，然后从系统中选择文件。 接受的文件类型： 
  - 文档（.doc、.xls、.ppt、.txt、.pdf）
  - 图像（.jpg、.png）
  - 视频（mkv）
  - 压缩文件（.zip、rar）
- 在您的文件在窗格中进行解析后，选择 "**关闭"，** 这将自动保存文件附件。 然后，您将看到 "已**上载文档**" 下方列出的文件。 
- 若要下载或删除文档，请从文档列表下选择 "**管理文档**"。 在浮出控件窗格中，单击或点击文档行以突出显示它，然后选择 "**下载**" 或 "**删除"。**

**实现、测试和其他说明**

- 若要添加这三个字段中任一字段的注释，请选择 "**编辑实现说明**" 下面任何 "thse" 字段。
- 当浮出控件窗格打开时，在文本字段中输入备注，然后选择 "**保存并关闭**"。
- 若要编辑便笺，请选择 "**编辑实现说明**"，进行编辑，然后选择 "**保存并关闭**"。

"备注" 字段中没有字符限制。 建议保留备注摘要，以便可以从 "改进操作详细信息" 页轻松查看和编辑它们。

## <a name="change-improvement-action-status"></a>更改改进操作状态

您可以为每个改进操作记录实现状态和日期以及测试状态和日期。 以下是可用的字段和状态选项：

- **实现状态**：从以下状态选项中选择：
    - **未实现**-操作尚未实现
    - 已**实现**-操作已实现
    - **替代实现**-如果你使用其他第三方工具或执行 Microsoft 建议中未包含的其他操作，请选择此选项
    - **计划**的操作计划实施
    - **不在范围内**-与您的组织不相关的操作的选项;选择此状态会将操作从计分中排除;取消选中它将在记分中包含操作
- **实现日期**：实现状态设置为 "已**实现**" 或 "**替代实现**" 时的可用字段;在 "日历" 弹出窗口中切换以选择日期
- **测试状态**：从以下选项中选择：
    - **未评估**-测试尚未启动
    - 已成功测试已**通过**的实施
    - **失败的低风险**-测试失败，低风险
    - **失败中等风险**-测试失败，中等风险
    - **失败的高风险**-测试失败，高风险
- **测试日期**：在 "日历" 弹出窗口中切换以选择日期

> [!NOTE]
> 任何具有编辑权限的用户都可以编辑 "实施" 和 "测试状态" 字段，而不只是**分配**给用户的。

## <a name="assign-improvement-action-to-assessor-for-completion"></a>将 "改进" 操作分配给评估员以完成

完成工作并上传证据后，下一步是设置实现状态和日期，并将 "改进" 操作分配给评估员进行验证。

评估员的类型包括：

- 在组织中对控件进行验证的内部评估员
- 外部评估员检查、验证和认证合规性（如审核 Microsoft 云服务的第三方独立组织）

评估员验证工作并检查文档，并选择相应的测试状态。

**如果测试状态为 "已传递"**：该操作已完成，并且**积分**显示了已达到的可能积分，并在总体合规性分数中计入数量。

**如果测试状态为 "失败" 的任何级别**：该操作不符合要求，并且评估员可以将其分配给适当的用户进行其他工作。

## <a name="solutions-page"></a>解决方案页

"**解决方案" 页面**是继续执行改进操作以增加成绩的另一个起点。 

若要转到 "解决方案" 页，请选择仪表板上的 "**解决方案**" 选项卡，或选择 "查看下面的**所有解决方案**，**这些解决方案将影响您**的仪表板的右上部分中的分数。

"解决方案" 页显示按解决方案组织的挣和潜在点的份额。 查看此视图中的剩余点和改进操作可帮助您了解哪些解决方案需要更直接关注。

### <a name="filtering-your-solutions-view"></a>筛选解决方案视图

筛选你查看的解决方案： 

1. 选择评估列表左上角的 "**筛选器**"。
2. 在飞出的 "**筛选器**" 窗格中，在所需条件（标准和规章、解决方案、操作类型、合规性管理器组、类别）旁边进行检查。
3. 选择 "**应用**" 按钮。 筛选窗格将关闭，您将看到筛选出的视图。

您还可以通过从评估列表上方的**组**下拉菜单中选择分组类型来修改视图，以查看按组、产品或法规进行的评估。

### <a name="taking-actions-from-the-solutions-page"></a>从 "解决方案" 页采取操作

"解决方案" 页显示已连接到 "改进操作" 的组织解决方案。 该表列出了每个解决方案对总体成绩的贡献、在该解决方案中实现的分数提升点，以及在该解决方案中分组的其余改进操作的数量，该解决方案可增加成绩。 

可以通过以下两种方法从该屏幕执行操作：

1. 在预期解决方案的行上，在 "**剩余操作**" 列下，单击或点击超链接的数字。 这将转到 "改进操作" 屏幕的筛选视图，显示该解决方案的未经测试的改进操作。

2. 在预期解决方案的行中的 "**打开解决方案**" 列下，选择 "**打开**"。 这将转到 Microsoft 365 和 Office 365 安全性和合规性中心中的解决方案或位置，您可以采取建议的措施。

## <a name="assessments-page"></a>评估页

"评估" 页面列出了您选择的针对您的组织进行跟踪的评估。 您的合规性分数分母由所有跟踪的评估决定。 您添加的评定越多，您在改进操作页面上看到的改进操作越多，分数分母越高。

若要转到 "评估" 页面，请选择仪表板上的 "**评估**" 选项卡。

在此页面上，您可以快速查看有关每个评估的重要信息：

- **状态**：评估中所有改进操作的完成状态将作为以下之一列出：
    - **不符合**：评估中的改进操作尚未实施和测试成功;工作尚未开始
    - **正在进行**：工作在实施或测试改进操作过程中正在进行。例如，这可能意味着已为工作分配评估中的改进操作，正在实施和测试的过程中
- **评估进度**：评估最终完成工作所占的百分比，由成功测试的控件数衡量。
- **客户管理的操作**：为满足客户托管的控制措施而完成的操作的数量
- **Microsoft 管理的操作**：满足 microsoft 托管控件的实现的已完成操作的数量
- **评估组**：你创建的组的名称，评估驻留在此
- **相关服务**：关联的 Microsoft 365 服务
- **相关管理法规**：评估要满足的法规标准、策略或法律

### <a name="default-assessments"></a>默认评估

默认情况下，您将在 "评估" 页上看到 "Microsoft 365 数据保护基准评估"。 如果要添加更多评估以涵盖其他法规和标准，可以在合规性管理器中管理评估。 合规性分数提供了10个开箱即用的评估，其中包括 GDPR、CCPA、ISO 27001、ISO 27018、NIST 800-53、NIST 800-171、NIST CSF、CSA CCM、FFIEC、HIPAA/高科技和 FedRAMP。

### <a name="managing-assessments"></a>管理评估

在公共预览过程中，用于查看、创建、导出和存档评估的功能将保留在合规性管理器工具中。 

若要管理评估，请选择 "评估" 列表顶部的 **"管理合规性管理器" 中的 "管理评估**"。

"评估" 列表顶部的另一个链接（**合规性管理器中的 "microsoft 操作**"）将转到合规性管理器中的页面，其中显示了有助于满足合规性分数的 Microsoft 控件。

### <a name="filtering-your-assessments-view"></a>筛选评估视图

筛选评估的视图：

1. 选择评估列表左上角的 "**筛选器**"。
2. 在飞出的 "**筛选器**" 窗格中，在所需条件（标准和规章，合规性管理器组）旁边进行检查。
3. 选择 "**应用**" 按钮。 筛选窗格将关闭，您将看到筛选出的视图。

您还可以通过从评估列表上方的**组**下拉菜单中选择分组类型来修改视图，以查看按组、产品或法规进行的评估。

### <a name="managing-improvement-actions-within-an-assessment"></a>管理评估中的改进措施

从 "**客户管理的操作**" 列下的 "评估" 列表中，选择预期评估的行上的链接文本。 这将为您显示 "改进操作" 页面的筛选视图，其中显示了该评估中的操作。

## <a name="reporting"></a>Reporting

您可以在合规性分数中导出所有改进操作的报告。 从 "**改进操作**" 页中，选择屏幕左上角的 "**导出**"，在操作列表之上。 这将生成一个包含所有改进操作的 Excel 工作表和 "**改进操作**" 页面上显示的筛选器类别，可以在其中查看并保存到本地计算机。

您还可以从合规性管理器导出报告。 在合规性管理器中，转到 "**控件信息**" 选项卡，然后选择屏幕右上部分的 "**导出**"。 这将生成可查看和保存的 Excel 工作表。
