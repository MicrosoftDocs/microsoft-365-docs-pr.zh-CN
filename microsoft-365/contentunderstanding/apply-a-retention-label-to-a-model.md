# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>将保留标签应用于文档理解模型 (预览) 

> [!Note] 
> 本文中的内容适用于 Project Cortex 私人预览。 了解[有关 Project Cortex 的详细信息](https://aka.ms/projectcortex)。

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

您可以轻松地将 [保留标签](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) 应用于 Project Cortex 中的文档理解模型。

保留标签允许您将保留设置应用于文档了解模型标识的文档。  例如，您可能希望您的模型不仅能够识别上载到文档库的任何合同续订文档，还可以对其应用保留标记，以便永远不会删除这些文档。

您可以通过模型设置将保留标签应用于文档理解模型。 

> [!Note]
> 若要使保留标签可用于您的模型，需要 [在 Microsoft 365 合规性中心中创建和发布](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels)这些标签。
