# <a name="push-content-types-to-a-hub"></a>将内容类型推送到中心

若要使重要的内容类型对 SharePoint 库和列表更加一致，可以将它们推送到您选择的中心。 这会自动将其添加到在与中心关联的网站上创建的任何新列表和库以及添加到中心的任何新网站。

若要使此功能正常运行，必须已发布要推送的内容类型。

将内容类型推送到集线器

1. 在 SharePoint 管理中心中，展开 " **内容服务**"，然后单击 " **内容类型库**"。

2. 单击要推送到集线器的内容类型。

3. 在命令栏中单击 " **编辑** "。
 
4. 单击 " **选择中心站点**"。
 
5. 选择所需的中心网站，然后单击 **"确定"**。
 
6. 单击“**保存**”。

首次将内容类型推送到现有中心 & 它的现有关联网站时，在访问中心或关联的网站时，可能需要一小时的时间，以供在网站中更新的设置。 任何新的中心关联都不需要此等待，并将在几分钟后反映这些设置。 

配置此配置后，具有这些设置的内容类型将在几分钟内与中心在任何新关联的网站中可用。 一旦可用，创建的任何新列表或库都会在几分钟内自动将其添加到该内容类型。 仅当推送的内容类型从文档内容类型直接或间接派生时，才会添加到文档库中，并且仅当它不直接或间接从文档内容类型派生时，才会将该内容类型添加到列表中。
