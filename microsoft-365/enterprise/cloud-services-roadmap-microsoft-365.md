# <a name="cloud-services-roadmap-for-microsoft-365"></a>适用于 Microsoft 365 的云服务路线图

若要获得 Microsoft 365 for enterprise 的创造力、团队合作和工作效率优势，请部署最适合您组织需求的云服务。

如果您有现有的基于服务器的 Exchange、SharePoint 或 Skype for Business 部署，请参阅 [客户端和服务器软件路线图](client-server-software-roadmap-microsoft-365.md) ，将整个组织迁移到适用于企业的 Microsoft 365。 本指南包括 Microsoft Office 客户端产品、本地 Office Server 产品和基于 Microsoft Windows 的设备。

## <a name="deploy"></a>部署

若要部署云服务，请执行以下操作：

- [获取服务就绪](configure-services-and-applications.md)
- [将本地数据迁移到 Microsoft 365](migrate-data-to-office-365.md)
- 获取为用户设置的云服务
  - [Exchange Online](https://docs.microsoft.com/Exchange/exchange-online)
  - [SharePoint Online](https://docs.microsoft.com/sharepoint/sharepoint-online)
  - [Skype for Business](https://docs.microsoft.com/SkypeForBusiness/skype-for-business-online)
  - [Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/Teams-overview)
  - [Yammer](https://support.office.com/article/e1464355-1f97-49ac-b2aa-dd320b179dbe)
- [培训用户](https://docs.microsoft.com/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

## <a name="manage"></a>管理

若要管理云服务，请执行以下操作： 

- [检查服务运行状况](view-service-health.md)
- [了解你的支持选项](https://support.office.com/article/Contact-support-for-business-products-Admin-Help-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)
- 管理云服务
  - [Exchange Online](https://docs.microsoft.com/Exchange/exchange-online)
  - [SharePoint Online](https://support.office.com/article/79eb0420-8cbd-4bcb-a90b-ddc7d3ab4b3a)
  - [Skype for Business](https://docs.microsoft.com/SkypeForBusiness/skype-for-business-online)
  - [Microsoft Teams](https://docs.microsoft.com//MicrosoftTeams/quality-of-experience-review-guide)
  - [Yammer](https://support.office.com/article/e1464355-1f97-49ac-b2aa-dd320b179dbe)
- [为云服务执行性能调整](tune-microsoft-365-performance.md)

## <a name="how-microsoft-does-cloud-services-for-microsoft-365"></a>Microsoft 如何为 Microsoft 365 提供云服务

若要了解 Microsoft 的 IT 部门如何部署或正在管理 Microsoft 365 云服务，请执行以下操作：

1. 转到 [MICROSOFT IT 展示](https://www.microsoft.com/itshowcase)。
2. 选择 " **搜索内容**"。
3. 在**精简结果**中，选择 "**访问群体**" 下的 " **IT 专业人员**"，并在 "**产品**" 下选择你感兴趣的云服务
