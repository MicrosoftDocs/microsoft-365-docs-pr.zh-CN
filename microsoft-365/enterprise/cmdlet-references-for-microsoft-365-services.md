# <a name="cmdlet-references-for-microsoft-365-services"></a>Microsoft 365 服务的 cmdlet 参考 

*本文适用于 Microsoft 365 企业版和 Office 365 企业版。*

本文提供了各种 Microsoft 365 服务的 cmdlet 参考以及 PowerShell 支持的每个 Microsoft 365 服务的连接说明。
  
> [!NOTE]
> 若要一次连接到所有服务，请参阅在单个服务窗口中连接到所有 [Microsoft 365 Windows PowerShell服务](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)。
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Azure Active Directory PowerShell cmdlet

适用于 Graph 的 Azure Active Directory PowerShell cmdlet 参考主题在适用于 Graph 文档的 [Azure Active Directory PowerShell 的参考部分中](/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)。

Azure Active Directory 模块用于Windows PowerShell cmdlet 参考主题位于 Azure Active [Directory (MSOnline](/powershell/azure/active-directory/overview?view=azureadps-1.0)文档) 部分中。

有关 Microsoft 365 PowerShell 连接说明，请参阅使用 PowerShell 连接到[Microsoft 365。](connect-to-microsoft-365-powershell.md)
  
## <a name="exchange-online-powershell-cmdlets"></a>Exchange Online PowerShell cmdlet

Exchange Online cmdlet 参考主题在 Exchange [Online PowerShell](/powershell/exchange/exchange-online-powershell)文档的参考部分。
  
有关 Exchange Online PowerShell 的连接说明，请参阅[连接到 Exchange Online PowerShell。](/powershell/exchange/connect-to-exchange-online-powershell)
  
> [!NOTE]
> Exchange Online PowerShell 中提供了其他服务（如 SharePoint Online、Skype for Business Online 和 Microsoft 365 用户活动）的报告 cmdlet。 有关详细信息，请参阅 [Exchange Online 中的报告 cmdlet](/powershell/exchange/exchange-online-powershell)。 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>SharePoint Online PowerShell cmdlet

有关 SharePoint Online cmdlet，请参阅 Index [of Windows PowerShell for SharePoint Online cmdlets](/powershell/module/sharepoint-online/)。
  
有关 SharePoint Online PowerShell 的连接说明，请参阅设置 [SharePoint Online](/powershell/sharepoint/sharepoint-online/connect-sharepoint-online)命令行管理程序Windows PowerShell环境。
  
## <a name="skype-for-business-online-powershell-cmdlets"></a>Skype for Business Online PowerShell cmdlet

有关 Skype for Business Online cmdlet 参考主题，请参阅[Skype for Business Online cmdlet。](/previous-versions//mt228132(v=technet.10))
  
有关 Skype for Business Online PowerShell 的连接说明，请参阅使用 PowerShell 管理[Skype for Business Online。](manage-skype-for-business-online-with-microsoft-365-powershell.md)

有关安全与合规& PowerShell 的连接说明，请参阅连接到安全与&[中心 PowerShell。](/powershell/exchange/connect-to-scc-powershell)
