# <a name="step-4-simplify-password-resets"></a>第 4 步：简化密码重置

** 此步骤是可选的，适用于 Microsoft 365 企业版的 E3 和 E5 版本

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

在此步骤中，将启用自助密码重置 (SSPR)，以允许用户重置或解除锁定其密码或帐户。为提醒你避免误用或滥用，可以使用详细报告，跟踪用户访问系统的时间，并进行通知。

请参阅[启用密码重置说明](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-best-practices)。
