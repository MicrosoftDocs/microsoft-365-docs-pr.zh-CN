<span data-ttu-id="6230b-109">使用身份验证库（如 Active Directory 身份验证库 (ADAL) 或 Microsoft 身份验证库 (MSAL) ）启用新式身份验证。</span><span class="sxs-lookup"><span data-stu-id="6230b-109">Modern authentication is enabled through the use of an authentication library, like Active Directory Authentication Library (ADAL) or Microsoft Authentication Library (MSAL).</span></span> <span data-ttu-id="6230b-110">新式身份验证是客户端用于对 Microsoft 365 资源进行身份验证和授权访问的客户端。</span><span class="sxs-lookup"><span data-stu-id="6230b-110">Modern authentication is what clients use to authenticate and authorize access to Microsoft 365 resources.</span></span> <span data-ttu-id="6230b-111">新式身份验证利用 OAuth，为客户端提供了访问 Microsoft 365 服务的安全机制，而无需访问用户凭据。</span><span class="sxs-lookup"><span data-stu-id="6230b-111">Modern authentication leverages OAuth and provides a secure mechanism for clients to access Microsoft 365 services, without requiring access to user credentials.</span></span> <span data-ttu-id="6230b-112">登录时，用户直接使用 Azure Active Directory 进行身份验证，并接收访问/刷新令牌对作为返回。</span><span class="sxs-lookup"><span data-stu-id="6230b-112">At sign-in, the user authenticates directly with Azure Active Directory and receives an access/refresh token pair in return.</span></span> <span data-ttu-id="6230b-113">访问令牌授予客户端对 Microsoft 365 租户中相应资源的访问权限。</span><span class="sxs-lookup"><span data-stu-id="6230b-113">The access token grants the client access to the appropriate resources in the Microsoft 365 tenant.</span></span> <span data-ttu-id="6230b-114">刷新令牌用于在当前访问令牌过期时获取新的访问令牌或刷新令牌对。</span><span class="sxs-lookup"><span data-stu-id="6230b-114">A refresh token is used to obtain a new access or refresh token pair when the current access token expires.</span></span>
