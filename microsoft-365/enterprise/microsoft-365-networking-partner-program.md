# <a name="microsoft-365-networking-partner-program"></a>Microsoft 365 网络合作伙伴计划

网络连接直接影响用户使用 Microsoft 365 快速工作、高效协作和简化业务流程的能力。 对于处于数字化转型任何阶段的客户，网络设计是一个关键方面，应在问题对用户体验产生负面影响之前主动解决。

随着客户采用 Microsoft 365 提高企业工作效率，Microsoft 已观察到网络性能和生成的最终用户协作体验受到用户与 Microsoft 365 之间路径中的网络解决方案直接影响的常见趋势。 为了帮助合作伙伴设计最佳网络解决方案并帮助客户就此类解决方案做出明智决定，我们构建了 Microsoft 365 网络合作伙伴计划。

Microsoft 365 网络合作伙伴计划进一步加深了与网络合作伙伴的协作，并确定了符合 Microsoft 365 网络要求、建议和最佳做法的关键产品和解决方案。 Microsoft 365 网络合作伙伴计划的目标是通过轻松发现经过验证的合作伙伴解决方案来提升客户改进 Microsoft 365 体验的能力，这些解决方案一致地证明与客户部署中的最佳 Microsoft 365 连接性的关键原则保持一致。

为了现代化企业网络以与 Microsoft 365 建立出色的连接，客户通常依赖网络解决方案提供商、本地或基于云的安全服务以及系统集成商来规划、设计和实现云服务的网络连接。 客户经常询问 Microsoft 他们的网络体系结构和解决方案是否适用于 Microsoft 365，以及它们是否与 Microsoft 的 [Microsoft 365](https://aka.ms/PNC)网络连接原则一致。

Microsoft 365 网络合作伙伴计划可帮助回答这些问题，并证明 Microsoft 致力于帮助客户构建和优化其网络体系结构，从而获得最佳 Microsoft 365 体验。 Microsoft 365 团队正在与许多网络行业合作伙伴合作，以帮助确保最佳连接的关键原则在本机内置于其网络产品和解决方案中。

Microsoft 365 网络合作伙伴计划将合格的网络解决方案指定为"适用于 Microsoft 365"。 只有在与 Microsoft 365 一起使用时，满足 Microsoft 严格测试要求、具有无缝安装体验并证明高质量网络体验的设备或解决方案才能获得"使用 Microsoft 365"标志。 Microsoft [365](https://www.microsoft.com/microsoft-365/partners/O365networkingpartners)网络合作伙伴计划页面上列出了这些经过验证的合作伙伴解决方案，以及解决方案详细信息和了解详细信息的链接。

如果你是 Microsoft 365 客户，可以通过此页面使网络解决方案提供商知道此计划 [https://aka.ms/Office365NPP](https://aka.ms/Office365NPP) 。

对于想要查询 Microsoft 365 网络合作伙伴计划的网络解决方案提供商， [https://aka.ms/NPPsignup](https://aka.ms/NPPsignup) 请填写表单。 感谢你的关注，我们期待有一个美观的合作关系。
