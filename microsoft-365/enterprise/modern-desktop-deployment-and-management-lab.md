# <a name="modern-desktop-deployment-and-management-lab-kit"></a>新式桌面部署和管理实验室工具包

这些可下载的实操实验室着重于 Windows 10 部署和 Office 365 部署，以及相关的配置和管理注意事项后期部署。强烈建议为准备 Windows 7 生命周期的组织进行此培训，但该培训同样适用于当前使用 Windows 10 和 Office 365 增强版或 Office 2019 的用户。其中包含有关 Windows 10、Office 365 专业增强版、企业移动性 + 安全性和相关产品和服务的指南。

这些实验室旨在帮助你计划、测试和验证运行 Windows 10 企业版和 Office 365 专业增强版的新式桌面的部署和管理。此实验室涵盖新式桌面部署轮形图中所列的步骤和工具，包括 System Center Configuration Manager、Windows Analytics、Office 自定义工具、OneDrive、Windows Autopilot 等。

作为 Windows 10 和 Office 365 专业增强版的[新式桌面部署](http://www.aka.ms/howtoshift)过程的一部分，在开始探索部署工具更新和测试与部署相关的自动化时，建议首先生成沙盒或隔离的实验室环境。

实验室工具包可免费下载，并使用试用版软件。

[**下载**](https://aka.ms/mddlabs_evalcenter)

**完整的实验室环境**

实验室为你提供了自动预配的虚拟实验室环境，其中包括域加入桌面客户端、域控制器、Internet 网关和完全配置的 ConfigMgr 实例。实验室包含以下产品的评估版本：

  - Windows 10 企业版，版本 1803
  - Windows 7
  - Office 365 专业增强版，版本 1807
  - System Center Configuration Manager，版本 1802
  - 适用于 Windows 10 的 Windows 评估和部署工具包，版本 1803
  - Microsoft Deployment Toolkit
  - Microsoft Application Virtualization (App-V) 5.1
  - Microsoft BitLocker 管理和监视 2.5 SP1
  - Windows Server 2016
  - Microsoft SQL Server 2014

此外，该实验室旨在与以下试用版连接：

  - Microsoft 365 企业版 E5

或
  - Office 365 企业版 E5
  - 企业移动性 + 安全性

**分步实验室**

详细的实验室指南向你介绍多个部署和管理方案，其中包括：

**设备和应用就绪情况**

  - Windows Analytics
  - Internet Explorer 的企业模式和企业模式网站列表

**目录和网络就绪情况**

  - Azure Active Directory 和 Microsoft 365 的基本设置
  - 使用传递优化、ConfigMgr 中的对等缓存和 LEDBAT 来进行网络优化
  - **新建** ConfigMgr 和 Microsoft Intune 共同管理
  - **新建**远程访问 (VPN)

**Office 和 LOB 应用交付**

  - 使用 System Center Configuration Manager 进行 Office 365 专业增强版部署
  - 使用 Microsoft Intune 进行 Office 365 专业增强版部署
  - 使用 Microsoft Intune 进行应用部署和管理
  - 使用适用于企业的 Microsoft Store 进行应用部署和自助式安装
  - 桌面桥应用程序转换为 UWP
  - **新建** Windows 应用认证工具包
  - 使用适用于 IE 的企业模式进行浏览器兼容性修正

**用户文件和设置迁移**

  - 用户状态迁移工具，作为 ConfigMgr 和 MDT 中电脑刷新和替换任务序列的一部分
  - OneDrive 已知文件夹移动
  - 企业状态漫游

**安全性和符合性**

  - BitLocker 设备加密
  - Windows Defender 防病毒
  - Windows Hello 企业版
  - BIOS 转换为 UEFI 作为基于虚拟化的安全性的启用程序
  - Windows Defender Credential Guard
  - Windows Defender 应用程序防护
  - Windows Defender 攻击防护
  - Windows Defender 应用程序控制
  - Windows Defender 高级威胁防护

**OS 部署和功能更新**

  - OS 映像创建
  - ConfigMgr 中的 OS 部署任务序列
      - 裸机
      - 刷新
      - 替换
      - 升级
  - MDT 中的 OS 部署任务序列
  - 使用 ConfigMgr 中的升级任务序列进行功能更新
  - Windows Autopilot

**Office 和 Windows 作为服务**

  - 使用 Configuration Manager 进行软件更新管理
  - Configuration Manager 中的 Office 365 专业增强版更新管理
  - 使用 Microsoft Intune 向 Windows 10 应用移动设备管理
