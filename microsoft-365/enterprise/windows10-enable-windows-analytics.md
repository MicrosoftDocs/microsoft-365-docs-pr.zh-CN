# <a name="step-4-monitor-device-health-and-compliance"></a>第 4 步：监视设备运行状况和合规性

![](./media/deploy-foundation-infrastructure/win10enterprise_icon-small.png)

Windows 分析使用诊断数据提供丰富的、 可操作的信息可帮助您获得深度见解的运营效率和您的环境中的 Windows 10 设备运行状况。

Windows 分析的设备运行状况功能提供主动建议，以帮助检测并修正问题影响最终用户。此新的服务使用诊断数据提供无需其他基础结构要求此类见解。主动补救最终用户问题，可以降低支持成本和提高效率。查找要很快发布的其他功能，其中将增强功能和此新的服务的值。

请参阅[监视设备的设备运行状况的运行状况](https://docs.microsoft.com/windows/deployment/update/device-health-monitor)了解详细信息，请开始，并使用设备运行状况。

作为临时检查点，请查看对应于此步骤的[退出条件](windows10-exit-criteria.md#crit-windows10-step4)。
