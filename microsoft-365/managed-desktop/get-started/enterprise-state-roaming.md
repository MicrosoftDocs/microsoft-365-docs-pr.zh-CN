# <a name="enable-enterprise-state-roaming"></a>启用企业状态漫游

为了获得最佳 Microsoft 托管桌面体验，请[](/azure/active-directory/devices/enterprise-state-roaming-overview)启用企业状态漫游，这将允许用户将用户和应用程序设置数据安全地同步到云。 这意味着无论登录哪个 Windows 设备，他们都具有相同的体验。 例如，如果将其中一台 Microsoft 托管桌面设备替换为新设备，则其外观和行为与上一设备完全相同。

若要启用企业状态漫游，请按照在 Azure [Active Directory](/azure/active-directory/devices/enterprise-state-roaming-enable)中启用企业状态漫游中的步骤操作，然后返回到本文档。

如果你对企业状态漫游有任何困难，请联系管理员 [支持](../working-with-managed-desktop/admin-support.md)人员。
