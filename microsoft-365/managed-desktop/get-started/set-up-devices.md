# <a name="set-up-microsoft-managed-desktop-devices"></a>设置 Microsoft 托管桌面设备

可以在 Microsoft 托管桌面中同时使用新的和现有的设备。

## <a name="to-obtain-new-devices"></a>获取新设备

我们建议与已批准的设备合作伙伴之一合作。 你可以与 Microsoft 帐户联系人合作，以了解有关建立设备合作关系的更多帮助。 但本质上，此过程如下所示：

1. 查看当前 [批准的设备的列表](../service-description/device-list.md)。
2. 订购设备，与批准的设备合作伙伴合作。
3. 到达后，执行下列任一操作：
    - [自行注册新设备](register-devices-self.md)
    - 与合作伙伴合作注册设备
4. [让用户准备好使用](get-started-devices.md) Microsoft 托管桌面设备

## <a name="to-reuse-existing-devices"></a>重用现有设备

1. 查看当前 [批准的设备列表](../service-description/device-list.md) ，以确定哪些设备已批准用于 Microsoft 托管桌面。 如果重复使用现有设备，必须重新创建映像。
2. 执行下列任一操作：
    - [自行注册现有设备](register-reused-devices-self.md)
    - 与合作伙伴合作注册设备
3. [让用户准备好使用](get-started-devices.md) Microsoft 托管桌面设备
