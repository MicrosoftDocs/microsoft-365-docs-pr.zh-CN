# <a name="microsoft-managed-desktop-devices"></a>Microsoft 托管桌面设备

Microsoft 托管桌面定期计算服务中包含的设备。本主题概述了一个设备，以在 Microsoft 托管桌面中注册的要求。

在 Microsoft 托管桌面上的详细信息，请参阅[Microsoft 托管桌面文档和资源](https://docs.microsoft.com/microsoft-365/managed-desktop/)。 

<!-- Microsoft 365 E5; Device as a Service -->
<!-- Split from device & technologies topic. Destination topic for aka.ms/device-list  -->若要注册 Microsoft 托管桌面：

- 设备必须是以下模型和配置和 #x00B9; 之一：
    - Surface Laptop = #x00B2;: 256 GB / Intel i5 / 8 GB RAM 
    - Surface Pro = #x00B2;: 256 GB / Intel i5 / 8 GB RAM
    - 2 = #x00B2; Surface Book: 512 GB / Intel i7 / 16 GB RAM / 15 英寸显示
    - 转到和 #x00B2; 公开： 128 GB / Intel 4415Y / 8 GB RAM
- 设备必须具有图像的 & #x00B3;:
    - 具有最新版本的 Windows 10 Enterprise 或专业人员;
    - 具有最新版本的 Microsoft Office 365 Click to Run;
    - 具有最新的 OEM 提供驱动程序安装;和
    - 未预加载任何其他软件
- 设备必须未完成 Windows 首次运行体验。
- 设备必须与 Windows 自动执行某些操作注册并已分配的 Microsoft 托管桌面自动执行某些操作配置文件。

**脚注**

& #x00B9;可在任何时候对此列表添加。删除将 90 天之前不允许从注册这些设备发布对此页。

& #x00B2;功能和特性的这些设备的详细信息，请参阅[比较外围设备](https://www.microsoft.com/surface/devices/compare-devices)。

& #x00B3;通过新的设备获取或重新映像以前获取的设备可以实现这一点。
