此外, 设备还必须:     

- 具有来自工厂的图像, 或通过重新映像执行以下操作:
    - 具有最新的 Windows 10 专业版或企业版或早期版本。
    - 具有最新版本的 Microsoft Office 365 单击以运行或立即发布以前的版本;
    - 安装了最新的 OEM 提供的驱动程序;并
    - 没有其他预加载的软件
    - 带有5VS04AV 的 HP 设备必须具有 Microsoft 托管设备映像
- 尚未完成 Windows 首次运行体验。
- 使用[Windows Autopilot](https://docs.microsoft.com/en-us/windows/deployment/windows-autopilot/user-driven-aad)注册, 并分配 Microsoft 托管桌面 Autopilot 配置文件。
