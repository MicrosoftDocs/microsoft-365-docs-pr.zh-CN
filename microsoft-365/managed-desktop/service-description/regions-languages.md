# <a name="microsoft-managed-desktop-supported-regions"></a>Microsoft 托管桌面支持的区域

本文详细介绍了可以使用 Microsoft 托管桌面的区域及其支持位置。 有关 Microsoft 托管桌面支持的用于用户体验的语言详细信息，请参阅 [为用户本地化设备](../get-started/localization.md)。

您仍可以在这些区域之外使用托管设备，而不会中断 Microsoft 托管桌面服务。 例如，位于英国的员工在出差到亚洲、欧洲或南非时，可以安全地工作并接收其托管设备的更新。

有关用户和管理员体验的 Microsoft 托管桌面支持的语言详细信息，请参阅 [本地化用户体验](../get-started/localization.md)。

有关 Microsoft 托管桌面的用户支持详细信息，请参阅 [Microsoft 托管桌面支持](support.md)。

## <a name="availability-of-the-service"></a>服务的可用性

以下国家/地区中的组织可以订阅 Microsoft 托管桌面：

- 美国
- 加拿大 (，不包括) 
- 英国
- 爱尔兰
- 比利时
- 卢森堡
- 荷兰
- 瑞典
- 芬兰
- 挪威
- 丹麦
- 冰岛
- 澳大利亚
- 新西兰
