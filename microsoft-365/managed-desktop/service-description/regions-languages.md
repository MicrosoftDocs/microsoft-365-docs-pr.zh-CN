# <a name="microsoft-managed-desktop-supported-regions-and-languages"></a>Microsoft 托管桌面支持的区域和语言

本文详细介绍了 Microsoft 托管桌面支持的地区和语言以及可以使用的区域。

## <a name="supported-languages"></a>支持的语言

英语是 Microsoft 托管桌面用户能够使用的唯一语言。 此策略包括用户和管理员的所有用户界面，以及管理员支持和用户支持[的所有](../working-with-managed-desktop/admin-support.md)[交互](../working-with-managed-desktop/end-user-support.md)。


您仍可以在英语区域之外使用托管设备，而不会中断 Microsoft 托管桌面服务。 例如，位于英国的员工在出差到亚洲、欧洲或南非时，可以安全地工作并接收其托管设备的更新。 

有关 Microsoft 托管桌面的用户支持详细信息，请参阅 [Microsoft 托管桌面支持](./support.md)。

## <a name="availability-of-the-service"></a>服务的可用性

以下国家/地区中的组织可以订阅 Microsoft 托管桌面：

- 美国
- 加拿大 (，不包括) 
- 英国
- 爱尔兰
- 比利时
- 卢森堡
- 荷兰
- 瑞典
- 芬兰
- 挪威
- 丹麦
- 冰岛
- 澳大利亚
- 新西兰

这些区域的用户和管理员都必须使用英语作为他们的设备和管理工具。
