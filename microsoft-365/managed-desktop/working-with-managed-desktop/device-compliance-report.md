# <a name="device-compliance-report"></a>设备合规性报告

此视图显示符合 Microsoft Intune 中设置的合规性策略的 Microsoft 托管桌面设备的百分比。

![显示符合策略的设备百分比的报告](../../media/mmd-device-compliance-percent.png)


选择 "**查看详细信息**" 以查看所有 Microsoft 托管桌面设备或不符合合规性策略的设备的特定信息。 使用下拉菜单在这些视图之间切换。 若要在此视图中显示，设备必须是物理计算机，并且已在最近30天内与 Intune 同步。

![包含所有设备或不兼容设备的选项卡的详细信息窗格。 显示详细的架构，包括序列号、设备名称、合规性状态和其他信息。 "下载完整数据" 按钮底部。](../../media/mmd-device-compliance-detail.png)

在此视图中，选择 "**下载完整数据**" 以导出具有相同信息的逗号分隔文件。 在导出文件中，作为**架构**显示的区域将显示为列。
