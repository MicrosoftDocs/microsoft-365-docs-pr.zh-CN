# <a name="work-with-reports"></a>使用报表

Microsoft 托管桌面提供了 IT 管理员在你的租户中使用的报告，可用于了解设备的各个方面。 您可以在 "**报告**" 下访问[管理门户](https://aka.ms/mmdportal)中的报告。


## <a name="device-compliance-report"></a>设备合规性报告

主视图显示符合 Microsoft Intune 中设置的合规性策略的 Microsoft 托管桌面设备的百分比。 您还可以查看所有 Microsoft 托管桌面设备的特定信息，或仅查看不符合合规性策略的设备，以及导出具有相同信息的文件。

了解有关[设备符合性报告](device-compliance-report.md)的详细信息。

## <a name="device-inventory-report"></a>设备清单报告

"**设备**视图" 提供了 "**导出全部**" 选项卡，用于[下载详细的清单报告](device-inventory-report.md)。
