# <a name="work-with-reports"></a>处理报告

Microsoft 托管桌面提供了 IT 管理员在你的租户中使用的报告，可用于了解设备的各个方面。 您可以在 "**报告**" 下访问[管理门户](https://aka.ms/mmdportal)中的报告。



## <a name="device-inventory-report"></a>设备清点报告

" **设备** 视图" 提供了 " **导出全部** " 选项卡，用于 [下载详细的清单报告](device-inventory-report.md)。
