# <a name="custom-detections-overview"></a>自定义检测概述

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**适用于：**
- Microsoft 365 Defender

通过自定义检测，您可以主动监视和响应各种事件和系统状态，包括可疑的泄露活动和错误配置的终结点。 这通过可自动触发警报和响应操作可自定义的检测规则实现。

自定义检测 [与高级搜寻](advanced-hunting-overview.md)一起运行，提供一种功能强大、灵活的查询语言，涵盖来自网络的一组广泛的事件和系统信息。 你可以将它们设置为定期运行，生成警报，并随时执行响应操作。

自定义检测提供：
- 针对从高级搜寻查询构建的基于规则的检测的警报
- 自动响应操作
