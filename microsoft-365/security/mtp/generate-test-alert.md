# <a name="generate-a-test-alert-in-your-microsoft-threat-protection-evaluation-lab"></a>在 Microsoft 威胁 Protection 评估实验室中生成测试警报  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**适用于：**
- Microsoft 威胁防护

至此，你已完成 Microsoft 威胁防护评估实验室的安装和配置，你可以运行威胁攻击模拟，以体验集成安全解决方案的检测、警报关联和自愈修复功能。  

1. 登录到 https://security.microsoft.com
2. 在主页上，查找标题为 " **模拟**" 的磁贴。  有关在 Microsoft 威胁防护中创建测试事件的步骤，请参阅演练指南。
