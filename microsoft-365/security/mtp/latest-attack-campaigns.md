# <a name="understand-the-latest-attack-campaigns-and-techniques-with-threat-analytics"></a>使用威胁分析了解最新的攻击活动和技术 

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


网络威胁变得越来越频繁和复杂。 对于组织来说，快速评估其安全状况和抵御特定新兴威胁的复原能力至关重要。

**威胁分析** 是在识别新出现的威胁（新技术、攻击活动和恶意软件爆发）后即发布的一组交互式报告。 这些报告可帮助你评估威胁对环境的影响，并提供包含、提高组织恢复能力并防止特定威胁的建议操作。

[详细了解 Microsoft Defender for Endpoint 中的威胁分析](/windows/security/threat-protection/microsoft-defender-atp/threat-analytics)。
