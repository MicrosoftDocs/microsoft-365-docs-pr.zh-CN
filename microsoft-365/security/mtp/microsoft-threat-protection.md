# <a name="microsoft-threat-protection"></a>Microsoft 威胁防护

Microsoft 威胁防护是一个全面的安全解决方案，其中提供了强大的安全功能，使当今的组织能够： 

- **保护资产**-防止未经授权的用户、设备和应用程序的资产。

- **连接点**即用无与伦比的光学器件，解决方案应用信号关联以将来自各种攻击面的警报组合在一起，为您提供有关威胁范围的更好的上下文。

- 提供**defenders** -通过提供对威胁的完整范围的可见性，你的安全 defenders 可以确定其影响并采取适当的补救措施。


通过 Microsoft 365 安全中心访问 Microsoft 威胁防护的功能。
