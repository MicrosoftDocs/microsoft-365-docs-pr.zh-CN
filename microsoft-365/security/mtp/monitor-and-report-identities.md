# <a name="identity-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Microsoft 365 安全中心中的标识监视和报告

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


监视组织中的身份，并跟踪可疑或风险行为。 在 **"报告的身份****"类别中**，可以跟踪：

* 检测到的异常最多的用户
* 条件访问策略报告的有风险用户数
* 组织中的全局管理员数

!["报告标识"类别页面](../../media/identities.png)

对于具有特定检测的用户，你可以浏览特定警报，并调查 Microsoft Defender 安全中心。 检测包括异常，例如从不熟悉的位置登录的用户。

有关完整的风险事件集，请参阅 Azure [Active Directory 风险事件](https://docs.microsoft.com/azure/active-directory/reports-monitoring/concept-risk-events)。
