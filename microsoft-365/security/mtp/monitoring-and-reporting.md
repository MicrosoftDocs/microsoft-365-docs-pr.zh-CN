# <a name="monitor-and-view-reports-in-the-microsoft-365-security-center"></a>在 Microsoft 365 安全中心中监视和查看报告

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

> 想要体验 Microsoft 365 Defender？ 可以在[实验室环境中对其进行评估或在](https://aka.ms/mtp-trial-lab)[生产中运行试点项目](https://aka.ms/m365d-pilotplaybook)。
>

Microsoft 365 安全中心提供了 Microsoft 365 环境中保护和安全状态的摘要。

安全中心包括一 **个"** 报告"部分，其中包含一系列覆盖各种领域的卡片。 安全分析师和管理员可以在日常操作中跟踪卡片。 在向下钻取时，卡片提供详细的报告，在某些情况下，提供管理选项。

## <a name="customize-views"></a>自定义视图

默认情况下，卡片分为以下类别：
  
* [标识](monitor-and-report-identities.md) - 用户帐户和凭据
* [数据](monitor-data.md) - 电子邮件和文档内容
* [设备](monitor-devices.md) - 计算机、移动电话和其他设备
* [应用](monitor-apps.md) - 程序和附加的联机服务

按主题 **切换到组**，重新排列卡片，并分组为下列主题：

* **风险** - 突出显示可能面临风险的实体（如帐户和设备）的卡片。 这些卡片还突出显示了可能的风险来源，例如新的威胁市场活动和特权云应用  
* **检测趋势** - 突出显示新威胁检测、异常和策略违反的卡片
* **配置和运行状况** - 涵盖安全控件的配置和部署的卡，包括设备载入到管理服务状态
* **其他** - 所有其他未在其他主题下分类的卡片
