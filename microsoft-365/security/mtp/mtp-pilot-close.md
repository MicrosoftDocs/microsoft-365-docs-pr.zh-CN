你当前处于结束和汇总阶段。

你刚刚运行了一个仅内存攻击模拟，该模拟在域控制器上远程执行代码。 你已了解 Microsoft Defender for Endpoint 和 Microsoft Defender for Identity 如何检测和创建有关恶意恶意活动的警报。 你还了解如何将来自不同源的警报与其他上下文信息一起传递到 Microsoft 365 安全中心门户中的单个事件。 通过体验此类集成，SOC 分析员可以进行调查并进行必要的操作。 你还创建了高级搜寻查询，该查询将标识用户打开或保存附件并基于该查询创建的检测的入站电子邮件。

所有测试都结束之后，你已结束该过程。

最终输出应为：

- 已完成的记分卡
- 试点结果的详细报告
- 如何向前推进的决定

将最终输出中的报告呈现给内部利益干系人 (在准备阶段确定的报告) Microsoft[](https://docs.microsoft.com/microsoft-365/security/mtp/prepare-mtpeval)联系人。 这样做可确保任何反馈都可用于改进产品和文档。

我们希望您喜欢此模拟。 开始在组织中大规模实现你学到的知识，以充分使用集成安全解决方案。
