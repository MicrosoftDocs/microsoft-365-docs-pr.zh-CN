# <a name="set-the-time-zone-for-microsoft-365-defender"></a>设置 Microsoft 365 Defender 的时区

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**适用于：**
- Microsoft 365 Defender



Microsoft 365 Defender 可以使用本地时区或 UTC 显示日期和时间信息。 所选时区将应用于 Microsoft 365 安全中心的以下功能中所示的所有日期和时间信息：
- 事件
- 自动调查和修复，包括操作中心
- 高级搜寻

若要为这些功能设置时区，请转到 **设置**  >  **Microsoft 365 Defender**  >  **时区** 。
