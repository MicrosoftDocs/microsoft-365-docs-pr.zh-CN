# <a name="microsoft-threat-protection-prerequisites"></a>Microsoft 威胁防护先决条件

**适用于：**
- Microsoft 威胁防护

[!include[Prerelease information](prerelease.md)]

了解有关运行和使用 Microsoft 365 安全的许可、硬件和软件要求以及其他配置设置。

## <a name="licensing-requirements"></a>许可要求
Microsoft 365 安全需要以下许可证之一：

- Microsoft 365 E5 
- Office 365 E5、企业移动性 + 安全性 E5 和 Windows E5

可从 [Microsoft 365 企业版页面](https://www.microsoft.com/en-us/microsoft-365/enterprise)获取这些许可证。

### <a name="check-your-existing--licenses"></a>检查现有许可证
转到位于 [admin.microsoft.com](https://admin.microsoft.com/) 的 Microsoft 365 管理中心查看现有许可证。 在管理中心，转到“计费”**** > “许可证”****。

需要分配 [Azure AD](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) 中的**计费管理员**或**全局读取者**角色，才能查看许可信息。 如果遇到访问问题，请联系全局管理员。  

## <a name="browser-requirements"></a>浏览器要求
可通过浏览器来访问 Microsoft 365 安全中心。 支持 Internet Explorer 和 Microsoft Edge。 还支持任何符合 HTML5 的浏览器。
