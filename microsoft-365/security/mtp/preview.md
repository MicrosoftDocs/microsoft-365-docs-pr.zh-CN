# <a name="microsoft-threat-protection-preview-features"></a>Microsoft 威胁防护预览功能

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**适用于：**
- Microsoft 威胁防护


Microsoft 威胁防护服务不断更新，以包含新的功能和功能增强。

了解 Microsoft 威胁防护预览版本中的新增功能，并通过启用预览体验，率先试用即将推出的功能。

有关普遍可用的新增功能详细信息，请参阅 [Microsoft 威胁防护中的新增功能](whats-new.md)。

## <a name="turn-on-preview-features"></a>启用预览功能
你将有权访问即将推出的功能并提供反馈，从而可在功能普遍可用之前帮助改善整体体验。 

启用预览体验设置，以率先体验即将推出的功能。

1. 在导航窗格中，选择“**设置**”。

2. 选择“**Microsoft 威胁防护**”。


3. 选择“**预览功能**” > “**启用预览功能**”。 

3. 选择“**保存**”。

当看到**启用预览功能**”复选框处于选中状态时，你会知道已开启了预览功能。 

## <a name="preview-features"></a>预览功能
当前预览版中提供以下功能和增强功能：

- **[Microsoft 威胁防护API](api-overview.md)** - 保护 (LOP) 级别 Microsoft 威胁防护 API 可基于共享事件和高级搜寻表自动执行工作流。 
- **[高级搜寻中的 IdentityDirectoryEvents 表](advanced-hunting-identitydirectoryevents-table.md)** - 查找涉及运行 Active Directory (AD) 的本地域控制器的事件。 该表包含一系列身份相关的事件以及域控制器上的系统事件。
- **[在高级搜寻中进行操作](advanced-hunting-take-action.md)** - 快速包含威胁或处理在[高级搜寻](advanced-hunting-overview.md)中发现的外泄资产。
- **[门户内架构参考](advanced-hunting-schema-tables.md#get-schema-information-in-the-security-center)** - 直接在安全中心获取有关高级搜寻架构表的信息。  除了表和列的描述之外，此便捷参考提供了有关支持的事件类型（`ActionType` 值）和示例查询的信息。
