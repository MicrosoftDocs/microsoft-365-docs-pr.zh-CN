# <a name="microsoft-365-defender-preview-features"></a>Microsoft 365 Defender preview 功能

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

>[!IMPORTANT]
>不提供任何服务级别协议的预览版本，不建议在生产工作负载。 某些功能可能不受支持或可能具有受约束的功能。

**适用于：**
- Microsoft 365 Defender

Microsoft 365 Defender 服务不断更新，以提供新的功能增强功能和功能。

了解 Microsoft 365 Defender preview 版中的新功能，并在首次使用时启用预览体验，以试用即将推出的功能。

有关可通用的新功能的详细信息，请参阅 [Microsoft 365 Defender 中的新增](whats-new.md)功能。

## <a name="turn-on-preview-features"></a>启用预览功能
你将有权访问即将推出的功能，您可以在中提供反馈，以帮助改进功能，使其在功能普遍可用之前获得全面体验。

启用预览体验设置，以率先体验即将推出的功能。

1. 在导航窗格中，选择“ **设置** ”。

2. 选择 " **Microsoft 365 Defender** "。


3. 选择“ **预览功能** ” > “ **启用预览功能** ”。 

3. 选择“ **保存** ”。

当看到 **启用预览功能** ”复选框处于选中状态时，你会知道已开启了预览功能。 

## <a name="preview-features"></a>预览功能
当前预览版中提供以下功能和增强功能：

- **[Microsoft 365 Defender api](api-overview.md)** -Lop-exo 级 Microsoft 365 Defender api 将使您能够基于共享事件和高级搜寻表自动执行工作流。 
- **[在高级搜索中执行操作](advanced-hunting-take-action.md)** -快速包含威胁或在 [高级](advanced-hunting-overview.md)搜索中找到的已泄露资产的地址。
- **[在门户架构参考中](advanced-hunting-schema-tables.md#get-schema-information-in-the-security-center)** ，直接在安全中心中获取有关高级搜寻架构表的信息。 除了表和列说明之外，此参考还包括受支持的事件类型 (`ActionType` 值) 和示例查询。
