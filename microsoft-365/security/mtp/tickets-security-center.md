# <a name="create-and-track-servicenow-tickets-in-the-microsoft-365-security-center"></a>在 Microsoft 365 安全中心创建和跟踪 ServiceNow 票证

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

>[!CAUTION]
>**ServiceNow 连接器的预览期已结束**<br>
>此功能不再可用。 感谢你提供反馈，并继续支持我们确定下一步。

[Microsoft 365](overview-security-center.md)安全中心已增强，能够在 ServiceNow 中本机创建和跟踪票证。 [了解有关 ServiceNow 的更多信息](https://www.servicenow.com/)

在安全中心，安全管理员可以直接向 ServiceNow 发送 [Microsoft 安全](microsoft-secure-score.md) 分数改进操作并创建票证。 可以同时创建事件管理和变更管理票证。 在安全中心主页和 ServiceNow 中跟踪票证。
