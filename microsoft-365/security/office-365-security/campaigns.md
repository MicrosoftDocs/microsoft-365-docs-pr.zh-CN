# <a name="campaign-views-in-atp"></a>ATP 中的 Campaign Views

Campaign Views 是安全 & 合规中心中的高级威胁防护 (ATP) 中的一项功能，可以对服务中的钓鱼攻击进行识别和分类。 Campaign Views 可以帮助你：

- 高效调查和应对钓鱼攻击。

- 更好地了解攻击范围。

- 向决策者展示价值。

借助 Campaign Views，可以让你比任何人都更快获得有关攻击的更全面概览。

## <a name="what-is-a-campaign"></a>什么是活动 (campaign)？

活动是针对一个或多个组织的协同式电子邮件攻击。 拉备用凭据和公司数据的电子邮件攻击是一个大且有趣的行业。 随着阻止攻击的技术不断增加，攻击者会尽努力确保继续存在成功。

Microsoft 利用跨整个服务中数量来说过多数量的防钓鱼、反垃圾邮件和反恶意软件数据来帮助确定活动。 我们将根据几个因素来分析和分类攻击信息。 例如：

- **攻击来源：源**IP 地址和发件人电子邮件域。

- **攻击邮件属性：** 邮件的内容、样式和语调。

- **攻击收件人**：收件人域名、收件人工作职能（管理员、高管等）、公司类型（大型、小型、公共、私有等）和行业。

- **攻击有效负载：** 邮件中的恶意链接、附件或其他有效负载。

活动可能短期有效，也可以跨多个天、数周或包含活动和非活动时段的月数。 市场活动可以针对你的特定组织进行发布，或者你的组织可能是拥有多个公司的大型活动的一部分。
