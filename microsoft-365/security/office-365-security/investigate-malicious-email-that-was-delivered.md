# <a name="investigate-malicious-email-that-was-delivered-in-office-365"></a>调查在 Office 365 中传递的恶意电子邮件

[Office 365 高级威胁](office-365-atp.md) 防护使你能够调查将组织中的人员置于风险中的活动，并采取措施来保护你的组织。 例如，如果您是组织的安全团队的一部分，您可以查找并调查已传递的可疑电子邮件。 可以通过使用威胁资源[管理器管理人员 (或实时检测功能或实时) 。 ](threat-explorer.md)

> [!NOTE]
> 在此处跳转到修正 [文章](https://docs.microsoft.com/microsoft-365/security/mtp/article-submission/remediate-malicious-email-delivered-office-365?view=o365-21vianet&branch=pr-en-us-4258)。

## <a name="before-you-begin"></a>准备工作

请确保满足以下要求：

- 你的组织已[将 Office 365 高级威胁](office-365-atp.md)[防护许可证分配给用户](../../admin/manage/assign-licenses-to-users.md)。

- [审核](../../compliance/turn-audit-log-search-on-or-off.md) 记录已为组织打开。

- 您的组织已为反垃圾邮件、反恶意软件、防网络钓鱼等定义了策略。 请参阅["Office 365 中的威胁防护"。](protect-against-threats.md)

- 你是全局管理员，或者你已在安全合规中心分配了安全管理员或搜索并 &amp; 清除角色。 请参阅 [安全合规中心 &amp; 中的权限](permissions-in-the-security-and-compliance-center.md)。 对于某些操作，你还必须分配有新的预览角色。

### <a name="check-the-delivery-action-and-location"></a>检查传递操作和位置

在[威胁 (和实时) ](threat-explorer.md)中，现在具有 **"传递操作和****传递位置"列**，而不是以前的"**传递状态"** 列。 这将更加完整地了解电子邮件的地位置。 这一更改的目标是使调查更易于安全操作团队，但最终结果是可以一目了然地了解有问题电子邮件的位置。

现在，传递状态分成两列：

- **传递操作** - 此电子邮件的状态是什么？

- **传递位置** - 该电子邮件的结果是什么结果？

由于存在现有策略或检测，导致对电子邮件采取的操作。 以下是电子邮件可以采取的操作：

- **已送** 达 - 电子邮件被传递到用户的收件箱或文件夹，用户可以直接访问它。

- **垃圾邮件** - 电子邮件被发送到用户的垃圾邮件文件夹或已删除文件夹，并且用户有权访问其"垃圾邮件"或"已删除"文件夹中的电子邮件。

- **已** 阻止 - 任何被隔离、失败或已删除的电子邮件。  (user.) 完全无法访问此功能) 

- **已替换** - 恶意附件替换为声明该附件是恶意附件的 .txt 文件的任何电子邮件。

传递位置显示运行后邮件的策略和检测的结果。 它链接到传递操作。 添加此字段是为了让你了解发现问题邮件时所采取的操作。 以下是传递位置可能的值：

- **收件箱或** 文件夹 - 电子邮件位于收件箱或文件夹中， (联系人中) 。

- **本地或外部** – 邮箱不存在于云上，但在本地。

- **垃圾邮件** 文件夹 – 电子邮件位于用户的"垃圾邮件"文件夹中。

- **"已删除邮件** "文件夹 – 电子邮件位于用户的"已删除邮件"文件夹中。

- **隔离 – 在隔离** 电子邮件，而不是在用户邮箱中的电子邮件。

- **失败** – 电子邮件无法访问邮箱。

- **Dropped** – 电子邮件丢失于邮件流中的某个地点。

### <a name="view-the-timeline-of-your-email"></a>查看电子邮件的日程表

**"电子邮件日程** 表"是威胁资源管理器中的一个字段，可以帮助你更轻松地寻线安全运营团队。 当在电子邮件上同时或接近同一时间发生多个事件时，这些事件会显示在时间线视图中。 某些在送达电子邮件后的事件将捕获到"特殊 **操作"列** 中。 将电子邮件的时间线信息与在送达后采取的任何特殊操作相组合，管理员将对策略和威胁处理 (（例如邮件的路由位置以及某些情况下的最终评估是) ）提供。
