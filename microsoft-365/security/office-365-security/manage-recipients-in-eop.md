# <a name="manage-recipients-in-standalone-eop"></a>在独立 EOP 中管理收件人

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**适用对象**
-  [Exchange Online Protection独立](exchange-online-protection-overview.md)

独立Exchange Online Protection (EOP) 组织Exchange Online邮箱支持以下类型的收件人：

- **邮件用户**：邮件用户是独立 EOP 组织中基本类型的用户帐户。 邮件用户在你的 EOP 组织中具有登录凭据，但他们具有外部电子邮件地址 (他们的邮箱位于 EOP 组织外部) 。

  有关在 EOP 中管理邮件用户的详细信息，请参阅[在 EOP 中管理邮件用户](manage-mail-users-in-eop.md)。

- **组**：可以创建以下类型的组：

  - 通讯组
  - 启用邮件功能的安全组

  有关在 EOP 中管理组的详细信息，请参阅[在 EOP 中管理组](manage-groups-in-eop.md)。
