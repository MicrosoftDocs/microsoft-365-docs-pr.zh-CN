# <a name="manage-recipients-in-standalone-eop"></a>在独立 EOP 中管理收件人

独立 Exchange Online Protection （EOP）没有 Exchange Online 邮箱的组织支持以下类型的收件人：

- **邮件用户**：邮件用户是独立 EOP 组织中的用户帐户的基本类型。 邮件用户在 EOP 组织中有登录凭据，但他们具有外部电子邮件地址（其邮箱位于您的 EOP 组织之外）。

  有关在 EOP 中管理邮件用户的详细信息，请参阅[在 EOP 中管理邮件用户](manage-mail-users-in-eop.md)。

- **组**：您可以创建以下类型的组：

  - 通讯组
  - 启用邮件功能的安全组

  有关在 EOP 中管理组的详细信息，请参阅[在 EOP 中管理组](manage-groups-in-eop.md)。
