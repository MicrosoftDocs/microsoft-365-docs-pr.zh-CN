# <a name="auto-forwarded-messages-insight-in-the-security--compliance-center"></a>安全与合规中心自动转发的&消息见解

"安全 &**合规**中心""邮件流"仪表板的[Mail flow dashboard](mail-flow-insights-v2.md)"自动转发的邮件"见解显示有关自动从组织转发到外部域中的收件人的邮件的信息。

![安全与合规中心内的"自动转发&"小部件](../../media/mfi-auto-forwarded-messages.png)

## <a name="auto-forwarded-messages-details"></a>自动转发的消息详细信息

当您单击小部件中的消息数量时，会显示一个浮出控件窗格，显示有关自动转发邮件的更多信息：

- **通过转发方法自动转发的消息**：

  - **按邮件流规则**
  - **按收件箱规则**
  - **通过 SMTP 转发**
  - 指向更多详细信息的 [转发报告](view-mail-flow-reports.md#forwarding-report) 的链接。

- **按域和用户自动转发的邮件**：

  - **转发到的前 5 个主要域**
  - **最后一周 (新) **
  - **前 5 个转发用户**
  - **上一 (新) **
  - 指向更多 [详细信息的转发修改报告](mfi-new-users-forwarding-email.md#forwarding-modifications-report) 的链接。

![安全与合规中心中的"自动转发邮件"报告的详细信息&浮出控件](../../media/mfi-auto-forwarded-messages-details.png)

## <a name="insights"></a>见解

此报告数据生成两个见解：

- [新用户转发电子邮件](mfi-new-users-forwarding-email.md)
- [转发的新域](mfi-new-domains-being-forwarded-email.md)
