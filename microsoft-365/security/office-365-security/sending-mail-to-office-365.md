# <a name="sending-mail-to-microsoft-365"></a>向 Microsoft 365 发送邮件

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]


这些文章可帮助外部发件人改进其信誉并提高他们向 Microsoft 365 中的用户传递电子邮件的能力。 它们还提供有关您如何报告垃圾邮件和网络钓鱼（即使您不是 Microsoft 365 用户）的一些信息。

如果你不是客户，但正在尝试向正在发送邮件的人发送邮件，你的位置就对了。 如果你是管理员，并且需要帮助来反垃圾邮件，那么这不是适合你的部分。 相反，请转到 [Microsoft 365 中的反垃圾邮件和反恶意软件保护](anti-spam-and-anti-malware-protection.md)。
