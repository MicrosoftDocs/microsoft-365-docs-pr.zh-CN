# <a name="microsoft-365-architecture-icons-and-templates"></a>Microsoft 365 体系结构图标和模板

帮助合作伙伴和客户设计和构建新解决方案非常重要。 我们指南中包含的体系结构图表可帮助传达设计决策和环境中组件之间的关系。 我们使用一组符号和图标以及 Visio 模板来创建我们产品的体系结构图表，并在此处提供这些工具，以帮助您为下一个解决方案生成自定义体系结构图表。

您可以在以下页面上的可下载文件中看到这些图标、符号和模板在操作：

- [工作效率图示](productivity-illustrations.md)
- [Microsoft 云体系结构模型](cloud-architecture-models.md)
